## Session Visibility Config Research

### Task

User wants to enable cross-agent session visibility. Error message mentioned `tools.sessions.visibility=all`.

### Findings

**Parameter Path:** `tools.sessions.visibility`

**Valid Values:**

- `self` - Only current session
- `tree` (default) - Current session + spawned sessions
- `agent` - Any session belonging to current agent ID
- `all` - Any session (cross-agent access; still requires `tools.agentToAgent`)

**Source:** OpenClaw docs at docs.openclaw.ai/concepts/session-tool

### Configuration Update Attempted

1. Config is stored in SQLite database at `~/.openclaw/operator1.db` (table `op1_config`)
2. Updated database with `tools.sessions.visibility: "all"` setting
3. Verified config parses correctly with Node.js JSON5 parser
