### 2026.3.22 (Major Release - Breaking Changes)

#### Key Breaking Changes

1. **Plugin Installation**: ClawHub now preferred over npm for safe package names
2. **Browser**: Legacy Chrome extension relay removed - use existing-session/user mode
3. **Image Generation**: Standardized on core `image_generate` tool, removed nano-banana-pro skill wrapper
4. **Plugin SDK**: `openclaw/extension-api` removed, use `openclaw/plugin-sdk/*` subpaths
5. **Message Discovery**: Requires `ChannelMessageActionAdapter.describeMessageTool(...)`
6. **Matrix Plugin**: New implementation backed by matrix-js-sdk, migration required
7. **Environment Variables**: Legacy `CLAWDBOT_*` and `MOLTBOT_*` removed, use `OPENCLAW_*`
8. **State Directory**: `.moltbot` state-dir removed, migrate to `.openclaw`
9. **Security**: Exec environment blocks JVM injection, glibc tunables, .NET hijack attempts
10. **Discord**: Native command deployment switched to Carbon reconcile
11. **Webhooks**: Stricter pre-auth limits (64KB/5s, concurrent request caps)
12. **Matrix**: Dropped room chatter no longer refreshes thread bindings
13. **Agents**: Media outputs migrated to `details.media` attachment path
14. **Android**: Contact search escapes SQL LIKE wildcards
15. **Usage**: Archived sessions now included in usage totals

#### Key New Features

1. **ClawHub Skills**: Native `openclaw skills search|install|update` commands
2. **Claude Marketplace**: Plugin marketplace registry resolution and install support
3. **Plugin Bundles**: Codex, Claude, and Cursor bundle discovery/install with skill mapping
4. **Chat Commands**: `/plugins` and `/plugin` for plugin management in chat
5. **GitHub Installs**: `openclaw update --tag main` for main branch installs
6. **Agent Defaults**: Per-agent thinking/reasoning/fast mode defaults
7. **BTW Command**: `/btw` for quick context-free side questions
8. **Sandbox Backends**: Pluggable sandbox system with OpenShell (mirror/remote modes)
9. **SSH Sandbox**: Core SSH backend with secret-backed auth
10. **Browser Profiles**: Custom userDataDir support for Chromium browsers

#### Model Updates

- Default OpenAI setup model: `openai/gpt-5.4`
- Codex model: `openai-codex/gpt-5.4`
- Centralized OpenAI defaults for chat/image/TTS/transcription/embedding
