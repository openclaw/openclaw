## Oppo Reno 5R Research

User asked about custom ROM options for Oppo Reno 5R (8GB RAM). Key findings:

- "5R" naming unclear — likely Reno 5 5G, Reno 5 Pro 5G, or Reno 5 4G
- Oppo bootloader unlock is practically impossible since 2021 (server-side blocks)
- OEM unlock toggle exists but doesn't work
- No LineageOS/ArrowOS community builds for Reno 5 series
- Alternatives: debloat via ADB, repurpose as media server/home automation hub

## Email & Lark MCP Server Research

User wants to add 2 MCP servers for email management:

### Gmail — `taylorwilsdon/google_workspace_mcp` ⭐ RECOMMENDED

User corrected my initial recommendation. This is better than `marlinjai/email-mcp`:

- **Package:** `workspace-mcp`
- **Coverage:** Gmail, Calendar, Drive, Docs, Sheets, Slides, Forms, Tasks, Contacts, Chat, Search, Apps Script
- **Tools:** 100+ tools
- **Features:** Tool tiers (core/extended/complete), CLI mode, OAuth 2.1, DXT installer
- **Config:**
  ```yaml
  google-workspace:
    type: stdio
    command: uvx
    args: ["workspace-mcp", "--tool-tier", "extended"]
    env:
      GOOGLE_OAUTH_CLIENT_ID: "..."
      GOOGLE_OAUTH_CLIENT_SECRET: "..."
  ```

### Lark/Feishu — `@larksuiteoapi/lark-mcp` ⭐ OFFICIAL

- **Official** Lark/Feishu MCP from Lark Suite team
- **Tools:** 400+ tools across 20+ domains
- **Presets:** `preset.light`, `preset.default`, `preset.im.default`, `preset.base.default`, `preset.calendar.default`, etc.
- **Config:**
  ```yaml
  lark:
    type: stdio
    command: npx
    args:
      [
        "-y",
        "@larksuiteoapi/lark-mcp",
        "mcp",
        "-a",
        "<APP_ID>",
        "-s",
        "<APP_SECRET>",
        "-t",
        "preset.default",
      ]
  ```
