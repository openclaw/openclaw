## Key Integration: Apify MCP

Apify provides **10,000+ APIs/scrapers** via MCP integration. Key benefit: scale scraping without token-heavy browser automation.

Setup:

1. Get Apify API token
2. Add to `~/.openclaw/.env` as `APIFY_API_KEY`
3. Use MCP configurator to select actors
4. Add JSON config to OpenClaw
