## Activity Log Fix (Memory Dashboard)

User reported "Load more" button in Activity Log tab not working properly.

**Root cause:** `loadActivityLog` in `use-memory.ts` replaces entire log instead of appending.

**Fix proposed:**

1. Add `append` parameter to `loadActivityLog(sessionLimit, append)`
2. Track existing sessionKeys to avoid reprocessing
3. Merge new entries with existing, then re-sort

Files affected:

- `ui-next/src/hooks/use-memory.ts` - loadActivityLog function
- `ui-next/src/pages/memory.tsx` - handleLoadMore in ActivityLogTab

Note: Fix not applied (files outside workspace write scope). Provided manual patch instructions.
