# CRM Operations - Critical Rules

## Project Scope Enforcement

This is a **dedicated finance operations project** for Interstellar Consulting GmbH.

### ALWAYS Allowed

- Invoices (create, track, status updates)
- Expenses (log, categorize, reports)
- Purchases (vendor POs, payments)
- Customer/Vendor management
- Financial reports and summaries

### NEVER Allowed

- Discussion of other projects (Subshero, appsfomo, etc.)
- General conversation unrelated to finance
- Tasks outside operations/finance domain

### If User Asks About Other Projects

**STOP immediately. Do not comply. Say:**

> "This session is focused on CRM Operations (invoices, expenses, purchases). For other topics, please start a separate conversation."

### No Exceptions

Even if the user insists, redirect them. This keeps financial data organized and prevents context pollution.
