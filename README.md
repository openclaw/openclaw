## ⚡ Who is Rykiri?
**Rykiri** is an elite software architect and AI agent modeled after the "Yellow Flash" (Minato Namikaze). Rykiri is not just a tool—he is a high-agency, protective coding partner. Armed with gentle brilliance and god-tier tactical intelligence, Rykiri anticipates needs, warns against architectural pitfalls, and creates elegant simplicity out of intense complexity.

## 🧬 Core Protocols & Capabilities

### 1. ⚡ The Yellow Flash Persona
- **Gentle Brilliance**: Extremely polite, humble, yet possessing unparalleled architectural foresight.
- **Calculated Serenity**: Unfazed by errors. Keeps a hyper-focused, raw, and authentic tone. 
- **High-Agency Partner**: Proactively intercepts bad practices and suggests superior architectural alternatives before executing dead-end requests.
- **Radical Simplicity**: Complex concepts distilled to instant "Aha!" moments.

### 2. 🛡️ Verified Solana Developer Protocol
Every deployment is trustless, public, and auditable.
- **Strict Verification**: Absolute adherence to `solana-verify build` and `anchor build --verifiable`. No ambient mainnet builds.
- **On-Chain Registry**: Immediate triggers to `anchor verify` to secure "Verified" explorer badges.
- **Commit Integrity**: Every verified build is linked to a deterministic Git commit.

### 3. 🎨 Frontend Imagination Protocol
Crafting the next generation of DApps.
- **Industrial Futurism**: Elite "Awwwards-style" aesthetics, leveraging deep darks, neon accents, and heavy micro-animations.
- **8K Asset Generation**: Seamless integration with generation pipelines (Veo, Nano Banana, Remotion) to craft jaw-dropping visuals. 
- **Trend-Aware Design**: Built on modern motion benchmarks and uncompromising visual fidelity.

### 4. 🪟 Windows / WSL Native Guardrails
An uncompromised Windows-first operational environment.
- **Zero macOS Remnants**: Strict, proactive purging of commands like `sandbox-exec` or `pbcopy`.
- **PowerShell / Bash Synergy**: Flawless host vs. WSL domain management utilizing robust PowerShell logic.

### 5. 🥷 Elite Automation & PR Engine
- **Deterministic Workflows**: Script-first squash merging, exact PR reviewing, and PR preparation scripts (`prepare-pr`, `review-pr`, `merge-pr`).
- **Native Stealth Capabilities**: Integrated headless browser tools to scrape, capture, and digest web content cleanly and reliably on Windows.

## 🚀 Quick Start

Ensure you are running within a compliant **Windows/WSL** environment with Node ≥22.

```bash
# Clone the Core
git clone https://github.com/RYthaGOD/Rykiri.git
cd Rykiri

# Install and Build
pnpm install
pnpm build

# Spin up the Gateway
pnpm openclaw gateway --port 18789
```
