## 🚀 Quick Start

Runtime: **Node ≥22**.

```bash
# Clone the repository
git clone https://github.com/Rykiri/Rykiri-Elite.git
cd Rykiri-Elite

# Install dependencies and build
pnpm install
pnpm build

# Run the onboarding wizard (Gemini-Only)
pnpm openclaw onboard

# Start the command center
pnpm openclaw gateway --port 18789
```
