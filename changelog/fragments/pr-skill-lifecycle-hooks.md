- **feat(hooks)**: Add `skill_read` plugin hook for skill lifecycle observability. When the agent reads a file inside a known skill directory, the `skill_read` hook fires with the skill name, base directory, read type (`entry` for SKILL.md or `sub` for supporting files), duration, and error state. This enables plugins to track skill activation patterns, measure end-to-end skill session durations, and build usage analytics dashboards — all without modifying the core skill system.
