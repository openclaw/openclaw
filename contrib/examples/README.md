# contrib/examples — 配置与 Pack 示例

> **非 Pack 真源。** 可复制结构到 sibling 仓 `claworks-packs/<pack-id>/` 再提交。  
> 运行时加载路径：`CLAWORKS_PACKS_DIR=../claworks-packs`。

| 目录                       | 用途                                   |
| -------------------------- | -------------------------------------- |
| `starter-pack/`            | 完整 mini pack（ontology + playbooks） |
| `starter-declarative/`     | 声明式 pack 片段                       |
| `starter-imperative/`      | 命令式 pack 片段                       |
| `*.openclaw.fragment.json` | Gateway 配置片段                       |
| `*.zh.md`                  | 中文操作说明                           |

详见 [`README-claworks.md`](./README-claworks.md)。
