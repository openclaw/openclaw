# Telegram 频道同步示例

此示例展示了如何使用 Telethon 库同步 Telegram 频道消息到本地文件。

## 特性

- ✅ 超时保护 (180秒)
- ✅ 消息数量限制 (单次100条)
- ✅ Telegram 限流处理
- ✅ 详细日志输出
- ✅ 连接复用

## 使用方法

1. 安装依赖: `pip install telethon`
2. 配置 API_ID 和 API_HASH
3. 运行脚本: `python3 sync_new_messages.py`

## 文件说明

- `sync_new_messages.py` - 通用频道同步脚本模板
