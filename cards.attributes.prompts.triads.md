# Spryng 64-Card Attribute Library

This is an idea library, not a prescription system.
Use it to surface familiar starting territory, then adapt responsibly to real context signals.

## Suggested Prompt Pattern

```text
We are [audience] trying to understand [urgency].
Safety is [safety], exposure should be [exposure], and authority is [authority].
Suggest a responsible starting posture, first 2-week probe, and what to avoid over-claiming.
```
