## 🚀 快速安装步骤

### 1. 下载SDK
访问上述链接之一，下载 `ABB_PC_SDK_2025_Setup.exe`

### 2. 运行安装程序
```batch
ABB_PC_SDK_2025_Setup.exe
```

### 3. 选择安装位置
```
C:\Program Files (x86)\ABB\SDK\PCSDK 2025
```

### 4. 选择完整安装
- ✅ 选择 "Complete Installation"
- ❌ 不要选择 "Minimal Installation"

### 5. 等待安装完成
通常需要 5-10 分钟

### 6. 验证安装
```powershell
Test-Path "C:\Program Files (x86)\ABB\SDK\PCSDK 2025\Bin\ABB.Robotics.Controllers.dll"
```

## 🔧 安装后编译

### 1. 运行编译脚本
```batch
cd D:\OpenClaw\Develop\openclaw\extensions\abb-robot-control\src
自动编译ABBBridge.bat
```

### 2. 安装edge-js
```bash
npm install edge-js
```

### 3. 启动服务
```bash
node openclaw.mjs gateway --port 18789
```

### 4. 打开浏览器
```
http://127.0.0.1:18789
```
