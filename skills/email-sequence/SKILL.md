# Email Sequence

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** 뷰티 네일 브랜드사 의사결정자 (구매팀장, 브랜드 디렉터, 대표)
- **Goal:** OEM/ODM 파트너십 문의 → 샘플 → 정규 파트너십
- **Tone:** 격식 있되 딱딱하지 않게 — 한국 비즈니스 이메일 관행 준수
- **Korean Email Custom:** 첫 인사는 격식 있게, 요청은 명확하게, 마무리는 열린 결말로

## 핵심 원칙

1. **이메일 하나 = 목적 하나** — 한 이메일에 여러 CTA 금지
2. **가치 먼저, 요청 나중** — 3개 이메일은 주고, 1개 이메일은 요청
3. **관련성 > 볼륨** — 10개 타겟 이메일 > 100개 스팸 이메일
4. **명확한 다음 단계** — 받는 사람이 "그래서 뭘 하라는 거지?"가 없어야 함
5. **후속 이메일 필수** — 응답률의 80%는 3번째 이후 이메일에서 발생

## 시퀀스 1: 콜드 아웃리치 (신규 브랜드사 발굴)

**목표:** 처음 접하는 브랜드사 담당자의 관심 유도

**길이:** 5개 이메일 (2주 간격)

---

### 이메일 1: 첫 접촉 (Day 0)

**제목:** [브랜드명] 네일 라인을 더 빠르게 론칭하는 방법

```
안녕하세요, [이름]님.

[브랜드명]의 최근 [SNS/컬렉션/행사]을 보고 연락드립니다.

저는 Enth의 [이름]입니다. Enth는 국내 200여 개 뷰티 브랜드의
네일 OEM/ODM 파트너로, 평균 6주 납기와 불량률 0.08%를 유지하고 있습니다.

[브랜드명]처럼 [시장/채널]에서 성장하는 브랜드에게
[구체적 혜택: 예: 소량 시작 가능한 파트너십]을 제공할 수 있을 것 같아서요.

혹시 이번 달 30분 정도 통화하실 수 있으신가요?
아니면 먼저 샘플부터 보내드릴까요?

감사합니다.
[서명]
```

---

### 이메일 2: 가치 제공 (Day 7)

**제목:** [브랜드명] 참고하실 만한 2026 네일 트렌드 리포트

```
안녕하세요, [이름]님.

지난주 연락 후 추가로 연락드립니다.

저희가 최근 발행한 "2026 S/S 네일 트렌드 리포트"가
도움이 될 것 같아 공유드립니다. [링크]

특히 [브랜드명]이 주력하는 [컬러/카테고리] 관련 내용이
페이지 12–15에 담겨 있습니다.

다음 시즌 기획에 참고하시고,
궁금한 점 있으시면 편하게 연락 주세요.

[서명]
```

---

### 이메일 3: 사회적 증거 (Day 14)

**제목:** 비슷한 규모의 브랜드가 Enth와 함께한 결과

```
안녕하세요, [이름]님.

[브랜드명]과 비슷한 단계의 인디 뷰티 브랜드가
Enth와 파트너십 후 첫 시즌에 어떤 결과를 얻었는지
짧게 공유드립니다.

[케이스 스터디 2–3문장 요약: 론칭 기간 단축, 품질 개선 등]

전체 사례는 여기서 보실 수 있습니다: [링크]

[브랜드명]에도 비슷한 결과를 드릴 수 있다고 생각합니다.
샘플 한 번 받아보시겠어요?

[서명]
```

---

### 이메일 4: 직접 제안 (Day 21)

**제목:** [브랜드명] — 무료 샘플 + 제조 역량 자료 제안

```
안녕하세요, [이름]님.

몇 차례 연락드렸는데 바쁘신 것 같아
이번이 마지막 연락입니다.

[브랜드명]을 위해 구체적인 제안을 드리고 싶습니다:

✓ 관심 있는 컬러/제품 무료 샘플 3종
✓ 맞춤 제조 역량 자료 (MOQ, 납기, 단가 기준)
✓ 30분 화상 미팅 (제조 프로세스 Q&A)

비용 없이 먼저 Enth의 역량을 확인해 보세요.
샘플 신청: [링크] (2분 소요)

[서명]
```

### 이메일 5: 재활성화 (Day 60 이후)

**제목:** [브랜드명] — 새로운 소식이 있어서요

```
안녕하세요, [이름]님.

두 달 전 연락드렸던 Enth [이름]입니다.

최근 [신제품 라인 / 새로운 서비스]를 준비했는데,
[브랜드명]에 맞을 것 같아 다시 연락드립니다.

[한 줄 새로운 소식 또는 관련 콘텐츠 링크]

혹시 지금 시점에 이야기 나눠보실 의향이 있으신가요?

[서명]
```

## 시퀀스 2: 웜 리드 육성 (샘플 신청 후)

**목표:** 샘플 신청자를 파트너십 문의로 전환

**길이:** 4개 이메일 (1주 간격)

| 이메일 | 타이밍           | 내용                                      |
| ------ | ---------------- | ----------------------------------------- |
| 1      | 샘플 발송 당일   | 샘플 발송 확인 + 확인 포인트 안내         |
| 2      | 샘플 수령 후 3일 | 샘플 피드백 요청 + 추가 컬러 제안         |
| 3      | 1주 후           | 파트너십 옵션 소개 (MOQ, 납기, 가격 기준) |
| 4      | 2주 후           | "다음 단계" 미팅 제안                     |

## 이메일 작성 가이드

### 제목 라인 (Subject Line) 공식

| 유형   | 공식                | 예시                                  |
| ------ | ------------------- | ------------------------------------- |
| 개인화 | [브랜드명] + [혜택] | "[브랜드명] 납기 2주 단축하는 방법"   |
| 호기심 | 질문형              | "네일 OEM에서 가장 많이 하는 실수는?" |
| 긴급성 | 시즌 + 마감         | "여름 시즌 OEM 마감 3주 전"           |
| 가치   | 숫자 + 결과         | "6주 → 4주: Enth 익스프레스 납기"     |

### 이메일 구조

```
훅 (1–2문장): 왜 지금 이 이메일인가?
맥락 (1–2문장): 관련성 있는 배경
가치 (2–3문장): 받는 사람이 얻는 것
CTA (1문장): 딱 하나의 명확한 다음 단계
서명: 이름, 직함, 연락처
```

### 한국 B2B 이메일 관행

- 첫 이메일: "안녕하세요, [이름]님." (이름 존칭 사용)
- 직급/회사명 확인 후 적절한 호칭 ("대표님", "담당자님")
- 과도한 "죄송합니다" 사용 지양 (자신감 있게)
- 마무리: "편하게 연락 주세요" (열린 결말 유지)
- 서명에 전화번호 반드시 포함 (한국 비즈니스는 전화 선호)

## 성과 지표

| 지표             | 목표 기준                            |
| ---------------- | ------------------------------------ |
| 오픈율           | 30% 이상 (B2B 이메일 업계 평균 ~22%) |
| 클릭율           | 5% 이상                              |
| 응답율           | 10% 이상 (콜드 시퀀스)               |
| 샘플 신청 전환율 | 응답자 대비 30% 이상                 |
