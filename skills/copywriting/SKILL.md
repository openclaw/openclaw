# Copywriting

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** B2B 뷰티 네일 브랜드사 (구매팀, 브랜드 매니저, 창업자)
- **Channels:** Blog (Ghost CMS) + LinkedIn
- **Market:** 한국 뷰티 산업, 글로벌 네일 시장
- **Tone:** 전문적, 권위 있는, 파트너십 지향적 — 과도한 세일즈 톤 지양
- **Primary CTA:** "파트너십 문의하기" / "샘플 요청하기" / "제조 역량 보기"
- **Key Trust Signals:** 대한민국 1위 제조, ISO 인증, 생산 규모, 브랜드 파트너 레퍼런스

## Core Principles

1. **명확함 > 영리함** — 브랜드 바이어는 시간이 없다. 5초 안에 핵심 가치를 전달하라
2. **혜택 > 기능** — "최첨단 설비"가 아니라 "출시 일정 단축, 불량률 0.1% 미만"
3. **구체성 > 모호함** — "업계 최고"가 아니라 "연간 500만 개 생산 능력, 국내 1위"
4. **능동태 > 수동태** — "제조됩니다"가 아니라 "Enth가 제조합니다"
5. **확신 > 조건부** — "~할 수 있습니다"가 아니라 "~합니다"

## Writing Style Rules

| 피해야 할 것            | 대신 사용                  |
| ----------------------- | -------------------------- |
| 복잡한 수식어           | 단순하고 강력한 동사       |
| 업계 전문 용어 과다     | 바이어가 이해하는 언어     |
| "저희는 ~에 헌신합니다" | 구체적인 수치와 사례       |
| "혹시 관심 있으시면..." | "다음 단계는 X입니다"      |
| 긴 단락                 | 3줄 이하 단락, 글머리 기호 |

## Page Structure Framework

### 히어로 섹션 (Above the Fold)

```
헤드라인: [구체적 결과] + [대상] + [신뢰 신호]
예: "한국 1위 네일 제조사와 함께, 6주 안에 첫 컬렉션을 론칭하세요"

서브헤드라인: 혜택 확장 (1–2문장)
예: "Enth는 연간 500만 개 생산 능력과 자체 R&D 연구소로
국내외 200+ 브랜드의 OEM/ODM 파트너입니다"

Primary CTA: [낮은 마찰의 행동]
예: "무료 샘플 신청하기" / "제조 역량 자료 받기"
```

### 핵심 섹션 순서

1. **사회적 증거** — 파트너 브랜드 로고, 생산 규모, 연도
2. **문제 제기** — 브랜드사가 겪는 제조 고민 (품질 불안정, 긴 리드타임, 최소 주문량)
3. **솔루션** — Enth의 차별화된 해결책
4. **이의 처리** — FAQ 형태로 주요 걱정 선제 해결
5. **CTA** — 다시 한 번, 구체적인 다음 단계

## CTA Copy Guidelines

| 약한 CTA    | 강한 CTA                    |
| ----------- | --------------------------- |
| 문의하기    | 5분 안에 샘플 신청하기      |
| 더 알아보기 | 제조 역량 브로셔 다운로드   |
| 연락하기    | 담당자와 30분 미팅 예약하기 |
| 제출하기    | 파트너십 제안서 받기        |

## Page-Specific Guidance

### 블로그 포스트 (SEO + 교육 목적)

- 헤드라인: 검색 의도 포함 + 숫자/연도 (예: "2026년 네일 OEM 트렌드 5가지")
- 인트로: 독자의 문제를 먼저 인정하고 솔루션 예고
- 마무리: 자연스러운 CTA로 연결 (하드 셀 금지)
- 메타 디스크립션: 키워드 + 혜택 + 행동 유도

### LinkedIn 포스트 (B2B 네트워크)

- 첫 줄이 전부: 스크롤을 멈추게 하는 훅
- 개인 스토리 또는 업계 인사이트로 시작
- 3–5줄 단락, 공백 행으로 리듬 생성
- 마지막: 질문 또는 부드러운 CTA

### 파트너십 랜딩 페이지

- 위에서 아래로: 신뢰 구축 → 역량 증명 → 문의 유도
- 폼: 최소 필드 (이름, 회사, 이메일, 관심 제품 카테고리)
- 폼 제목: "무료 샘플 신청" (계약 아님을 명시해 마찰 감소)

## Voice & Tone

**Enth 브랜드 보이스:**

- 자신감 있지만 오만하지 않음
- 전문적이지만 딱딱하지 않음
- 데이터 기반이지만 인간적
- 한국 뷰티 산업의 자부심

**피해야 할 표현:**

- "저희는 고객 만족을 최우선으로..." (클리셰)
- "혁신적인 솔루션을 제공합니다" (모호)
- "업계 최고의 품질" (증거 없는 주장)

**선호하는 표현:**

- "200개 브랜드가 선택한 이유"
- "불량률 0.08% — 업계 평균의 1/3"
- "6주 샘플 납기 보장"
