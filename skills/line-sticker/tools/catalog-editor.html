▸ How the AI uses each field to select a sticker

The AI reads the desc field as the primary signal for sticker selection.
AIはdescフィールドを主要なシグナルとしてスタンプを選択します。
Write desc as if describing the sticker to someone who cannot see it —
include the character pose, facial expression, action, and the emotional
context in which this sticker would feel natural.
スタンプが見えない人に説明するように記述してください。キャラのポーズ・表情・動作と、このスタンプが自然に感じられる感情的な文脈を含めてください。
The text field captures any printed words on the sticker exactly as shown,
including emoji. This is used to filter by language.
textフィールドは絵文字を含め印字されたテキストをそのまま記載します。言語フィルタとして使用されます。
