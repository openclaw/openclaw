# Marketing Psychology

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** B2B 뷰티 네일 브랜드사 의사결정자 (구매팀장, 브랜드 디렉터, 대표)
- **Channels:** Blog (Ghost CMS) + LinkedIn
- **B2B 구매 특성:** 집단 의사결정, 긴 구매 사이클, 리스크 회피 성향, 관계 중심
- **Key Insight:** B2B 구매자도 감정으로 결정하고 논리로 정당화한다

## 1급 사고 모델 (Foundational Thinking Models)

| 모델                     | Enth 적용                                                                           |
| ------------------------ | ----------------------------------------------------------------------------------- |
| **퍼스트 프린시플**      | "왜 브랜드사들이 제조사를 바꾸는가?" — 가격? 품질? 속도? 근본 원인 파악             |
| **Jobs To Be Done**      | 브랜드사가 OEM 파트너에게 기대하는 '일': 일정 준수, 품질 안정, 트렌드 제안          |
| **이코노미 오브 어텐션** | LinkedIn/블로그에서 3초 안에 가치를 전달하지 못하면 기회 없음                       |
| **파레토 법칙**          | 상위 20% 브랜드 파트너가 매출의 80% → 그들에게 집중한 콘텐츠                        |
| **기회비용**             | "Enth와 파트너하지 않으면 어떤 비용이 발생하는가?" — 리드타임, 품질 이슈, 기회 손실 |

## 2급 구매자 심리 이해

### B2B 의사결정자 편향

**확증 편향 (Confirmation Bias)**

- 이미 "한국 제조사가 좋다"고 생각하는 바이어에게 그 믿음을 강화할 데이터 제공
- 사례: "글로벌 뷰티 브랜드 82%가 한국 OEM을 선택한 이유"

**사회적 증거 (Bandwagon Effect)**

- "200개 브랜드가 이미 선택했습니다" — 구체적 숫자가 효과적
- 경쟁사가 Enth를 쓴다면 강력한 신호 (단, 비밀유지 계약 주의)

**손실 회피 (Loss Aversion)**

- 이득 강조보다 손실 회피가 2배 강력
- "지금 파트너를 바꾸지 않으면 다음 시즌 트렌드를 놓칩니다"
- "불량률 1%만 줄여도 연간 ○○만원 절감"

**현상 유지 편향 (Status-Quo Bias)**

- 기존 제조사에 머무르려는 관성 → "전환 비용 0, 첫 샘플 무료" 제공
- 작은 첫 발걸음으로 관성 극복: 샘플 → 소량 주문 → 정규 파트너

**닻 내리기 (Anchoring)**

- 가격 제시 전 고품질 기준 먼저 제시: "프리미엄 시장 기준 단가 ○○○원 → Enth 제안가 ○○원"
- 패키지 옵션: 고가 옵션 먼저 보여주면 중간 옵션이 합리적으로 보임

**IKEA 효과**

- 샘플 커스터마이징 과정에 바이어를 참여시킴 → 애착 형성 → 전환율 상승

**제로 가격 효과 (Zero-Price Effect)**

- "무료 샘플"은 단순 할인보다 훨씬 강력한 전환 드라이버

## 3급 설득 & 행동 유도

### 영향력 6원칙 (Cialdini) — Enth 적용

| 원칙                           | 적용 방법                                                                      |
| ------------------------------ | ------------------------------------------------------------------------------ |
| **호혜성 (Reciprocity)**       | 가치 있는 콘텐츠 무료 제공 (트렌드 리포트, 제조 가이드) → 나중에 파트너십 문의 |
| **일관성 (Commitment)**        | 샘플 요청 → 소량 주문 → 정규 파트너 → 장기 계약 (발걸음 기법)                  |
| **권위 (Authority)**           | "대한민국 1위", ISO 인증, 수상 이력, 미디어 노출 적극 활용                     |
| **호감 (Liking)**              | LinkedIn에서 개인적인 스토리와 업계 인사이트 공유 → 신뢰 형성                  |
| **희소성 (Scarcity)**          | "이번 시즌 신규 파트너 모집 5개사 한정" — 진정성 있을 때만 사용                |
| **사회적 증거 (Social Proof)** | 파트너 로고, 생산 수량, 브랜드 성공 사례 케이스 스터디                         |

### 행동 유도 프레임워크

**BJ Fogg 모델 (행동 = 동기 × 능력 × 트리거)**

- 동기: 손실 회피 + 성공 욕구 ("라이벌 브랜드는 이미 파트너십 중")
- 능력: 진입 장벽 최소화 (샘플 무료, 최소 주문량 낮게 시작)
- 트리거: 명확한 CTA + 딱 맞는 타이밍 (시즌 직전, 신제품 기획 시점)

## 4급 가격 심리

- **매력 효과 (Decoy Effect):** 3개 옵션 제시 시 중간 옵션 선택 유도
- **반올림 가격 vs 매력 가격:** B2B는 투명한 라운드 가격이 신뢰도 높음 (990원보다 1,000원)
- **정신 회계 (Mental Accounting):** 초기 비용을 "투자" 프레임으로 — "샘플 비용은 첫 주문 시 차감"
