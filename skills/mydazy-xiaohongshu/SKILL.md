# mydazy 小红书种草工作流（3-8 岁）

用于为 mydazy 规划、生成和迭代小红书内容，目标受众为 3-8 岁儿童家长。

## 触发条件（命中就用）

- “给 mydazy 做小红书”
- “写 3-8 岁亲子文案”
- “做种草内容/种草矩阵”
- “输出可直接发布的小红书文案”

## 输入信息采集（缺失就先补齐）

最少需要以下信息，缺什么就先问：

1. 产品信息：

- 品类、核心功能、价格区间、购买方式

2. 受众画像：

- 孩子年龄段（默认 3-8 岁）、家长关注点（启蒙/专注力/英语/安全等）

3. 内容目标：

- 引流私域 / 提升互动 / 促进下单（三选一主目标）

4. 风格偏好：

- 温和分享 / 专业测评 / 场景故事（默认“真实家长分享”）

默认值（用户未提供时）：

- 年龄：3-8 岁
- 语气：真实、克制、不夸张
- 目标：先互动后转化
- 每次交付：3 条笔记

## 标准工作流（端到端）

1. 信息结构化

- 产出“产品卖点-家长痛点-场景映射”三列表。
- 每条卖点只对应 1-2 个核心场景，避免堆叠。

2. 选题矩阵

- 输出 5 个选题方向，每个方向含：
  - 目标人群切片（如：一年级家长）
  - 核心冲突（如：孩子坐不住）
  - 解决路径（mydazy 在其中的角色）

3. 文案生成（每条笔记固定结构）

- 标题：提供 5 个（含 1 个对照型、1 个提问型、1 个场景型）
- 正文：按以下 6 段写作：
  1. 痛点场景（1-2 句）
  2. 真实经历/观察（2-3 句）
  3. 解决尝试（可含对比）
  4. mydazy 使用感（可量化但不绝对）
  5. 适用人群与不适用人群
  6. 互动引导（提问）
- 标签：6-10 个，优先“亲子教育/启蒙/年龄段/场景词”
- 评论区首评：1 条，用于补充信息和引导收藏私信

4. A/B 变体

- 同一主题给出 A/B 两版：
  - A：情绪共鸣强
  - B：信息密度高
- 明确建议先发哪版，并给出判断依据。

5. 发布节奏建议

- 7 天节奏（可直接执行）：
  - D1-D2：痛点共鸣
  - D3-D4：方法与对比
  - D5：真实反馈
  - D6：答疑贴
  - D7：总结+行动引导

## 合规边界（必须执行）

1. 禁用绝对化承诺：

- 不写“100% 有效”“立刻提升”“必买”。

2. 不做医疗化暗示：

- 不将普通教育产品描述为医疗干预。

3. 儿童场景表达克制：

- 强调“个体差异”“家长陪伴”。

4. 不伪造权威背书：

- 没有明确来源时，不能写“专家一致推荐”。
