Aesthetic directions

One skill, many worlds.

Concept stills that show the range of looks the skill can art-direct — each generated by its own fal.ai pipeline, the same art-direction engine you get in Mode B. They illustrate possible directions, not finished pages; for sites you can actually scroll end-to-end, jump to the live examples. In a real build, each becomes the depth-0.75 subject layer of a pinned chapter, parallaxing under HTML titles while the background morphs.

Live examples

Five aesthetics. One grammar.

Same engine, deliberately clashing aesthetics — proof the look is a variable, not a default. Each is a single build-free index.html, dependency-free vanilla JS on requestAnimationFrame, transform/opacity-only. All five render fully with zero image files. Click any to scroll the real thing.

▶
① Renaissance editorial

Classic Touch.

Warm, classical, ornate. Oil-painting heroes, gold↔oxblood morph, serif display. Mirrors w230.net/reinassence.

▶
② Brutalist portfolio

Maya Torres.

Cold, modern, severe. Giant grotesk type, monochrome + electric-blue accent, grey↔ink morph, scroll-driven 3D camera.

▶
③ Sci-fi noir

Vantascope.

Near-black void, deep-teal fog, crimson edge-light, film grain. Studio VANTASCOPE, title HOLLOW STAR — 4 chapters, scroll-linked 3D camera, vertical mask-wipe reveals.

▶
④ Quiet luxury

Maison Solenne.

Warm ivory + sand, muted cognac accent, thin-serif display, vast negative space. 220vh pins, ~3% background drift, letter-spacing-scrub title reveals.

▶
⑤ Gen-Z pop

Bloom.

Candy-pink + electric-lime gradients, bold rounded type, fast parallax with wide depth gaps, floating CSS phone-UI layer.

No framework, no build step, no API keys. Drop your own art into assets/ and edit the chapters data — the whole page regenerates. The motion is the constant; the aesthetic is yours.
