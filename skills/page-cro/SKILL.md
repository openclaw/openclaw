# Page CRO (Conversion Rate Optimization)

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** B2B 뷰티 네일 브랜드사 의사결정자
- **Key Conversion Goals:**
  - 파트너십 문의 폼 제출
  - 샘플 신청
  - 브로셔/역량 자료 다운로드
  - LinkedIn 팔로우 또는 DM 시작
- **B2B CRO 특성:** 긴 의사결정 사이클, 복수 의사결정자, 신뢰 우선

## CRO 분석 프레임워크 (영향도 순)

### 1순위: 가치 제안 명확성 (Value Proposition Clarity)

> "방문자가 5초 안에 이 페이지가 자신에게 왜 중요한지 이해할 수 있는가?"

체크리스트:

- [ ] 헤드라인이 "우리가 하는 일"이 아닌 "당신이 얻는 것"에 초점인가?
- [ ] 네일 브랜드 창업자/구매 담당자가 즉시 공감할 수 있는 언어인가?
- [ ] 경쟁사 대비 차별점이 명확한가? (단순 "1위"가 아닌 구체적 수치)

**Enth 예시:**

- ❌ "Enth는 네일 제품 전문 OEM/ODM 제조사입니다"
- ✅ "6주 납기 보장, 불량률 0.08% — 200개 브랜드가 선택한 이유"

### 2순위: 헤드라인 효과성

- 혜택 중심 헤드라인 테스트 (기능 vs 결과)
- 숫자/구체성 포함 버전 vs 감성 버전 A/B 테스트
- 서브헤드라인이 헤드라인을 보강하는지 확인

### 3순위: CTA 배치 & 문구

| CTA 항목    | 점검                                              |
| ----------- | ------------------------------------------------- |
| 페이지 상단 | 첫 화면 안에 CTA 버튼 1개 이상 존재하는가?        |
| 스크롤 중간 | 긴 페이지에서 중간 CTA 있는가?                    |
| 페이지 하단 | 마지막 CTA가 명확한 다음 단계를 제시하는가?       |
| 버튼 텍스트 | "제출하기" → "무료 샘플 신청하기"처럼 구체적인가? |
| 색상 대비   | 배경 대비 버튼이 눈에 띄는가?                     |

### 4순위: 시각적 위계 & 스캔 가능성

- F-패턴 또는 Z-패턴으로 핵심 정보 배치
- 블록 텍스트 대신 글머리 기호, 표, 이미지 활용
- 모바일에서도 핵심 내용이 스크롤 없이 보이는가?

### 5순위: 신뢰 신호 & 사회적 증거

**Enth 필수 신뢰 신호:**

```
□ 파트너 브랜드 로고 (익명 포함 가능)
□ 생산 능력 수치 (연간 ○○만 개)
□ 품질 인증 (ISO, CGMP 등)
□ 업력 / 설립 연도
□ 팀 사진 또는 공장 사진
□ 미디어 노출 (언론 보도)
□ 고객 후기 (실명 또는 "○○ 브랜드 구매팀장")
```

### 6순위: 이의 처리 (Objection Handling)

**B2B 네일 브랜드사 주요 이의:**

| 이의                      | 페이지에서 해소하는 방법                     |
| ------------------------- | -------------------------------------------- |
| "MOQ가 너무 높지 않을까?" | 소량 시작 옵션 명시 (예: 최소 500개부터)     |
| "품질을 믿을 수 있을까?"  | 샘플 무료 제공 + 품질 인증 배지              |
| "납기가 길지 않을까?"     | 평균 납기 수치 명시 (예: 샘플 2주, 양산 6주) |
| "가격이 비싸지 않을까?"   | 가치 대비 설명 또는 "무료 견적 받기" CTA     |
| "커뮤니케이션이 잘 될까?" | 전담 담당자 배정 명시, 연락처 공개           |

### 7순위: 마찰 포인트 제거

- 폼 필드 최소화: 이름, 회사명, 이메일, 관심 카테고리 (4개면 충분)
- 폼 제목을 "계약 상담"이 아닌 "무료 샘플 신청"으로 (낮은 심리적 장벽)
- 폼 하단: "영업일 기준 1일 내 답변 드립니다" (안심 메시지)

## 페이지별 CRO 체크리스트

### 홈페이지

- [ ] 5초 테스트: 무엇을 하는 회사인지 즉시 파악 가능한가?
- [ ] 파트너 브랜드 로고 섹션 존재하는가?
- [ ] 명확한 다음 단계 CTA (샘플 / 문의 / 역량 자료)

### 파트너십 랜딩 페이지

- [ ] 헤드라인이 브랜드사의 결과에 집중하는가?
- [ ] 신뢰 신호 3개 이상 존재하는가?
- [ ] 폼이 폴드 위 또는 스크롤 절반 이내에 있는가?
- [ ] 모바일에서 CTA 버튼이 엄지손가락으로 누르기 편한가?

### 블로그 포스트

- [ ] 인트로에서 독자의 문제를 인정하는가?
- [ ] 본문 중간에 자연스러운 CTA가 있는가?
- [ ] 마무리에 명확한 다음 단계가 있는가?
- [ ] 관련 포스트 링크로 체류 시간 연장하는가?

## 실험 아이디어 (A/B 테스트)

| 실험                                      | 예상 효과                           |
| ----------------------------------------- | ----------------------------------- |
| 헤드라인: "1위 제조사" vs "6주 납기 보장" | 구체적 혜택이 더 높은 전환율 예상   |
| CTA: "문의하기" vs "무료 샘플 신청하기"   | 낮은 마찰 CTA가 CTR 향상 예상       |
| 폼 위치: 페이지 상단 vs 하단              | 상단이 전환율 높을 수 있음          |
| 신뢰 신호: 로고 그리드 vs 케이스 스터디   | 케이스 스터디가 품질 리드 유도 예상 |
