# Content Strategy

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** B2B 뷰티 네일 브랜드사 (신규 브랜드 창업자, 기존 브랜드 구매 담당자)
- **Channels:** Blog (Ghost CMS, SEO) + LinkedIn (B2B 네트워크)
- **Core Goal:** 검색 + 소셜에서 "네일 OEM 파트너" 1순위 포지셔닝

## 검색 가능한 콘텐츠 vs 공유 가능한 콘텐츠

| 유형          | 채널     | 목적                                       |
| ------------- | -------- | ------------------------------------------ |
| **검색 가능** | 블로그   | Google 유입, 잠재 파트너 교육, 장기 트래픽 |
| **공유 가능** | LinkedIn | 빠른 인지도, 업계 신뢰, 바이럴 도달        |

두 채널은 상호 보완: 블로그 글을 LinkedIn으로 배포 → LinkedIn 인게이지먼트로 SEO 신호 강화.

## 콘텐츠 필러 & 토픽 클러스터

### Enth 4대 콘텐츠 필러

```
필러 A: 네일 제조 & 기술
  클러스터: OEM 프로세스, ODM 차이점, 원료 성분, 품질 관리, R&D

필러 B: OEM/ODM 파트너십 가이드
  클러스터: 파트너 선택 기준, MOQ, 리드타임, 샘플 프로세스, 계약

필러 C: 뷰티 트렌드 & 시장 인사이트
  클러스터: 시즌별 네일 트렌드, 컬러 예측, 글로벌 뷰티 시장, 한국 뷰티 현황

필러 D: 브랜드 성공 스토리
  클러스터: 파트너 케이스 스터디, 론칭 스토리, 업계 인터뷰
```

## 구매 여정별 키워드 전략

### 인지 단계 (Awareness)

검색자: "네일 제품은 어디서 만들지?"

| 키워드 유형    | 예시                                                    |
| -------------- | ------------------------------------------------------- |
| What is / 이란 | "네일 OEM 이란", "네일 ODM 차이"                        |
| How to / 방법  | "네일 브랜드 만드는 법", "네일 제품 제조 방법"          |
| Guide / 가이드 | "네일 OEM 완전 가이드", "nail manufacturer Korea guide" |

### 검토 단계 (Consideration)

검색자: "어떤 OEM 제조사가 좋을까?"

| 키워드 유형 | 예시                                             |
| ----------- | ------------------------------------------------ |
| Best / 최고 | "네일 OEM 업체 추천", "best nail OEM Korea"      |
| vs / 비교   | "네일 OEM vs ODM 비교", "국내 vs 해외 네일 제조" |
| 리뷰 / 후기 | "Enth 네일 OEM 후기", "한국 네일 제조사 비교"    |

### 결정 단계 (Decision)

검색자: "Enth에 문의해볼까?"

| 키워드 유형 | 예시                                            |
| ----------- | ----------------------------------------------- |
| 가격 / 단가 | "네일 OEM 가격", "네일 ODM 최소 주문량"         |
| 샘플        | "네일 OEM 샘플 신청", "네일 제조 샘플"          |
| 문의        | "Enth 네일 파트너십", "네일 제조사 파트너 문의" |

## 콘텐츠 우선순위 점수 매트릭스

| 기준               | 가중치 | 평가 방법                                  |
| ------------------ | ------ | ------------------------------------------ |
| 고객 영향도        | 40%    | 파트너십 결정에 얼마나 직결되는가?         |
| 콘텐츠-시장 적합성 | 30%    | Enth의 강점을 자연스럽게 보여줄 수 있는가? |
| 검색 잠재력        | 20%    | 월 검색량 + 경쟁도                         |
| 제작 리소스        | 10%    | 얼마나 빠르게, 저렴하게 만들 수 있는가?    |

점수 70점 이상 → 즉시 제작  
50–69점 → 다음 분기  
50점 미만 → 보류

## 콘텐츠 아이디어 소스

1. **키워드 데이터:** Google Search Console, Naver 키워드 도구, Ahrefs
2. **고객 질문:** 영업팀이 가장 자주 받는 질문 10가지
3. **포럼 리서치:** 뷰티 창업자 커뮤니티 (네이버 카페, 인스타 DM, LinkedIn 그룹)
4. **경쟁사 분석:** 상위 노출 경쟁사 블로그 주제 분석 → 더 잘 커버하기
5. **시즌 이벤트:** 패션 위크, 뷰티 박람회 (Cosmoprof, COSMOBEAUTY SEOUL) 전후

## 블로그 vs LinkedIn 콘텐츠 배분

```
블로그 (검색 최적화):
  — 심층 가이드, OEM 교육 콘텐츠, 트렌드 리포트 (1,500자+)
  — 목표: Google/Naver SEO 1페이지 진입
  — CTA: 샘플 신청, 브로셔 다운로드, 파트너십 문의

LinkedIn (관계 구축):
  — 인사이트 요약, 개인 스토리, 성공 사례 (300–800자)
  — 목표: 업계 인지도, 팔로워 신뢰, 파트너 발굴
  — CTA: 댓글 참여, DM 유도, 블로그 링크 공유
```

## 콘텐츠 캘린더 프레임워크

| 월      | 블로그 주제 예시              | LinkedIn 주제 예시        |
| ------- | ----------------------------- | ------------------------- |
| 1–2월   | 2026 봄/여름 네일 트렌드 예측 | 신년 파트너십 모집 공고   |
| 3–4월   | 봄 시즌 OEM 준비 타임라인     | 파트너 브랜드 성공 사례   |
| 5–6월   | 여름 네일 성분 트렌드         | Cosmoprof 참가 후기       |
| 7–8월   | 가을/겨울 기획 시작 가이드    | R&D 팀 인터뷰             |
| 9–10월  | 홀리데이 시즌 네일 트렌드     | 연말 파트너 모집          |
| 11–12월 | 연간 뷰티 시장 결산           | 내년도 파트너십 계획 공유 |

## 핵심 원칙

- 대부분의 블로그 콘텐츠는 `/blog` 아래 단일 경로로 통합 (링크 분산 방지)
- 별도 허브/스포크 구조는 "네일 제조 가이드" 같은 대주제에만 적용
- 모든 블로그 포스트는 LinkedIn에서 한 번 더 배포 → 복합 도달 효과
