# Legal Response Skill

金融詐欺・法務トラブルに対する緊急対応スキル

## 機能

1. **詐欺検出**: 不審な取引パターンを検出
2. **法的アドバイス**: 日本法に基づくアドバイス提供
3. **緊急連絡先**: 警察・銀行・弁護士への連絡支援
4. **書類作成**: 被害届テンプレート生成

## 緊急連絡先

| 機関 | 電話番号 | 用途 |
|------|----------|------|
| 警察 | 110 | 被害届・緊急通報 |
| ゆうちょ銀行 | 0120-108-420 | 口座凍結 |
| 法テラス | 0570-078374 | 無料法律相談 |
| 消費者ホットライン | 188 | 消費者相談 |

## 使用例

```
ユーザー: 詐欺被害に遭いました
→ 緊急連絡先の提示 + 対応手順のガイド

ユーザー: 口座凍結したい
→ ゆうちょ銀行への連絡方法を案内
```

## 法的根拠

- 振り込め詐欺救済法
- 刑法（詐欺罪・電子計算機使用詐欺罪）
- 民法（不当利得返還請求）
