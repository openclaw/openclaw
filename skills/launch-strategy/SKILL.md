# Launch Strategy

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** B2B 뷰티 네일 브랜드사 (신규 파트너 모집 또는 신제품 라인 출시)
- **Channels:** Blog (Ghost CMS) + LinkedIn + 이메일 + 업계 미디어
- **Launch Types:** 신규 제품 라인 출시, 신규 파트너 모집 캠페인, OEM 서비스 업그레이드 발표

## 핵심 철학

> 론칭은 단발성 이벤트가 아니다. 각 출시를 다음 출시를 위한 발판으로 쌓아라.

- 매번 배운 것을 기록하고 다음 론칭에 반영
- 파트너 모집은 "항상 열려 있는 채용 공고"처럼 운영 — 비수기에도 인지도 유지
- 첫 소규모 론칭 → 피드백 → 보완 → 대형 론칭 순서

## ORB 프레임워크 (Enth 적용)

### Owned Channels (Enth 소유)

- 블로그 (Ghost CMS): SEO 콘텐츠, 발표 포스트
- 이메일 리스트: 기존 파트너 + 관심 브랜드 리스트
- LinkedIn 페이지 + 개인 프로필
- 홈페이지 / 랜딩 페이지

### Rented Channels (알고리즘 의존)

- LinkedIn 알고리즘 (팔로워 피드)
- Naver 검색 결과
- Instagram (뷰티 브랜드 팔로워)
- YouTube (제조 공정 교육 영상)

### Borrowed Channels (파트너 채널)

- 뷰티 브랜드 대표들의 LinkedIn 공유
- 업계 미디어 (뷰티누리, 뷰티한국, WWD Korea)
- 뷰티 인플루언서 / 유튜버 콜라보
- COSMOBEAUTY SEOUL 등 박람회 네트워크

## 5단계 론칭 프레임워크

### Phase 1: 내부 론칭 (D-30)

- 영업팀, 생산팀에 먼저 공유
- 내부 FAQ 작성 (예상 질문 미리 준비)
- 블로그 + 랜딩 페이지 초안 완성

### Phase 2: 알파 론칭 (D-21)

- 기존 최상위 파트너 5–10개사에 사전 공개
- 피드백 수집 → 핵심 메시지 수정
- 케이스 스터디 파트너 확보

### Phase 3: 베타 론칭 (D-14)

- LinkedIn 티저 포스트 시작 ("곧 공개됩니다")
- 샘플 또는 사전 신청 페이지 오픈
- 이메일 리스트에 사전 공지

### Phase 4: 얼리 액세스 (D-7)

- LinkedIn + 블로그 정식 발표
- 이메일 발송: 기존 파트너 + 리드 리스트
- 업계 미디어 보도자료 발송
- "얼리 파트너" 혜택 제공 (예: 첫 3개월 우선 납기)

### Phase 5: 풀 론칭 (D-Day)

- 전체 채널 동시 발행
- LinkedIn: 창업자 개인 포스트 + 회사 페이지
- 블로그: 상세 발표 포스트
- 이메일: 전체 리스트 발송
- 파트너 브랜드들의 공유 유도

## 론칭 체크리스트

### 론칭 전 (Pre-Launch)

- [ ] 랜딩 페이지 완성 + 모바일 테스트
- [ ] CTA 폼 작동 확인 + 자동 응답 이메일 설정
- [ ] 블로그 포스트 초안 완성 (예약 발행 설정)
- [ ] LinkedIn 포스트 3–5개 초안 준비
- [ ] 보도자료 준비 (업계 미디어 3–5곳 리스트)
- [ ] 기존 파트너에게 "공유해주세요" 요청 메시지 준비
- [ ] UTM 파라미터 세팅 (채널별 성과 추적)

### 론칭 당일 (Launch Day)

- [ ] 오전 9시: LinkedIn 메인 포스트 발행
- [ ] 오전 9시: 블로그 포스트 발행
- [ ] 오전 9시 30분: 이메일 발송
- [ ] 오전 10시: 보도자료 발송
- [ ] 종일: 댓글/DM 즉시 응답 (첫 60분이 알고리즘 결정)
- [ ] 오후: 파트너 공유 확인 + 감사 메시지

### 론칭 후 (Post-Launch)

- [ ] 3일 후: 성과 수치 공유 포스트 ("첫 3일 반응")
- [ ] 1주일 후: FAQ 기반 후속 콘텐츠
- [ ] 1개월 후: 초기 파트너 성공 사례 공유
- [ ] 분기 후: 론칭 회고 문서화

## 업계 특화 전략

### COSMOBEAUTY SEOUL / 박람회 연계

- 박람회 전 2주: "부스에서 만나세요" 티저 캠페인
- 박람회 기간: 라이브 업데이트, 현장 사진/영상
- 박람회 후 1주: 미팅 후속 + 신규 리드 이메일 시퀀스

### 시즌별 파트너 모집 캠페인

- 봄 시즌 (1–2월): 여름 컬렉션 OEM 파트너 모집
- 가을 시즌 (7–8월): 홀리데이 컬렉션 파트너 모집
- 상시: "소량 파트너십" 프로그램 (인디 브랜드 대상)

### 레퍼럴 프로그램

- 기존 파트너 브랜드가 신규 브랜드 소개 → 인센티브 (우선 납기, 단가 혜택)
- B2B는 입소문(Word of Mouth)이 가장 강력한 획득 채널
