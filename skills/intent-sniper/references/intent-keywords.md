# Intent Keywords Reference

## High-Intent Patterns (Score 5 — Reply Immediately)

```
"looking for a [category] tool"
"anyone recommend [category]"
"what's the best [category]"
"need a [category] solution"
"shopping for [category]"
"evaluating [category] tools"
"which [category] should I use"
"help me choose [category]"
```

## Medium-High Intent (Score 4 — Reply with Solution)

```
"alternative to [competitor]"
"switching from [competitor]"
"[competitor] is too expensive"
"[competitor] doesn't have [feature]"
"frustrated with [competitor]"
"[competitor] support is terrible"
"outgrew [competitor]"
"[competitor] vs [other competitor]"
```

## Medium Intent (Score 3 — Reply if Very Relevant)

```
"how to [task your product does]"
"best way to [task]"
"struggling with [problem]"
"tips for [problem area]"
"how do you handle [problem]"
"workflow for [task]"
"automate [task]"
```

## Low Intent (Score 2 — Skip Usually)

```
"interesting article about [topic]"
"the future of [category]"
"[category] trends in [year]"
"thoughts on [broad topic]"
```

## Negative Keywords (Never Reply)

```
"hiring" (unless job-sniper agent)
"meme" / "joke" / "lol"
"[political topic]"
"[controversial topic]"
"spam" / "scam" / "fake"
"[competitor] is amazing" (don't attack happy customers)
```

## Platform-Specific Search Queries

### Reddit

```
site:reddit.com "best [category]" -site:reddit.com/r/AskReddit
site:reddit.com "alternative to [competitor]"
subreddit:[niche] "looking for" OR "recommend" OR "best"
```

### Quora

```
site:quora.com "best [category] tool"
site:quora.com "alternative to [competitor]"
site:quora.com "how to [task]"
```

### X (Twitter)

```
"[category] recommendations" -is:retweet
"alternative to [competitor]" -is:retweet
"best [category]" min_replies:2
"need help with [problem]" -is:retweet
```

### Hacker News

```
site:news.ycombinator.com "Ask HN" "[category]"
site:news.ycombinator.com "Show HN" "[competitor]"
```
