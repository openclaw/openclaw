# Social Content

## Enth Context

- **Company:** Enth — 대한민국 1위 네일 제조회사
- **Target Audience:** B2B 뷰티 네일 브랜드사 (구매팀, 브랜드 매니저, 창업자)
- **Primary Channel:** LinkedIn (B2B 의사결정자 다수)
- **Secondary Channel:** Blog (Ghost CMS — SEO 목적, 교육 콘텐츠)
- **Content Goal:** 브랜드 인지 → 전문성 신뢰 → 파트너십 문의 유도

## 플랫폼별 전략

### LinkedIn (최우선)

| 항목          | 가이드                                     |
| ------------- | ------------------------------------------ |
| 빈도          | 주 3–4회                                   |
| 베스트 시간   | 화–목 오전 8–10시, 오후 5–7시              |
| 베스트 포맷   | 텍스트 포스트, 문서 카루셀, 짧은 영상      |
| 팔로우 목표   | 뷰티 브랜드 대표, 구매 담당자, 업계 미디어 |
| 알고리즘 핵심 | 처음 60분 인게이지먼트가 도달 범위 결정    |

**LinkedIn 콘텐츠 유형:**

1. 업계 인사이트 (트렌드, 시장 데이터)
2. 제조 역량 소개 (공장 투어, 생산 공정, 품질 관리)
3. 파트너 성공 사례 (익명 또는 동의 후 공개)
4. 대표/팀의 개인 스토리 (신뢰 구축)
5. Q&A / 교육 콘텐츠 (OEM 프로세스 설명)

### Blog / Ghost CMS (SEO 목적)

| 항목 | 가이드                                        |
| ---- | --------------------------------------------- |
| 빈도 | 주 1–2회                                      |
| 길이 | 1,500–3,000자 (필러 콘텐츠), 500–800자 (뉴스) |
| 목적 | 검색 트래픽 획득, LinkedIn 발행 소스          |
| 포맷 | 긴 형식 가이드, 트렌드 리포트, OEM 교육       |

## Enth 콘텐츠 필러 5가지

```
필러 1: 제조 전문성
  — 네일 제조 공정, 원료, 품질 관리, R&D
  — 예: "젤 네일의 점도 비밀: OEM 제조사가 말하지 않는 것"

필러 2: 뷰티 트렌드 리포트
  — 시즌별 네일 트렌드, 컬러 트렌드, 글로벌 시장 동향
  — 예: "2026 S/S 네일 컬러 트렌드 10선 — 제조사 관점"

필러 3: OEM/ODM 교육
  — 처음 네일 브랜드 론칭하는 법, OEM vs ODM 차이, MOQ 이해하기
  — 예: "네일 브랜드 창업자를 위한 OEM 완전 가이드"

필러 4: 파트너십 스토리
  — 브랜드 성장 사례, 콜라보 성공 스토리
  — 예: "소규모 인디 브랜드에서 편집샵 입점까지 — Enth와 함께한 18개월"

필러 5: 업계 뉴스 & 해설
  — 뷰티 산업 규제, 성분 트렌드, 글로벌 시장 뉴스
  — 예: "EU 네일 규제 변경 — 한국 OEM 브랜드가 알아야 할 것"
```

## 훅 공식 (Hook Formulas)

**호기심 훅:**

> "대부분의 네일 브랜드가 OEM 파트너를 고를 때 놓치는 한 가지"

**스토리 훅:**

> "3년 전, 한 작은 인디 브랜드가 처음 Enth에 연락했을 때..."

**가치 훅:**

> "네일 OEM 단가를 30% 낮추는 3가지 방법 (제조사 입장에서)"

**역발상 훅:**

> "샘플이 무료라고 좋은 제조사가 아닙니다. 이유는..."

**숫자 훅:**

> "Enth가 연간 500만 개를 생산하면서 배운 5가지 교훈"

## 콘텐츠 배치 전략 (Batching)

**주간 루틴 (2–3시간):**

- 월요일: 이번 주 포스트 3–4개 초안 작성
- 화요일: 편집 + 이미지/카루셀 제작
- 수요일 오전: LinkedIn 포스트 1 발행
- 목요일 오전: LinkedIn 포스트 2 발행
- 금요일: 블로그 포스트 발행 + LinkedIn 공유

**콘텐츠 재활용 시스템:**

```
블로그 포스트 (1,500자)
  → LinkedIn 텍스트 포스트 (요약, 3–5 인사이트)
  → LinkedIn 카루셀 (핵심 포인트 슬라이드)
  → LinkedIn 짧은 영상 스크립트 (60초)
  → 뉴스레터 섹션
```

## 인게이지먼트 전략 (LinkedIn)

**일일 30분 루틴:**

1. 내 포스트의 댓글에 답글 (10분)
2. 타겟 브랜드 관련 포스트에 의미있는 댓글 (10분)
3. 팔로우할 신규 뷰티 브랜드 담당자 찾기 + 연결 (10분)

**DM 전략:**

- 먼저 콘텐츠에 가치를 줬을 때만 DM 접근
- 첫 DM: 판매 금지, 공통 관심사 또는 그들의 콘텐츠에 반응
- 예: "[포스트] 잘 읽었습니다. 저희도 비슷한 고민을 하고 있어서..."

## 핵심 지표

| 단계             | 지표                                           |
| ---------------- | ---------------------------------------------- |
| 인지 (Awareness) | 팔로워 증가율, 포스트 도달 범위, 블로그 방문자 |
| 인게이지먼트     | 좋아요/댓글/공유율, 평균 체류 시간             |
| 전환             | 파트너십 문의 수, 샘플 요청 수, CTA 클릭율     |
