# Chrome Extension - Amazon Research Tools

The Helium 10 Chrome Extension provides on-page analysis while browsing Amazon.

## Installation
1. Chrome Web Store → Search "Helium 10"
2. Click "Add to Chrome"
3. Log in with Helium 10 account
4. Extension icon appears in toolbar

## Accessing Tools

Click the Helium 10 icon in Chrome toolbar while on Amazon to see available tools.

## Xray - Market Analysis

**Purpose**: Analyze entire search results page or product category at once.

### How to Use
1. Search for a product on Amazon
2. Click Helium 10 extension → Xray
3. View aggregated data for all visible products

### Data Provided
| Metric | Description |
|--------|-------------|
| Revenue | Estimated monthly revenue |
| Price | Current listing price |
| BSR | Best Seller Rank |
| Reviews | Review count |
| Rating | Star rating |
| FBA Fees | Estimated fulfillment fees |
| Net | Estimated net profit |
| Sales | Monthly unit sales estimate |

### Features
- **Sort by any column** - Find top performers
- **Filter sponsored/organic** - Separate paid vs organic
- **Sales charts** - Click product for historical trends
- **Quick Cerebro** - "Keywords" link opens Cerebro for that ASIN
- **Multi-select** - Check multiple ASINs → Run keyword search

### Xray Keywords
View keyword data directly from search results:
1. Run Xray on search page
2. Click "Xray Keywords" tab
3. See keywords driving traffic to those results

## ASIN Grabber

**Purpose**: Bulk extract ASINs from search results.

### How to Use
1. Search on Amazon
2. Click extension → ASIN Grabber
3. Extracts up to 100 ASINs with data

### Output Data
- ASIN
- Search rank position
- Current price
- BSR
- Review count

### Use Cases
- Build competitor ASIN lists for Cerebro
- Track seasonal/trending products
- Export for spreadsheet analysis

## Profitability Calculator

**Purpose**: Instant margin calculation for any product.

### How to Use
1. Open any Amazon product page
2. Extension auto-displays profitability widget at page bottom
3. Enter your costs

### Inputs
- Manufacturing cost (per unit)
- Freight/shipping cost
- Fulfillment cost (if not FBA)

### Outputs
- Amazon referral fees
- FBA fees
- Net profit per unit
- Profit margin %
- ROI %

### Tips
- Auto-pulls dimensions and weight from listing
- Compare FBA vs FBM scenarios
- Factor in returns (typically 2-5%)

## Review Insights

**Purpose**: Analyze product reviews for market intelligence.

### How to Use
1. Open product page on Amazon
2. Click extension → Review Insights
3. View analysis of all reviews

### Features
| Feature | Use |
|---------|-----|
| Star distribution | Quality overview |
| Common phrases | What customers mention most |
| Export reviews | Download for analysis |
| Review Analysis | Word frequency across reviews |
| Product Variations | Compare reviews by variant |
| Questions tab | Common customer questions |

### Product Research Use
- Negative reviews reveal product improvement opportunities
- Common complaints = differentiation points
- Questions reveal information gaps in market

## Demand Analyzer

**Purpose**: Validate product demand from Alibaba.

### How to Use
1. Go to Alibaba.com
2. Search for product
3. Click "Analyze Product Demand on Amazon"
4. Enter keyword to analyze

### Output
- Search volume
- Top products
- Keyword suggestions
- Links to deeper Helium 10 tools

## Supplier Finder

**Purpose**: Find Alibaba suppliers for Amazon products.

### How to Use
1. Open Amazon product page
2. Run Xray
3. Click "Find Suppliers on Alibaba"

### Filters
- Country
- Vendor type
- Vendor age
- SGS verification
- Rating

## Search Expander

**Purpose**: Discover keywords while typing in Amazon search.

### How It Works
- Start typing in Amazon search bar
- Extension shows expanded keyword suggestions
- View search volume for each suggestion
- See historical trends

## Settings & Configuration

### Access Settings
Click extension icon → Gear icon

### Key Settings
- **Show Xray automatically**: Auto-open on search pages
- **Show profitability calculator**: Auto-display on product pages
- **Default marketplace**: Set your primary market
- **Data refresh**: How often to update cached data

## Tips

1. **Keep extension active** - Data loads as you browse naturally
2. **Use quick filters** - Sort Xray results to find patterns
3. **Bookmark ASIN lists** - Export frequently for later research
4. **Combine with web tools** - Extension for quick looks, web app for deep dives
5. **Check multiple variations** - Click through product variants for full picture
