## Project Context

This project uses OpenClaw. Key services: OpenClaw gateway (not NanoClaw). Config files: always verify the correct config location before editing API keys. Restart command: confirm the gateway binary is in PATH before suggesting restart commands.

## Environment

When working on Windows, always ask which shell environment the user is in (PowerShell, Git Bash, CMD, or WSL2) before providing commands. Do not assume a Unix environment.
