# 学习笔记

这个目录用于存放 OpenClaw 学习过程中的笔记和流程文档。

## 文件说明

- roadmap.md: 学习路线与阶段目标。
- study-log.md: 每次学习记录。
- process-template.md: 流程梳理模板。

## 记录约定

- 说明文字使用中文。
- 调用函数请保留原始函数名，不翻译、不改写（例如 runEmbeddedAgent）。

## 建议使用方式

1. 先在 roadmap.md 明确阶段目标。
2. 每次学习后在 study-log.md 增加一条记录。
3. 需要梳理完整链路时，使用 process-template.md。
