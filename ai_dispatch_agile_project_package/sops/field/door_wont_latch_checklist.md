# Field SOP — Door Misalignment / Won’t Latch

## Before you start
- Check site safety (egress requirements, ADA)
- Confirm authorized contact on-site
- Confirm scope + NTE

## Checklist (required)
- [ ] Photo BEFORE: door edge + strike alignment
- [ ] Verify hinge condition (loose screws, sag)
- [ ] Verify strike plate alignment
- [ ] Adjust as needed (document)
- [ ] Test latch engagement 10 cycles
- [ ] Photo AFTER: door latched + alignment
- [ ] Note any recommended follow-up (replacement hinges/strike)

## Completion package requirements
- Before + after photos attached
- Completion notes include what changed and tests performed
- Signature OR explicit “no signature reason”
