## Declaration

Any automated decision made by an OpenClaw agent about or affecting an individual must pass the Burgess Principle binary test before execution.

This repository's governance materials formalize OpenClaw's adoption of that framework and should be read together with the canonical source at <https://github.com/ljbudgie/burgess-principle>.
