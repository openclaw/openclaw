# Архітектурна Специфікація: ЗІРКОВЕ НЕБО ПАМ'ЯТІ v3.0

## (Проект "Другий Мозок" — memory-hybrid)

> Статус: **Етапи 1–3 РЕАЛІЗОВАНІ** ✅ | Дата: 2026-03-11

---

## 🌌 Філософія

Система `memory-hybrid` — це когнітивна архітектура для ШІ-агентів, натхненна нейронаукою (гіпокамп → неокортекс) та дослідженнями MemGPT (UC Berkeley, 2023) і рекурсивної сумаризації (Wang et al., 2023).

Три рівні, як у справжнього мозку:

1. **Зірки (Радар):** Легковагові summary (1 речення) всіх спогадів. Агент ЗАВЖДИ їх бачить.
2. **Телескоп:** Інструмент `memory_fetch_details` — Агент може витягти повний текст конкретного спогаду за ID.
3. **Рефлексія:** Meta-аналіз всіх спогадів — психологічний профіль користувача.

---

## 🛠 Реалізовані Компоненти

### Етап 1 ✅ — Виправлення Бази

- Обмеження результатів пошуку через `slice(0, limit)` в AMHR
- Захист від переповнення контексту 15K

### Етап 2 ✅ — Фабрика Зірок

- Функція `generateMemorySummary()` в `capture.ts`
- Поле `summary` в схемі LanceDB
- Автогенерація summary при кожному збереженні факту

### Етап 3 ✅ — Радар + Телескоп + Rolling Summary

- **Радар:** `before_agent_start` інжектить `<star-map>` замість повних текстів
- **Телескоп:** Інструмент `memory_fetch_details(ids: [...])` — повний текст за ID
- **Rolling Summary Stack:** `stack.ts` — стиснення кожного кроку розмови до ~30 слів

## 📊 Формула Гібридного Скорінгу (7 каналів)

```
Score = 0.42·Вектор + 0.10·Новизна + 0.16·Важливість + 0.08·Граф + 0.08·Підкріплення + 0.10·Темпоральність + 0.06·Емоційність
```
