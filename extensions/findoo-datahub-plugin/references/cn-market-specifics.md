# A-Share Market Specifics

A 股市场独有规则与分析维度。通用个股分析见 equity skill，本文档聚焦 A 股特色。

---

## 1. 质押风险分级

| 质押比例 | 风险     | 说明                 |
| -------- | -------- | -------------------- |
| <10%     | 安全     |                      |
| 10-30%   | 需关注   |                      |
| 30-50%   | 高风险   | 股价下跌可能触发平仓 |
| >50%     | 极高风险 | 随时可能爆仓         |

**平仓线逻辑:**

- 预警线 = 质押价格 x 160%
- 平仓线 = 质押价格 x 140%
- 连锁风险: 高质押 + 股价下跌 → 触发平仓 → 被迫卖出 → 进一步下跌 → 踩踏

**DataHub:** `fin_stock(endpoint="pledge/stat")`

## 2. 增减持信号矩阵

| 信号                 | 含义         | 风险 |
| -------------------- | ------------ | ---- |
| 大股东连续减持 >3 次 | 资金撤离     | 高   |
| 高管集中减持         | 内部人不看好 | 高   |
| 减持公告 + 股价高位  | 高位套现     | 高   |
| 减持后质押比例上升   | 资金链紧张   | 极高 |
| 大股东自掏腰包增持   | 看好前景     | 低   |
| 员工持股计划增持     | 利益绑定     | 低   |

**DataHub:** `fin_stock(endpoint="ownership/shareholder_trade")`

## 3. 回购类型解读

| 类型         | 解读                 | 积极程度   |
| ------------ | -------------------- | ---------- |
| 注销型回购   | 减少总股本，提升 EPS | 最积极     |
| 员工激励回购 | 用于股权激励         | 中性偏积极 |
| 市值管理回购 | 稳定股价             | 中性       |
| 回购但未注销 | 可能未来仍会卖出     | 中性偏消极 |

**力度评估:** 回购额/市值 >1% 力度大; 上限远高于现价 → 真诚度高; 进度 >50% → 执行力好

## 4. 解禁冲击评估

| 维度             | 高冲击             | 低冲击              |
| ---------------- | ------------------ | ------------------- |
| 解禁比例 vs 流通 | >10% 流通盘        | <3% 流通盘          |
| 股东类型         | 财务投资者 (PE/VC) | 控股股东/战略投资者 |
| 持有成本 vs 现价 | 成本远低于现价     | 成本接近或高于现价  |
| 历史减持行为     | 有减持习惯         | 从未减持            |

**减持时间规律:**

- 解禁当日通常不大量卖出 (大宗交易需报备)
- 解禁后 1-2 周: 减持高峰
- 大股东: 90 天内减持 <=1% (集合竞价)

**DataHub:** `fin_stock(endpoint="ownership/share_float")`

## 5. 板块轮动

### 典型轮动路径

**牛市:** 券商 → 周期 (有色/化工) → 消费 (白酒/医药) → 科技 (半导体/软件) → 小盘/垃圾股

**结构性行情:** 政策催化板块 → 业绩兑现板块 → 低位补涨板块

### 轮动信号

| 信号            | 含义         | 操作        |
| --------------- | ------------ | ----------- |
| 板块启动 3 日内 | 主升浪初期   | 关注龙头    |
| 连涨 >5 日      | 主升浪中期   | 跟随但控仓  |
| 高位放量滞涨    | 可能见顶     | 减仓/换板块 |
| 缩量回调        | 正常调整     | 逢低布局    |
| 补涨股开始表现  | 行情接近尾声 | 准备切换    |

### 识别工具链

```
Step 1: fin_index(endpoint="thematic/ths_index") → 全部概念列表
Step 2: 选取 10-15 核心概念 → fin_index(endpoint="thematic/ths_daily")
Step 3: 计算 5 日/20 日涨跌幅排名变化
Step 4: 排名上升最快的概念 = 当前轮动方向
```

## 6. 政策敏感度

| 政策类型 | 影响方向 | 影响行业            | 持续    |
| -------- | -------- | ------------------- | ------- |
| 货币宽松 | 利好     | 券商/地产/有色      | 1-3 月  |
| 财政刺激 | 利好     | 基建/建材/机械      | 3-6 月  |

## 7. 打新策略

### A 股打新特点

- 市值门槛: 沪市 1 万、深市 5000 (T-2 日市值)
- 中签率: 通常 0.02%-0.05% (科创板略高)
- 破发风险: 注册制 IPO 约 10-20% 首日破发

### 打新策略矩阵

| 维度     | 优先打    | 谨慎打       | 回避          |
| -------- | --------- | ------------ | ------------- |
| 发行 PE  | <行业均值 | 接近行业均值 | >行业均值 50% |
| 规模     | 5-20 亿   | <2 亿        | >50 亿        |
| 行业景气 | 高景气    | 一般         | 衰退          |
| ROE      | >15%      | 10-15%       | <10%          |

### 中签后策略

- 核准制 (主板): 开板后观察 1-2 天，放量滞涨卖出
- 注册制 (科创/创业): 首日可卖; >100% 分批卖; 30-100% 次日观察; 破发评估基本面

## 9. CAS 会计特殊性

- CAS "营业利润" 包含投资收益、公允价值变动，与 IFRS Operating Profit 口径不同
- 证监会强制披露 "扣非归母净利润"，IFRS 无此概念
- 政府补贴计入 "其他收益" 或 "营业外收入"，IFRS 通常扣除后列报
- 研发资本化条件执行宽松，可能高估资产

详见 `references/financial-ratios-cn.md` CAS vs IFRS 差异速查表。
