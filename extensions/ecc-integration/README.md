# Everything-Claude-Code Integration for OpenClaw

A hybrid AI agent system combining ECC's expertise with OpenClaw's operational capabilities.

## Three Core Rules

1. **Rules > Freedom** - All agents operate under strict governance rules
2. **One Agent/One Task** - Each agent handles exactly one task at a time
3. **Claude Code Integration** - All operations backed by ECC knowledge and expertise

## Architecture

```
Mission Control (Mobile/Web)
    ↓
Governance Layer (Your 3 Rules)
    ↓
Hybrid Agent System
├── ECC Skills (Architecture, Security, Patterns)
└── OpenClaw Core (Channels, UI, Extensions)
    ↓
Self-Improvement Engine
```

## Quick Start

```bash
# Install dependencies
pnpm install

# Run governance setup
openclaw governance init

# Start agent system
openclaw agent run --mode hybrid
```

## Commands

- `/agent-task` - Create single-agent task
- `/agent-status` - View all agent states
- `/security-scan` - Run ECC AgentShield audit
- `/skill-create` - Generate skills from codebase
- `/instinct-status` - View learned patterns

## Configuration

Edit `.openclaw/governance.json` to customize rules and agent behavior.
