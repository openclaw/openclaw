# RCS OpenClaw channel

Official OpenClaw channel plugin for RCS Business Messaging.

RCS brings the conversational surfaces OpenClaw relies on to plain phone-number
messaging: delivery and read receipts, rich media and suggested actions, long
messages without SMS segment splitting, and a carrier-verified branded sender
identity. Twilio RCS Business Messaging is the initial supported provider; the
channel surface (`rcs` id, `rcs:+E164` addressing, pairing/allowlist policy,
status capture) is provider-neutral.

## Docs

See `docs/channels/rcs.md` in the OpenClaw repository, or the published docs at `https://docs.openclaw.ai/channels/rcs`.
