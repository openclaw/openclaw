# OpenClaw LLMRouter Provider

Official OpenClaw provider plugin for [LLMRouter](https://llmrouter.sh), an OpenAI-compatible
routing API that sends `model:"auto"` requests to the cheapest model meeting the query's needs.

Install from OpenClaw:

```bash
openclaw plugins install @openclaw/llmrouter-provider
openclaw gateway restart
```

See <https://docs.openclaw.ai/providers/llmrouter> for setup and configuration.
