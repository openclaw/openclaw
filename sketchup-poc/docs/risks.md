# Risks

## Teknik riskler

- SketchUp automation yüzeyi sürüme göre farklı davranabilir
- Büyük modellerde traversal yavaş olabilir
- Attribute dictionary verisi düzensiz olabilir
- Nested component/group yapıları normalize ederken edge-case çıkabilir

## Ürün riskleri

- Çok fazla ham veri çıkıp anlamlı sunum değeri düşebilir
- Kullanıcının beklediği şey yorum/analiz iken sistem sadece dump üretebilir

## Azaltma yaklaşımı

- Önce az ama kaliteli alanlar
- Önce summary değeri, sonra veri zenginliği
- Geometry export'ı sonraya bırak
- Manual refresh ile başla
