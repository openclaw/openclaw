# Orchestrator Notes

Ceviz tarafı snapshot'ı tüketir, doğrular ve özet üretir.

## Görevler

- Snapshot dosyasını oku
- SchemaVersion doğrula
- Sayısal özetleri çıkar
- Tag/component frekanslarını hesapla
- Markdown summary üret
- Basit uyarı kuralları çalıştır

## Örnek kullanıcı aksiyonları

- modeli tara
- snapshot yenile
- sahneleri özetle
- tag dağılımını göster
- seçili nesneyi özetle
- en çok tekrar eden component'leri göster
