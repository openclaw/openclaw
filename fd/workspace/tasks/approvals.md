# Pending Approvals

Actions waiting for DA's explicit approval before execution.

Approvals expire after 60 minutes. If no response is received, the
action is not executed and is moved back to the queue.

---

## How to Approve

Reply to the approval request with:
- `approve` — execute the action
- `deny` — cancel the action
