> **Note :** Le journal complet des modifications est maintenu en anglais pour des raisons de cohérence technique et de versionnement.

## À propos de ce document

Ce fichier documente tous les changements notables apportés à OpenClaw, organisés par version. Le format suit les [Directives de Journalisation des Changements](https://keepachangelog.com/fr/1.0.0/) et ce projet adhère au [Versionnement Sémantique](https://semver.org/lang/fr/).

## Catégories de changements

- **Added** (Ajouté) : nouvelles fonctionnalités
- **Changed** (Modifié) : changements dans les fonctionnalités existantes
- **Deprecated** (Déprécié) : fonctionnalités bientôt supprimées
- **Removed** (Supprimé) : fonctionnalités supprimées
- **Fixed** (Corrigé) : corrections de bugs
- **Security** (Sécurité) : changements liés à la sécurité

## Comment mettre à jour

Pour mettre à jour votre installation d'OpenClaw vers la dernière version :

```bash
# Mise à jour npm globale
npm update -g openclaw

# Ou avec pnpm
pnpm update -g openclaw

# Ensuite, lancez doctor pour vérifier la migration
openclaw doctor
```

Pour plus de détails sur les mises à jour, consultez le [Guide de mise à jour](https://docs.openclaw.ai/install/updating).
