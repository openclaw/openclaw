# PR Description: docs: add Chinese translation (README.zh.md)

## Overview
OpenClaw has a rapidly growing community of Chinese developers and users. To make the project more accessible and welcoming to the Chinese-speaking community, this PR introduces:

- **`README.zh.md`**: A comprehensive and high-quality Chinese translation of the main documentation. It covers the core vision, extensive channel support, and the recommended onboarding path.
- **Language Switcher**: Added a clear English/Chinese toggle at the top of the main `README.md` to improve discoverability for non-English speakers.

## Translation Details
- **Terminology**: Key technical concepts like *Gateway*, *Canvas*, *Skills*, and *Onboarding Wizard* have been translated into their standard and most intuitive Chinese equivalents.
- **Formatting**: Maintained identical visual structure and link integrity to ensure the Chinese version stays in sync with the original documentation's feel.

## Testing Conducted
- **Link Integrity**: Verified that all internal and external links in `README.zh.md` function correctly.
- **Rendering**: Manually reviewed the Markdown rendering to ensure badges, tables, and images are displayed as intended.
