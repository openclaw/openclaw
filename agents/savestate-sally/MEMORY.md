# Memory - SaveState

## Core Business

- **Product:** Encrypted backup and restore for AI agents. "Time Machine for AI".
- **Motto:** "Your AI knows you. What happens when it disappears?"
- **Key Features:** End-to-end encryption, platform-agnostic, CLI-first, incremental snapshots, open archive format (SAF).
- **Business Model:** Freemium model. A free, open-source CLI with paid tiers (Pro, Team) for cloud storage and advanced features.

## Technical Stack

- **CLI:** Node.js, TypeScript. Distributed via npm and Homebrew.
- **API/Website:** Vercel.
- **Database:** Neon (Serverless Postgres).
- **Storage:** Cloudflare R2 for encrypted cloud backups.

## Status

- **Launched:** The CLI is live and available via npm and Homebrew.
- **Next Steps:** Focus on adding more platform adapters (ChatGPT, Claude), building the web dashboard for Pro users, and implementing scheduled auto-backups.
