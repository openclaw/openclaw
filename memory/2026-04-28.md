# 2026-04-28

- **CevizWatch (Apple Watch) Geliştirmeleri & Hata Çözümleri:**
  - Mert, Apple Developer Free hesabı aldı. Sideloadly ile yükleme yoluna gidildi.
  - Sideloadly'de alınan `%40` "InvalidWatchKitApp" kurulum hatası incelendi.
  - `Info.plist` içerisine modern watchOS 9+ için gerekli olan `<key>WKRunsIndependently</key><true/>` eklendi, eski `WKWatchKitApp` kaldırıldı.
  - Sideloadly'nin resign (yeniden imzalama) işlemi sırasında iOS ve WatchOS Bundle ID eşleşmesinin kopmaması için `project.yml` üzerinden dinamik bağlama (`$(IOS_BUNDLE_ID).watchkitapp`) yapıldı.
  - GitHub Actions (`ci.yml`), unsigned (imzasız) watch app çıktılarında hata vermeyecek (fail-fast kaldırıldı) şekilde esnetildi.
