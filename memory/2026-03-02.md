## 오늘 발생한 실수 / 주의사항

1. **BOM 제거 스크립트에서 `ReadAllText` 인코딩 주의** — Windows에서 기본 인코딩이 시스템에 따라 다를 수 있음. Python `pathlib.read_text(encoding='utf-8')`로 검증 필수
2. **프론트매터 수정 시 줄바꿈 보존** — `strip()` 사용 시 `---\n---`가 `------`로 합쳐질 수 있음. 수정 후 반드시 line-by-line 확인 필요
3. **Python 스크립트 직접 작성 후 실행** — 한국어/이모지 포함 코드는 PowerShell 인라인 `-c`로 실행 불가, 파일로 저장 후 실행
