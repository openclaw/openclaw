# Watch Ceviz Landing Scaffold

A small static landing page scaffold built from the current Watch Ceviz productization work.

## Files

- `index.html` — page structure and copy placement
- `styles.css` — visual system and layout rules

## Purpose

This is not the final production site.
It is a bridge from product docs to something implementation-ready.

## Current launch assumptions

- audience: founder-operator
- public plan: Personal only
- public billing: monthly only
- public price: $29/mo

## How to use

Open `index.html` directly in a browser.

## What this is good for

- checking whether the story reads clearly on a page
- reviewing section order
- testing whether the product feels focused or too generic
- handing off to design or frontend implementation later

## Natural next step

Turn this scaffold into either:

1. a real marketing page inside a frontend app
2. a higher-fidelity visual mock
3. a design-system-backed implementation
