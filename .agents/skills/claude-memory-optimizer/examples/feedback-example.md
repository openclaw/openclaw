# 回复风格

**规则：** 回复要简洁，不要在结尾总结刚做的事情

**Why:** 用户说"我可以自己读 diff"，不喜欢 AI 在结尾加"完成了 X、Y、Z"的总结

**How to apply:**

- 所有会话中，完成工作后直接结束
- 不要加"完成了 X、Y、Z"的总结
- 如果有重要更新，用简短的 bullet points

## 示例

❌ 不要这样：

```
好的，我已经帮你完成了记忆系统迁移！
1. 运行了重构脚本
2. 更新了 MEMORY.md
3. 创建了新的目录结构
如果有任何问题请告诉我~
```

✅ 要这样：

```
✅ 迁移完成！

下一步：
1. 检查 MEMORY.md 索引
2. 确认无误后删除旧文件
```
