# Xiaohongshu Single-Platform SOP (Phase 1)

## Goal

Run a reliable, repeatable growth loop on Xiaohongshu first. Expand to other platforms only after this SOP is stable.

## Workflow

1. Industry scan

- Inputs: trend signals, competitor content patterns, parent pain points.
- Output: prioritized opportunity list for today.

2. Topic and copy pack

- Inputs: opportunity list.
- Output: post-ready package (titles, body, tags, first comment, DM opener).

3. Compliance and conversion

- Inputs: draft copy + conversion scripts.
- Output: compliant A/B scripts and risk-marked lines.

4. Retention loop

- Inputs: incoming parent leads and Q&A.
- Output: follow-up cadence, referral script, churn prevention actions.

5. Night review

- Inputs: day metrics (impressions, engagement, inquiries, conversions).
- Output: next-day tactical adjustments and resource allocation.

## Exit Criteria for Expansion

Expand to a second platform only when all are true for 14 consecutive days:

- Daily SOP executed with no blocking errors.
- Human approval SLA met for high-risk items.
- Conversion signal is stable (no severe volatility).
- Compliance risk remains controlled.
