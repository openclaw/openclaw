# DNA Brand Guide

## Logo Design Brief

### Style Direction (Based on 2025/2026 Trends)

**Primary approach:** Modern minimalist with subtle personality

**Key trends we're incorporating:**
1. **Monoline design** — Single-weight strokes, clean and modern
2. **Geometric abstraction** — DNA concept as abstract shape, not literal helix
3. **Gradient mesh** — Subtle color transitions for depth
4. **Custom letterform** — Modified "DNA" typography with unique character

### Logo Concepts

**Concept 1: Abstract Double Helix**
- Two intertwining lines forming abstract "connection" shape
- Can be read as DNA strand or data streams merging
- Monoline, single color works but gradient adds depth

**Concept 2: Lettermark "D"**
- Stylized "D" with internal helix pattern
- Geometric, works at any size
- Modern tech aesthetic

**Concept 3: Wordmark "DNA"**
- Custom typography with connected letters
- The "N" bridges D and A like a neural connection
- Clean, memorable, scalable

**Recommended: Concept 3** — Wordmark with connected letters
- Most versatile
- Memorable
- Works at all sizes
- Reads clearly

### Color Palette

**Primary Colors (2026 Tech Trends):**

| Color | Hex | Use |
|-------|-----|-----|
| **Deep Violet** | `#6366F1` | Primary brand color |
| **Electric Indigo** | `#818CF8` | Gradients, accents |
| **Soft Lavender** | `#C7D2FE` | Backgrounds, light mode |
| **Near Black** | `#0F0D15` | Text, dark mode bg |
| **Pure White** | `#FFFFFF` | Light backgrounds |

**Why these colors:**
- Violet/Indigo trending for AI/tech in 2025-2026
- Differentiates from typical blue (too common)
- Works in dark and light modes
- Conveys innovation + trust

**Gradient:**
```css
background: linear-gradient(135deg, #6366F1 0%, #818CF8 50%, #A78BFA 100%);
```

### Typography

**Heading:** Inter (Variable) — Bold weight
**Body:** Inter (Variable) — Regular weight
**Code:** JetBrains Mono

*Why Inter:* Modern, highly legible, variable font support, free

## Voice & Tone

### Brand Voice
- **Confident** but not arrogant
- **Technical** but accessible
- **Direct** — no fluff
- **Helpful** — genuinely useful

### Writing Style
- Short sentences
- Active voice
- Avoid jargon unless necessary
- Use "you" — speak to the reader

### Examples

❌ "DNA leverages cutting-edge AI technology to facilitate seamless multi-platform communication experiences."

✅ "DNA connects your AI assistant to WhatsApp, Telegram, and more. One AI, everywhere you chat."
