## Pre-Production Notes

### ElevenLabs Voice Setup (Most Natural Settings)

**Recommended voice:** "Adam" or "Josh" (natural male) / "Rachel" (natural female)

**Settings for human-like output:**
```
Model: Eleven Multilingual v2
Stability: 0.50 (adds natural variation)
Similarity Boost: 0.75 (maintains voice character)
Style Exaggeration: 0 (most natural)
Speed: 1.0x (normal pace)
```

**Pro tips for natural sound:**
1. Add natural pauses with `<break time="0.5s"/>` in SSML
2. Use punctuation to control pacing (commas = short pause)
3. Write conversationally — contractions, simple words
4. Avoid long sentences (max 15-20 words)
5. Read script aloud first — if it sounds robotic when read, rewrite it
