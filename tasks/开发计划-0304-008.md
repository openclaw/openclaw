F-1: BacktestConfig 增加 regimeDetector

文件: extensions/findoo-trader-plugin/src/shared/types.ts:94-99

在 BacktestConfig 接口末尾添加可选字段:

export interface BacktestConfig {
capital: number;
commissionRate: number;
slippageBps: number;
market: MarketType;
/\*_ Optional regime detector. Defaults to "sideways" if absent. _/
regimeDetector?: { detect: (bars: OHLCV[]) => MarketRegime };
}

向后兼容: optional 字段，所有现有调用不受影响。

F-2: BacktestEngine 使用 regimeDetector

文件: extensions/findoo-trader-plugin/src/strategy/backtest-engine.ts:117

将:
regime: "sideways",

regime: config.regimeDetector?.detect(history) ?? "sideways",
