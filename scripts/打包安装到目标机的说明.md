1. 执行一下命令打包：
   pnpm install
   # 建议先构建一次，打包会走快路径（prepack 只校验 dist，不再跑 build/ui:build）：
   pnpm build && pnpm ui:build
   bash scripts/package-openclaw-mac.sh
   # 若 dist 不完整，脚本会自动先 build + ui:build 再打包。
   # 强制每次都让 prepack 全量构建（慢）：OPENCLAW_MAC_PACK_FORCE_PREPACK_BUILD=1 bash scripts/package-openclaw-mac.sh
2. 然后讲生成的文件 .artifacts/openclaw-mac-bundles/\*.tar.gz 拷贝到目标机上。
3. 在目标机上接下这个压缩包，进入解压后的文件夹 执行一下命令
   bash install-local-openclaw.sh
4. 打开终端执行一下命令启动 openclaw：
   openclaw gateway install
   openclaw gateway start
