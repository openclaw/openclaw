## 지시 사항
1. 먼저 `agent-01-lin.md` ~ `agent-05-yuri.md` 5개 파일을 전부 읽어
2. 각 작업의 상세 내용을 파악해
3. Phase 1(린 버그 수정)부터 직접 실행 시작
4. Phase 2는 가능한 범위에서 병렬 처리 (백그라운드 에이전트 활용)
5. Phase 3(예린 검수)는 Phase 1~2 완료 후 실행
6. 스킬 확인 + 브랜치 정리는 위 작업과 병렬로 진행 가능
7. 매 단계 완료 시 `.claude/context/report-to-boss.md`에 진행 보고 작성
