<objective>
Create the heartbeat scanner that discovers, filters, and claims tasks from queue.md on agent heartbeat.

Purpose: This is the core autonomous task pickup engine. On each heartbeat, agents need to (1) check for an active task to resume, (2) if idle, scan queue.md for claimable tasks matching their capabilities with satisfied dependencies, (3) claim the highest-priority match. This implements AGNT-05, AGNT-06, AGNT-08, AGNT-09.

Output: `src/projects/heartbeat-scanner.ts` with scanAndClaimTask function. Full TDD test coverage.
