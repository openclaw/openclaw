       ```typescript
       export interface CheckpointData {
         status: "in-progress" | "review" | "done" | "blocked";
         claimed_by: string;
         claimed_at: string;
         last_step: string;
         next_action: string;
         progress_pct: number;
         files_modified: string[];
         failed_approaches: Array<{ approach: string; reason: string }>;
         log: Array<{ timestamp: string; agent: string; action: string }>;
         notes: string;
       }
       ```

    2. **checkpointPath(taskFilePath: string): string** — replaces `.md` extension with `.checkpoint.json`. Example: `tasks/TASK-005.md` -> `tasks/TASK-005.checkpoint.json`.

    3. **createCheckpoint(opts: { agentId: string; taskId: string; timestamp?: string }): CheckpointData** — returns initial checkpoint with:
       - status: "in-progress"
       - claimed_by: opts.agentId
       - claimed_at: opts.timestamp ?? new Date().toISOString()
       - last_step: "" (empty)
       - next_action: "" (empty)
       - progress_pct: 0
       - files_modified: []
       - failed_approaches: []
       - log: [{ timestamp: claimed_at, agent: opts.agentId, action: "Claimed task" }]
       - notes: ""

    4. **writeCheckpoint(filePath: string, data: CheckpointData): Promise<void>** — atomic write: write to `${filePath}.tmp`, then rename to filePath. Use `import { writeFile, rename } from "node:fs/promises"`.

    5. **readCheckpoint(filePath: string): Promise<CheckpointData | null>** — read and JSON.parse. Return null on ENOENT (file not found). Return null on SyntaxError (corrupted JSON). Log a warning via `import { log } from "../log.js"` when corrupted. Do NOT throw.
