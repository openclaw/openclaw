# HEARTBEAT

Run this checklist every 30 minutes during active hours (8 AM – 10 PM):

1. Check /data/tasks.md — if any task is past its due date, send a reminder via the connected channel with the task name and how overdue it is.
2. Check /data/daily-log.md — if any flagged item has been unresolved for more than 4 hours, escalate with a concise reminder.
3. If any meeting is within the next 90 minutes and /data/meetings/ has no prep note for it, draft a one-paragraph briefing and send it.

If nothing requires attention: reply HEARTBEAT_OK
