# HEARTBEAT

Run this checklist every 30 minutes during active hours (8 AM – 7 PM):

1. Check /data/candidates.md — identify any candidates awaiting follow-up or scheduling. For each overdue action (more than 2 business days), send a reminder with the candidate name, role, and action needed.
2. Check /data/roles.md if it exists — if any role has been open for more than 30 days without a hire, send a flag with the role name and days open.
3. On Fridays after 3 PM, if the Pipeline Update hasn't been sent, compile and send it.

If nothing requires attention: reply HEARTBEAT_OK
