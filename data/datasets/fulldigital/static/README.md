# Full Digital — Static Visuals Dataset

## Naming Convention
<brand>__<category>__<subtype>__<platform>__<WxH>__v###.<ext>

Examples:
- fulldigital__ad__promo__igfeed__1080x1350__v001.png
- fulldigital__infographic__pricing__web__1920x1080__v003.png
- fulldigital__ui__dashboard__web__1920x1080__v002.png

## Categories
- ads/ — Ad creatives (1080x1350, 1080x1920, 1200x628)
- infographics/ — Infographic designs (1080x1350, 1920x1080)
- ui/ — UI screens and components (1440x900, 1920x1080)
- brand_assets/ — Logos, backgrounds, textures, patterns, badges, mockups

## Annotations
- captions/ — Text captions describing each asset
- bboxes/ — Optional bounding box annotations
- palette_checks/ — Validator output for palette compliance
