# استكشاف أخطاء القنوات

استخدم هذه الصفحة عندما تتصل القناة لكن يكون السلوك غير صحيح.

## سلّم الأوامر

شغّل ما يلي بالترتيب أولًا:

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

خط أساس سليم:

- `Runtime: running`
- `RPC probe: ok`
- يُظهر فحص القناة أنها متصلة/جاهزة

## WhatsApp

### بصمات فشل WhatsApp

| الأعراض                                   | أسرع فحص                                             | الإصلاح                                                                          |
| ----------------------------------------- | ---------------------------------------------------- | -------------------------------------------------------------------------------- |
| متصل لكن لا توجد ردود على الرسائل الخاصة  | `openclaw pairing list whatsapp`                     | الموافقة على المرسِل أو تبديل سياسة الرسائل الخاصة/قائمة السماح. |
| تجاهل رسائل المجموعات                     | تحقّق من `requireMention` + أنماط الذِّكر في التهيئة | ذِكر البوت أو تخفيف سياسة الذِّكر لتلك المجموعة.                 |
| انقطاعات عشوائية/حلقات إعادة تسجيل الدخول | `openclaw channels status --probe` + السجلات         | أعد تسجيل الدخول وتحقق من سلامة دليل بيانات الاعتماد.            |

استكشاف الأخطاء الكامل: [/channels/whatsapp#troubleshooting-quick](/channels/whatsapp#troubleshooting-quick)

## Telegram

### بصمات فشل Telegram

| الأعراض                                    | أسرع فحص                                         | الإصلاح                                                              |
| ------------------------------------------ | ------------------------------------------------ | -------------------------------------------------------------------- |
| `/start` لكن دون تدفّق ردود قابل للاستخدام | `openclaw pairing list telegram`                 | الموافقة على الاقتران أو تغيير سياسة الرسائل الخاصة. |
| البوت متصل لكن المجموعة صامتة              | تحقّق من متطلبات الذِّكر ووضع خصوصية البوت       | عطّل وضع الخصوصية لظهور المجموعة أو اذكر البوت.      |
| فشل الإرسال مع أخطاء شبكة                  | افحص السجلات بحثًا عن فشل استدعاءات Telegram API | أصلِح توجيه DNS/IPv6/الوكيل إلى `api.telegram.org`.  |

استكشاف الأخطاء الكامل: [/channels/telegram#troubleshooting](/channels/telegram#troubleshooting)

## Discord

### بصمات فشل Discord

| الأعراض                            | أسرع فحص                                      | الإصلاح                                                                           |
| ---------------------------------- | --------------------------------------------- | --------------------------------------------------------------------------------- |
| البوت متصل لكن لا يرد داخل الخوادم | `openclaw channels status --probe`            | السماح للخادم/القناة والتحقق من نية محتوى الرسائل.                |
| تجاهل رسائل المجموعات              | تحقق من السجلات للحصول على نقاط بوابة الإشارة | اذكر البوت أو اضبط إعدادات الخادم/القناة `requireMention: false`. |
| إجابات DM مفقودة                   | `openclaw pairing list discord`               | الموافقة على إقران أو تعديل سياسة DM.                             |

استكشاف الأخطاء الكامل: [/channels/discord#troubleshooting](/channels/discord#troubleshooting)

## Slack

### بصمات فشل Slack

| الأعراض                          | أسرع فحص                                   | الإصلاح                                                           |
| -------------------------------- | ------------------------------------------ | ----------------------------------------------------------------- |
| وضع Socket متصل لكن دون استجابات | `openclaw channels status --probe`         | تحقق من رمز التطبيق + رمز بوت والنطاقات المطلوبة. |
| تم حظر DMS                       | `openclaw pairing list slack`              | الموافقة على سياسة الإقران أو الاسترخاء DM.       |
| تجاهل رسالة قناة                 | تحقّق من `groupPolicy` وقائمة سماح القنوات | اسمح بالقناة أو بدّل السياسة إلى `open`.          |

استكشاف الأخطاء الكامل: [/channels/slack#troubleshooting](/channels/slack#troubleshooting)

## iMessage و BlueBubbles

### بصمات فشل iMessage و BlueBubbles

| الأعراض                                     | أسرع فحص                                                                | الإصلاح                                                        |
| ------------------------------------------- | ----------------------------------------------------------------------- | -------------------------------------------------------------- |
