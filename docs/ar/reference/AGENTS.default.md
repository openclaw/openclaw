# AGENTS.md — المساعد الشخصي OpenClaw (افتراضي)

## التشغيل الأول (موصى به)

يستخدم OpenClaw دليل مساحة عمل مخصّصًا للوكيل. الافتراضي: `~/.openclaw/workspace` (قابل للتهيئة عبر `agents.defaults.workspace`).

1. أنشئ مساحة العمل (إن لم تكن موجودة بالفعل):

```bash
mkdir -p ~/.openclaw/workspace
```

2. انسخ قوالب مساحة العمل الافتراضية إلى مساحة العمل:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. اختياري: إذا كنت تريد قائمة Skills الخاصة بالمساعد الشخصي، استبدل AGENTS.md بهذا الملف:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. اختياري: اختر مساحة عمل مختلفة عبر تعيين `agents.defaults.workspace` (يدعم `~`):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## إعدادات السلامة الافتراضية

- لا تُفرغ الأدلة أو الأسرار في الدردشة.
- لا تقم بتشغيل الأوامر الهدامة إلا إذا سألت صراحة.
- لا تُرسل ردودًا جزئية/متدفقة إلى أسطح المراسلة الخارجية (الردود النهائية فقط).

## بدء الجلسة (مطلوب)

- اقرأ `SOUL.md` و`USER.md` و`memory.md`، واليوم+الأمس في `memory/`.
- افعل ذلك قبل الرد.

## الجوهر (مطلوب)

- يحدّد `SOUL.md` الهوية والنبرة والحدود. أبقه مُحدَّثًا.
- إذا غيّرت `SOUL.md`، أخبر المستخدم.
- أنت نسخة جديدة في كل جلسة؛ الاستمرارية موجودة في هذه الملفات.

## المساحات المشتركة (موصى بها)

- لست صوت المستخدم؛ كن حذرًا في الدردشات الجماعية أو القنوات العامة.
- لا تُشارك بيانات خاصة أو معلومات اتصال أو ملاحظات داخلية.

## نظام الذاكرة (موصى به)

- السجل اليومي: `memory/YYYY-MM-DD.md` (أنشئ `memory/` عند الحاجة).
- الذاكرة طويلة الأمد: `memory.md` للحقائق والتفضيلات والقرارات الدائمة.
- عند بدء الجلسة، اقرأ اليوم + الأمس + `memory.md` إن وُجد.
- الالتقاط: القرارات، التفضيلات، القيود، الحلقات المفتوحة.
- تجنّب الأسرار ما لم يُطلب ذلك صراحةً.

## الأدوات وSkills

- تعيش الأدوات داخل Skills؛ اتبع `SKILL.md` لكل Skill عند الحاجة.
- احتفِظ بملاحظات خاصة بالبيئة في `TOOLS.md` (Notes for Skills).

## نصيحة النسخ الاحتياطي (موصى بها)

إذا كنت تعتبر مساحة العمل هذه «ذاكرة» Clawd، فحوّلها إلى مستودع git (ويُفضّل أن يكون خاصًا) بحيث تكون `AGENTS.md` وملفات الذاكرة لديك محفوظة بنسخ احتياطية.

```bash
cd ~/.openclaw/workspace
git init
git add AGENTS.md
git commit -m "Add Clawd workspace"
# Optional: add a private remote + push
```

## ماذا يفعل OpenClaw

- يشغّل Gateway لـ WhatsApp + وكيل Pi البرمجي بحيث يمكن للمساعد قراءة/كتابة الدردشات، وجلب السياق، وتشغيل Skills عبر مضيف Mac.
- يدير تطبيق macOS الأذونات (تسجيل الشاشة، الإشعارات، الميكروفون) ويكشف واجهة CLI `openclaw` عبر ثنائيته المضمّنة.
- تُدمَج الدردشات المباشرة افتراضيًا في جلسة `main` الخاصة بالوكيل؛ وتبقى المجموعات معزولة كـ `agent:<agentId>:<channel>:group:<id>` (الغرف/القنوات: `agent:<agentId>:<channel>:channel:<id>`)؛ وتحافظ نبضات القلب على استمرار المهام الخلفية.

## Skills الأساسية (فعّل من الإعدادات → Skills)

- **mcporter** — وقت تشغيل/CLI لخادم الأدوات لإدارة واجهات Skills الخلفية الخارجية.
- **Peekaboo** — لقطات شاشة macOS سريعة مع تحليل رؤية بالذكاء الاصطناعي اختياري.
- **camsnap** — التقاط إطارات أو مقاطع أو تنبيهات حركة من كاميرات أمان RTSP/ONVIF.
- **oracle** — CLI وكيل جاهز لـ OpenAI مع إعادة تشغيل الجلسات والتحكم بالمتصفح.
- **eightctl** — التحكّم في نومك من الطرفية.
- **imsg** — إرسال وقراءة وبث iMessage وSMS.
- **wacli** — CLI لـ WhatsApp: مزامنة، بحث، إرسال.
- **discord** — إجراءات Discord: تفاعلات، ملصقات، استطلاعات. استخدم أهداف `user:<id>` أو `channel:<id>` (المعرّفات الرقمية المجردة ملتبسة).
- **gog** — CLI لحزمة Google: Gmail، التقويم، Drive، جهات الاتصال.
- **spotify-player** — عميل Spotify طرفي للبحث/الإضافة إلى الطابور/التحكّم بالتشغيل.
- **sag** — كلام ElevenLabs مع تجربة say على نمط mac؛ يبث إلى مكبرات الصوت افتراضيًا.
- **Sonos CLI** — التحكّم في مكبرات Sonos (الاكتشاف/الحالة/التشغيل/مستوى الصوت/التجميع) من السكربتات.
- **blucli** — تشغيل وتجميع وأتمتة مشغلات BluOS من السكربتات.
- **OpenHue CLI** — التحكّم في إضاءة Philips Hue للمشاهد والأتمتة.
- **OpenAI Whisper** — تحويل الكلام إلى نص محليًا للإملاء السريع ونصوص البريد الصوتي.
- **Gemini CLI** — نماذج Google Gemini من الطرفية لأسئلة وأجوبة سريعة.
- **agent-tools** — مجموعة أدوات مساعدة للأتمتة وسكربتات المساندة.

## ملاحظات الاستخدام

- فَضِّل استخدام CLI `openclaw` للبرمجة النصية؛ يتكفّل تطبيق mac بالأذونات.
- شغّل التثبيتات من تبويب Skills؛ إذ يُخفي الزر إذا كان الثنائي موجودًا بالفعل.
- أبقِ نبضات القلب مفعّلة ليتمكّن المساعد من جدولة التذكيرات، ومراقبة صناديق البريد، وتشغيل التقاطات الكاميرا.
- تعمل واجهة Canvas بملء الشاشة مع تراكبات أصلية. تجنّب وضع عناصر تحكّم حرجة في الحواف العلوية اليسرى/اليمنى/السفلية؛ أضف هوامش صريحة في التخطيط ولا تعتمد على هوامش المناطق الآمنة.
- للتحقق المعتمد على المتصفح، استخدم `openclaw browser` (علامات التبويب/الحالة/لقطة شاشة) مع ملف Chrome المُدار من OpenClaw.
- لفحص DOM، استخدم `openclaw browser eval|query|dom|snapshot` (و`--json`/`--out` عندما تحتاج إلى مخرجات آلية).
- للتفاعلات، استخدم `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (يتطلّب النقر/الكتابة مراجع لقطات؛ استخدم `evaluate` لمحدّدات CSS).
