# TOOLS.md - ملاحظات أدوات المستخدم (قابل للتحرير)

هذا الملف مخصّص لملاحظاتك _أنت_ حول الأدوات والاصطلاحات الخارجية.
لا يعرّف هذا الملف الأدوات الموجودة؛ إذ يوفّر OpenClaw أدوات مضمّنة داخليًا.

## أمثلة

### imsg

- إرسال رسالة iMessage/SMS: صف من/ماذا، وأكّد قبل الإرسال.
- فضّل الرسائل القصيرة؛ وتجنّب إرسال الأسرار.

### sag

- تحويل النص إلى كلام: حدّد الصوت، والمتحدث/الغرفة المستهدفة، وما إذا كان البث سيتم بشكل متدفّق.

أضِف أي شيء آخر تريد أن يعرفه المساعد عن سلسلة أدواتك المحلية.
