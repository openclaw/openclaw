## Full list

- تكامل WhatsApp عبر WhatsApp Web ‏(Baileys)
- دعم روبوت Telegram ‏(grammY)
- دعم روبوت Discord ‏(channels.discord.js)
- دعم روبوت Mattermost ‏(plugin)
- تكامل iMessage عبر imsg CLI المحلي ‏(macOS)
- جسر وكيل لـ Pi في وضع RPC مع بثّ الأدوات
- البثّ والتقسيم إلى مقاطع للاستجابات الطويلة
- توجيه متعدد الوكلاء لجلسات معزولة لكل مساحة عمل أو مُرسِل
- مصادقة الاشتراك لـ Anthropic وOpenAI عبر OAuth
- الجلسات: المحادثات المباشرة تُدمَج في `main` مشترك؛ والمجموعات معزولة
- دعم الدردشة الجماعية مع تفعيل قائم على الإشارات
- دعم الوسائط للصور والصوت والمستندات
- خطاف اختياري لتفريغ الملاحظات الصوتية
- WebChat وتطبيق شريط القوائم لنظام macOS
- عُقدة iOS مع الإقران وسطح Canvas
- عُقدة Android مع الإقران وCanvas والدردشة والكاميرا

<Note>
تمت إزالة مسارات Legacy Claude وCodex وGemini وOpencode. Pi هو مسار وكيل البرمجة الوحيد.
</Note>
