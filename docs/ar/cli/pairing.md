# `openclaw pairing`

الموافقة على طلبات اقتران الرسائل المباشرة أو فحصها (للقنوات التي تدعم الاقتران).

ذات صلة:

- تدفّق الاقتران: [الاقتران](/channels/pairing)

## الأوامر

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
