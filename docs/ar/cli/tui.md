# `openclaw tui`

افتح واجهة المستخدم الطرفية المتصلة بـ Gateway «البوابة».

ذات صلة:

- دليل TUI: [TUI](/web/tui)

## أمثلة

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
