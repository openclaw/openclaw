# `openclaw reset`

إعادة تعيين التهيئة/الحالة المحلية (مع الإبقاء على تثبيت CLI).

```bash
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```
