# `openclaw logs`

Gateway-Datei-Logs per RPC verfolgen (funktioniert im Remote-Modus).

Verwandt:

- Logging-Überblick: [Logging](/logging)

## Beispiele

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
