# `openclaw tui`

Öffnet die Terminal-UI, die mit dem Gateway verbunden ist.

Verwandt:

- TUI-Leitfaden: [TUI](/web/tui)

## Beispiele

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
