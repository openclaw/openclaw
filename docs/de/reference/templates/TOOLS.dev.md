# TOOLS.md – Hinweise zu Benutzerwerkzeugen (bearbeitbar)

Diese Datei ist für _Ihre_ Notizen zu externen Werkzeugen und Konventionen.
Sie definiert nicht, welche Werkzeuge existieren; OpenClaw stellt intern integrierte Werkzeuge bereit.

## Beispiele

### imsg

- Eine iMessage/SMS senden: Beschreiben Sie Empfänger/Anlass und bestätigen Sie vor dem Senden.
- Bevorzugen Sie kurze Nachrichten; vermeiden Sie das Senden von Geheimnissen.

### sag

- Text-to-Speech: Geben Sie Stimme, Ziel-Lautsprecher/-Raum und an, ob gestreamt werden soll.

Fügen Sie alles Weitere hinzu, was der Assistent über Ihre lokale Toolchain wissen soll.
