# `openclaw tui`

Open de terminal-UI die verbonden is met de Gateway.

Gerelateerd:

- TUI-gids: [TUI](/web/tui)

## Voorbeelden

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
