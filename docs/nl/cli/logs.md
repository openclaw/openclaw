# `openclaw logs`

Gateway-bestandslogs tailen via RPC (werkt in de modus op afstand).

Gerelateerd:

- Logoverzicht: [Logging](/logging)

## Voorbeelden

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
