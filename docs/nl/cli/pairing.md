# `openclaw pairing`

DM-pairingverzoeken goedkeuren of inspecteren (voor kanalen die pairing ondersteunen).

Gerelateerd:

- Pairing-flow: [Pairing](/channels/pairing)

## Commands

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
