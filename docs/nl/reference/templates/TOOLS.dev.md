# TOOLS.md - Gebruikerstoolnotities (bewerkbaar)

Dit bestand is voor _jouw_ notities over externe tools en conventies.
Het definieert niet welke tools bestaan; OpenClaw levert intern ingebouwde tools.

## Voorbeelden

### imsg

- Stuur een iMessage/SMS: beschrijf wie/wat, bevestig vóór verzending.
- Geef de voorkeur aan korte berichten; vermijd het verzenden van geheimen.

### sag

- Tekst-naar-spraak: specificeer stem, doelspeaker/-ruimte en of er gestreamd moet worden.

Voeg toe wat je maar wilt dat de assistent moet weten over je lokale toolchain.
