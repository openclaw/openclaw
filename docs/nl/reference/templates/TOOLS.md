# TOOLS.md - Lokale notities

Skills definiëren _hoe_ tools werken. Dit bestand is voor _jouw_ specifieke zaken — alles wat uniek is aan jouw setup.

## Wat hoort hier

Dingen zoals:

- Cameranamen en -locaties
- SSH-hosts en aliassen
- Voorkeursstemmen voor TTS
- Luidspreker-/kamernamen
- Apparaat-bijnamen
- Alles wat omgevingsspecifiek is

## Voorbeelden

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## Waarom scheiden?

Skills worden gedeeld. Jouw setup is van jou. Door ze gescheiden te houden kun je skills bijwerken zonder je notities te verliezen, en skills delen zonder je infrastructuur prijs te geven.
