# TOOLS.md - Notes Locales

Skills définissent _comment_ tools marchent. Fichier pour _vos_ spécifiques — trucs uniques à votre setup.

## Qu'est ici

Choses comme :

- Noms caméras et locations
- Hosts SSH et alias
- Voix préférées pour TTS
- Noms speaker/room
- Nicknames device
- N'importe quoi environment-specific

## Exemples

```markdown
### Caméras

- living-room → Zone main, 180° wide angle
- front-door → Entrée, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Voix préférée : "Nova" (warm, slightly British)
- Speaker défaut : Kitchen HomePod
```

## Pourquoi Séparer?

Skills partagés. Votre setup vôtre. Garder séparés signifie mettre à jour skills sans perdre notes, et partager skills sans leaker infrastructure.
