# `openclaw skills`

Inspecter les compétences (intégrées + espace de travail + remplacements gérés) et voir ce qui est éligible vs exigences manquantes.

Connexe :

- Système de compétences : [Compétences](/fr-FR/tools/skills)
- Config de compétences : [Config de compétences](/fr-FR/tools/skills-config)
- Installations ClawHub : [ClawHub](/fr-FR/tools/clawhub)

## Commandes

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
