# `openclaw webhooks`

Aides webhook et intégrations (Gmail Pub/Sub, aides webhook).

Connexe :

- Webhooks : [Webhook](/fr-FR/automation/webhook)
- Gmail Pub/Sub : [Gmail Pub/Sub](/fr-FR/automation/gmail-pubsub)

## Gmail

```bash
openclaw webhooks gmail setup --account vous@exemple.com
openclaw webhooks gmail run
```

Voir [documentation Gmail Pub/Sub](/fr-FR/automation/gmail-pubsub) pour les détails.
