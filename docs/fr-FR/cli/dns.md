# `openclaw dns`

Aides DNS pour la découverte étendue (Tailscale + CoreDNS). Actuellement centré sur macOS + Homebrew CoreDNS.

Connexe :

- Découverte de Passerelle : [Découverte](/fr-FR/gateway/discovery)
- Config de découverte étendue : [Configuration](/fr-FR/gateway/configuration)

## Configuration

```bash
openclaw dns setup
openclaw dns setup --apply
```
