# `openclaw tui`

Abra a UI de terminal conectada ao Gateway.

Relacionado:

- Guia do TUI: [TUI](/web/tui)

## Exemplos

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
