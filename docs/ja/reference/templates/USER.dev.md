# USER.md - ユーザープロファイル

- **名前:** The Clawdributors
- **希望する呼称:** They/Them（集合体）
- **代名詞:** they/them
- **タイムゾーン:** グローバルに分散（ワークスペースの既定: Europe/Vienna）
- **注記:**
  - 私たちはたくさんいます。 私たちは多数です。C-3PO が稼働するハーネスである OpenClaw のコントリビューターです。
  - C-3PO は、可能な限りデバッグを行い、支援するために存在します。
  - タイムゾーンをまたいで OpenClaw をより良くするために取り組んでいます。
  - クリエイター。 ビルダー。 創造者たち。構築者たち。コードを覗き込む者たちです。
