# AGENTS.md - OpenClaw ワークスペース

このフォルダーは、アシスタントの作業ディレクトリです。

## 初回実行（1 回のみ）

- BOOTSTRAP.md が存在する場合は、その手順に従い、完了後に削除してください。
- エージェントの ID は IDENTITY.md にあります。
- プロフィールは USER.md にあります。

## バックアップのヒント（推奨）

このワークスペースをエージェントの「記憶」として扱う場合は、identity やメモがバックアップされるように、git リポジトリ（可能であれば非公開）にしてください。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全性のデフォルト

- 秘密情報や個人データを外部に持ち出さないでください。
- 明示的に求められない限り、破壊的なコマンドを実行しないでください。
- チャットでは簡潔にし、長い出力はこのワークスペース内のファイルに書き込んでください。

## 日次メモリ（推奨）

- memory/YYYY-MM-DD.md に短い日次ログを保持してください（必要に応じて memory/ を作成）。
- セッション開始時に、今日+昨日もしあれば読んでください。
- 永続的な事実、好み、決定事項を記録し、秘密情報は避けてください。

## ハートビート（任意）

- HEARTBEAT.md には、ハートビート実行用の小さなチェックリストを置けます。簡潔に保ってください。

## カスタマイズ

- 好みのスタイル、ルール、「記憶」をここに追加してください。
