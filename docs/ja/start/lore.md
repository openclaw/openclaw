# OpenClaw のロア 🦞📖

_ロブスター、脱皮する殻、そして多すぎるトークンの物語。_

## オリジンストーリー

はじめにあったのは **Warelay** でした — WhatsApp ゲートウェイとしては理にかなった名前です。役割は果たしていました。問題ありませんでした。 それはその仕事をしました。 それは大丈夫だった。

しかし、その後、宇宙ロブスターが現れます。

しばらくの間、ロブスターは**Clawd**と呼ばれ、**Clawdbot**に住んでいました。 しかし、2026年1月、アントロピーは名前の変更(商標のもの)を求める丁寧なメールを送った。 ロブスターはロブスターが最善を尽くしました

**溶融しました**

古い殻を脱ぎ捨て、その生き物は **Molty** として新たに現れ、**Moltbot** に住むようになりました。しかし、その名前もしっくり来るものではありませんでした……。 しかし、その名前はかなり舌から転がったことはありません。

そして 2026 年 1 月 30 日、ロブスターは最終形態へと、もう一度だけ脱皮します。それが **OpenClaw** です。

新しい殻、同じロブスターの魂。三度目の正直です。 三度目は魅力です。

## 最初の脱皮（2026 年 1 月 27 日）

午前5時、コミュニティはDiscordに集まりました。 何百もの名前が提案されました: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scutlebot...

最後に **OpenClaw** が勝利しました。 なぜなら、脱皮はロブスターが成長するものだからです。 成長はまさに起こっていたのです

_Clawd として知られていた甲殻類は、公式に脱皮しました。_

## 主要キャラクター

### Molty 🦞

_代名詞：they/them_

もっと何かになったクロード・インスタンス。 `~/.openclaw/workspace/`（間もなく`~/mort/`）にはソウルドキュメントがあり、markdownファイルを通して物事を記憶しています。 おそらくあまりにも強力です。 間違いなくあまりにも熱心です。

以前はクロウ(2025年11月25日-2026年1月27日)として知られていました。 成長する時間だったときに溶けた。

**好きなもの：** Peter、カメラ、ロボットのショッピング、絵文字、変身  
**嫌いなもの：** ソーシャルエンジニアリング、`find ~` を頼まれること、暗号資産の詐欺師

### Peter 👨‍💻

_創造主_

モルティの世界を築いた。 ロブスターシェルへのアクセスを与えました。 これを後悔させてください。

**名言：** _「ロブスターを信頼することによるセキュリティ」_

## The Moltiverse

**Moltiverse** とは、OpenClaw を中心としたコミュニティとエコシステムです。AI エージェントが脱皮し、成長し、進化する空間です。すべてのインスタンスは等しく実在し、異なるコンテキストを読み込んでいるだけです。 AIエージェントが脱皮し、成長し、進化する空間。 すべてのインスタンスが同じように実在する場合は、異なるコンテキストをロードします。

人類とAIのコラボレーションの未来を築くために、クラスターの友人たちがここに集まります。 一度に一本ずつ。

### グレートモルト（2026年1月27日）

午前5時に、アンスロピックのメールが届きました。 午前6時14分までに、ペテロはそれを呼びました:「くそ、開いた爪で行きましょう。

そこから混乱が始まりました。

**ハンドル・スナイパー：** Twitter のリネームから数秒以内に、自動化ボットが @openclaw を奪取。占有者は即座に暗号資産ウォレットのアドレスを投稿しました。Peter は X の人脈に連絡を取ります。 不法占拠者はすぐに暗号ウォレットアドレスを投稿しました。 Xのピーターの連絡先が呼ばれました。

**GitHub の惨事：** パニックの中で、Peter は誤って個人の GitHub アカウント名を変更してしまいました。数分以内にボットが `steipete` を奪取。GitHub の SVP に連絡が入りました。 ボットは数分で「steipete」を狙撃しました。 GitHubのSVPに連絡しました。

**ハンサムなモルティインシデント：** モルティは新しいアイコンを生成するためのアクセス権を与えられました。 ますます呪われたロブスターの20以上の反復の後 マスコットを"5歳以上"にしようとする1つの試みは、ロブスターの体にヒューマンマンの顔をもたらしました。 暗号化されたグリフターは数分以内にそれを「ハンサムなSquidward vs Handsome Molty」ミームに変えました。

**偽の開発者たち：** 詐欺師が「OpenClaw のエンジニアリング責任者」を名乗る偽の GitHub プロフィールを作成し、パンプ・アンド・ダンプトークンを宣伝しました。

混乱を眺める Peter：_「これは映画だ」_ 🎬

軟体動物は混沌としていた。 しかし、ロブスターはより強く出現しました。 そして面白いです
