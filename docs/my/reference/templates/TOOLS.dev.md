# TOOLS.md - အသုံးပြုသူ ကိရိယာ မှတ်စုများ (ပြင်ဆင်နိုင်သည်)

ဤဖိုင်သည် ပြင်ပကိရိယာများနှင့် သဘောတူညီချက်များအကြောင်း _သင်၏_ ကိုယ်ပိုင် မှတ်စုများအတွက် ဖြစ်သည်။
မည်သည့် ကိရိယာများ ရှိသည်ကို သတ်မှတ်ပေးခြင်း မဟုတ်ပါ။ OpenClaw သည် အတွင်းပိုင်းတွင် built-in ကိရိယာများကို ပံ့ပိုးပေးထားသည်။

## ဥပမာများ

### imsg

- iMessage/SMS ပို့ရန် — မည်သူ/မည်သည့်အရာကို ပို့မည်ကို ဖော်ပြပြီး ပို့မည့်အခါ မတိုင်မီ အတည်ပြုပါ။
- မက်ဆေ့ချ်များကို ချုံ့ချုံ့ရေးပါ၊ လျှို့ဝှက်ချက်များကို မပို့ပါနှင့်။

### sag

- စာသားမှ အသံသို့ (Text-to-speech) — အသံ (voice)၊ ပစ်မှတ် စပီကာ/အခန်း နှင့် စီးဆင်းပို့ဆောင်မည် (stream) မဖြစ်မဖြစ်ကို သတ်မှတ်ပါ။

သင့်ဒေသခံ ကိရိယာစနစ် (local toolchain) အကြောင်း အကူအညီပေးရန် အက်ဆစ်တန့် သိထားစေလိုသည့် အရာများကို ထပ်မံ ထည့်သွင်းနိုင်ပါသည်။
