# TOOLS.md - ဒေသဆိုင်ရာ မှတ်စုများ

Skills define _how_ tools work. 9. ဒီဖိုင်က သင့်ရဲ့ အထူးအချက်အလက်တွေ — သင့် setup နဲ့ သက်ဆိုင်ပြီး ထူးခြားတဲ့ အရာတွေပါ။

## ဒီမှာ ဘာတွေ ထည့်ရမလဲ

ဥပမာအားဖြင့်—

- ကင်မရာအမည်များနှင့် တည်နေရာများ
- SSH ဟို့စ်များနှင့် အမည်တိုများ
- TTS အတွက် နှစ်သက်သော အသံများ
- စပီကာ/အခန်း အမည်များ
- စက်ပစ္စည်း အမည်ပြောင်များ
- ပတ်ဝန်းကျင်အလိုက် သီးခြား အရာအားလုံး

## ဥပမာများ

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```
