# `openclaw sessions`

သိမ်းဆည်းထားသော စကားပြော ဆက်ရှင်များကို စာရင်းပြုလုပ်ပါ။

```bash
openclaw sessions
openclaw sessions --active 120
openclaw sessions --json
```
