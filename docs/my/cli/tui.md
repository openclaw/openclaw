# `openclaw tui`

Gateway နှင့် ချိတ်ဆက်ထားသော terminal UI ကို ဖွင့်ပါ။

ဆက်စပ်အချက်အလက်များ—

- TUI လမ်းညွှန်: [TUI](/web/tui)

## ဥပမာများ

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
