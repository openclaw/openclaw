# `openclaw pairing`

(pairing ကို ထောက်ပံ့သည့် ချန်နယ်များအတွက်) DM pairing တောင်းဆိုမှုများကို အတည်ပြုခြင်း သို့မဟုတ် စစ်ဆေးကြည့်ရှုခြင်း။

ဆက်စပ်အကြောင်းအရာများ—

- Pairing လုပ်ငန်းစဉ်: [ပူးတွဲချိတ်ဆက်ခြင်း](/channels/pairing)

## အမိန့်များ

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
