# `openclaw uninstall`

Gateway ဝန်ဆောင်မှု + ဒေသခံ ဒေတာကို ဖယ်ရှားပါ (CLI သည် ဆက်လက်ရှိနေမည်)။

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```
