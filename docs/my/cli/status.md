# `openclaw status`

ချန်နယ်များ + ဆက်ရှင်များအတွက် ရောဂါရှာဖွေမှုများ။

```bash
openclaw status
openclaw status --all
openclaw status --deep
openclaw status --usage
```

မှတ်ချက်များ:

- `--deep` သည် တိုက်ရိုက် စမ်းသပ်စစ်ဆေးမှုများ (WhatsApp Web + Telegram + Discord + Google Chat + Slack + Signal) ကို လုပ်ဆောင်သည်။
- အေးဂျင့်များကို များစွာ ဖွဲ့စည်းထားပါက အေးဂျင့်တစ်ခုချင်းစီအလိုက် ဆက်ရှင်သိုလှောင်မှုများကို အထွက်တွင် ပါဝင်သည်။
- ရရှိနိုင်ပါက အနှစ်ချုပ်တွင် Gateway + node ဟို့စ် ဝန်ဆောင်မှု၏ ထည့်သွင်းတပ်ဆင်မှု/လည်ပတ်နေမှု အခြေအနေကို ပါဝင်ဖော်ပြသည်။
- အနှစ်ချုပ်တွင် အပ်ဒိတ် ချန်နယ် + git SHA (source checkout များအတွက်) ကို ပါဝင်ဖော်ပြသည်။
- အပ်ဒိတ် အချက်အလက်များကို အနှစ်ချုပ်တွင် ပြသသည်; အပ်ဒိတ် ရရှိနိုင်ပါက `openclaw update` ကို လုပ်ဆောင်ရန် အကြံပြုချက်ကို status က ပြသသည် ([Updating](/install/updating) ကို ကြည့်ပါ)။
