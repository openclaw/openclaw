# အသုံးပြုမှုခြေရာခံခြင်း

## အဘယ်အရာဖြစ်သနည်း

- ပံ့ပိုးသူများ၏ အသုံးပြုမှု endpoint များမှ အသုံးပြုမှု/ကွိုတာကို တိုက်ရိုက် ဆွဲယူပါသည်။
- ခန့်မှန်းကုန်ကျစရိတ် မပါဝင်ပါ။ ပံ့ပိုးသူက အစီရင်ခံထားသော အချိန်ကာလများသာ အသုံးပြုပါသည်။

## မည်သည့်နေရာတွင် ပြသပါသလဲ

- chat များတွင် `/status`: session tokens နှင့် ခန့်မှန်းကုန်ကျစရိတ် (API key သာ) ပါဝင်သော emoji များပြည့်ဝသည့် status card ဖြစ်ပါသည်။ ရရှိနိုင်ပါက **လက်ရှိ model provider** အတွက် provider usage ကို ပြသပါသည်။
- ချတ်များတွင် `/usage off|tokens|full`: တုံ့ပြန်မှုတစ်ခုချင်းစီအလိုက် အသုံးပြုမှု footer (OAuth တွင် token များသာ ပြသ)။
- ချတ်များတွင် `/usage cost`: OpenClaw ဆက်ရှင် လော့ဂ်များမှ စုစည်းထားသော ဒေသခံ ကုန်ကျစရိတ် အကျဉ်းချုပ်။
- CLI: `openclaw status --usage` သည် ပံ့ပိုးသူတစ်ခုချင်းစီအလိုက် အပြည့်အစုံ ခွဲခြမ်းချက်ကို ပုံနှိပ်ပြသပါသည်။
- CLI: `openclaw channels list` သည် ပံ့ပိုးသူ ဖွဲ့စည်းပြင်ဆင်မှုနှင့်အတူ တူညီသော အသုံးပြုမှု snapshot ကို ပုံနှိပ်ပြသပါသည် (`--no-usage` ကို အသုံးပြုပြီး မကျော်လွှားနိုင်)။
- macOS မီနူးဘား: Context အောက်ရှိ “Usage” အပိုင်း (ရရှိနိုင်သည့်အခါသာ)။

## ပံ့ပိုးသူများ + အထောက်အထားများ

- **Anthropic (Claude)**: auth profile များတွင် OAuth token များ။
- **GitHub Copilot**: auth profile များတွင် OAuth token များ။
- **Gemini CLI**: auth profile များတွင် OAuth token များ။
- **Antigravity**: auth profile များတွင် OAuth token များ။
- **OpenAI Codex**: auth profile များတွင် OAuth token များ (ရှိပါက accountId ကို အသုံးပြုသည်)။
- **MiniMax**: API key (coding plan key; `MINIMAX_CODE_PLAN_KEY` သို့မဟုတ် `MINIMAX_API_KEY`); ၅ နာရီကြာ coding plan window ကို အသုံးပြုသည်။
- **z.ai**: env/config/auth store မှတစ်ဆင့် API key။

ကိုက်ညီသော OAuth/API အထောက်အထားများ မရှိပါက အသုံးပြုမှုကို ဖုံးကွယ်ထားပါသည်။
