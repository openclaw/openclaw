## Full list

- WhatsApp Web (Baileys) မှတစ်ဆင့် WhatsApp ပေါင်းစည်းမှု
- Telegram ဘော့ ပံ့ပိုးမှု (grammY)
- Discord ဘော့ ပံ့ပိုးမှု (channels.discord.js)
- Mattermost ဘော့ ပံ့ပိုးမှု (plugin)
- local imsg CLI (macOS) မှတစ်ဆင့် iMessage ပေါင်းစည်းမှု
- tool streaming ပါဝင်သည့် RPC မုဒ်တွင် Pi အတွက် Agent bridge
- ရှည်လျားသော တုံ့ပြန်ချက်များအတွက် Streaming နှင့် chunking
- workspace သို့မဟုတ် ပို့သူအလိုက် သီးခြားဆက်ရှင်များအတွက် အေးဂျင့်အများအပြား လမ်းကြောင်းချမှတ်မှု
- OAuth မှတစ်ဆင့် Anthropic နှင့် OpenAI အတွက် Subscription auth
- Sessions: တိုက်ရိုက် စကားပြောများကို မျှဝေထားသော `main` ထဲသို့ စုစည်းသည်; အုပ်စုများသည် သီးခြားဖြစ်သည်
- မန်ရှင်းအခြေပြု အက်တီဗေးရှင်းပါဝင်သည့် အုပ်စုစကားပြော ပံ့ပိုးမှု
- ပုံများ၊ အသံနှင့် စာရွက်စာတမ်းများအတွက် မီဒီယာ ပံ့ပိုးမှု
- ရွေးချယ်နိုင်သော အသံမှတ်စု transcription hook
- WebChat နှင့် macOS မီနူးဘား အက်ပ်
- pairing နှင့် Canvas surface ပါဝင်သည့် iOS နိုဒ်
- pairing၊ Canvas၊ စကားပြော နှင့် ကင်မရာ ပါဝင်သည့် Android နိုဒ်
