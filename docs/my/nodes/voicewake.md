# Voice Wake (ကမ္ဘာလုံးဆိုင်ရာ နှိုးစကားများ)

OpenClaw သည် **အသံနှိုးစကားများကို ကမ္ဘာလုံးဆိုင်ရာ စာရင်းတစ်ခုအဖြစ်** **Gateway（ဂိတ်ဝေး）** ပိုင်ဆိုင်ထားသည်ဟု ဆက်ဆံသည်။

- **နိုဒ်တစ်ခုချင်းစီအလိုက် စိတ်ကြိုက် နှိုးစကားများ မရှိပါ**။
- **မည်သည့် နိုဒ်/အက်ပ် UI မဆို စာရင်းကို တည်းဖြတ်နိုင်သည်**။ ပြောင်းလဲမှုများကို Gateway က သိမ်းဆည်းပြီး အားလုံးထံ ထုတ်လွှင့်ပေးသည်။
- စက်တစ်လုံးချင်းစီတွင်တော့ **Voice Wake ဖွင့်/ပိတ်** ခလုတ်ကို ကိုယ်ပိုင်ထားရှိနေသည် (ဒေသတွင်း UX + ခွင့်ပြုချက်များ ကွဲပြားနိုင်သည်)။

## သိုလှောင်မှု (Gateway ဟို့စ်)

နှိုးစကားများကို Gateway ဟို့စ် စက်ပေါ်တွင် အောက်ပါနေရာ၌ သိမ်းဆည်းထားသည်—

- `~/.openclaw/settings/voicewake.json`

ပုံသဏ္ဍာန် (Shape) —

```json
{ "triggers": ["openclaw", "claude", "computer"], "updatedAtMs": 1730000000000 }
```

## ပရိုတိုကော

### နည်းလမ်းများ (Methods)

- `voicewake.get` → `{ triggers: string[] }`
- `voicewake.set` ကို ပါရာမီတာများ `{ triggers: string[] }` နှင့်အတူ → `{ triggers: string[] }`

မှတ်ချက်များ—

- 49. Empty lists များသည် defaults သို့ fallback လုပ်သည်။ 50. OpenClaw သည် `web_search` tool အတွက် Perplexity Sonar ကို အသုံးပြုနိုင်သည်။
- လုံခြုံရေးအတွက် ကန့်သတ်ချက်များ (အရေအတွက်/အရှည် အမြင့်ကန့်သတ်) ကို အတည်ပြုအကောင်အထည်ဖော်ထားသည်။

### ဖြစ်ရပ်များ (Events)

- `voicewake.changed` payload `{ triggers: string[] }`

လက်ခံရရှိသူများ—

- WebSocket ကလိုင်ယင့်အားလုံး (macOS အက်ပ်၊ WebChat စသည်)
- ချိတ်ဆက်ထားသော နိုဒ်အားလုံး (iOS/Android) နှင့် နိုဒ် ချိတ်ဆက်ချိန်တွင်လည်း “လက်ရှိ အခြေအနေ” ကို အစအနေဖြင့် ထုတ်လွှင့်ပေးသည်။

## ကလိုင်ယင့် အပြုအမူ

### macOS အက်ပ်

- ကမ္ဘာလုံးဆိုင်ရာ စာရင်းကို အသုံးပြု၍ `VoiceWakeRuntime` trigger များကို စစ်ဆေးကန့်သတ်သည်။
- Voice Wake ဆက်တင်များအတွင်း “Trigger words” ကို တည်းဖြတ်ရာတွင် `voicewake.set` ကို ခေါ်ပြီး အခြား ကလိုင်ယင့်များနှင့် စင့်ခ်အောင် ထုတ်လွှင့်မှုကို အားထားသည်။

### iOS နိုဒ်

- ကမ္ဘာလုံးဆိုင်ရာ စာရင်းကို `VoiceWakeManager` trigger ခံယူသိရှိမှုအတွက် အသုံးပြုသည်။
- Settings တွင် Wake Words ကို တည်းဖြတ်ရာတွင် `voicewake.set` ကို (Gateway WS မှတစ်ဆင့်) ခေါ်ပြီး ဒေသတွင်း နှိုးစကား ခံယူသိရှိမှုကိုလည်း တုန့်ပြန်မြန်ဆန်အောင် ထိန်းသိမ်းထားသည်။

### Android နိုဒ်

- Settings တွင် Wake Words တည်းဖြတ်ကိရိယာကို ပံ့ပိုးပေးသည်။
- ပြင်ဆင်ချက်များ အားလုံးနေရာတွင် စင့်ခ်ဖြစ်စေရန် Gateway WS မှတစ်ဆင့် `voicewake.set` ကို ခေါ်သည်။
