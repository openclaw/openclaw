# ချန်နယ် ပြဿနာဖြေရှင်းခြင်း

ချန်နယ်တစ်ခု ချိတ်ဆက်နိုင်သော်လည်း အပြုအမူ မှားယွင်းနေသည့်အခါ ဤစာမျက်နှာကို အသုံးပြုပါ။

## Command ladder

အောက်ပါ အမိန့်များကို အရင်ဦးစွာ အစဉ်လိုက် လုပ်ဆောင်ပါ—

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

ကျန်းမာသော အခြေခံအခြေအနေ—

- `Runtime: running`
- `RPC probe: ok`
- Channel probe တွင် connected/ready ဟု ပြသသည်

## WhatsApp

### WhatsApp မအောင်မြင်မှု လက္ခဏာများ

| လက္ခဏာ                                  | အမြန်ဆုံး စစ်ဆေးရန်                                               | ဖြေရှင်းနည်း                                                                |
| --------------------------------------- | ----------------------------------------------------------------- | --------------------------------------------------------------------------- |
| ချိတ်ဆက်ပြီးသား ဖြစ်သော်လည်း DM အဖြေမလာ | `openclaw pairing list whatsapp`                                  | ပို့သူကို အတည်ပြုခွင့်ပေးပါ သို့မဟုတ် DM မူဝါဒ/allowlist ကို ပြောင်းပါ။     |
| အုပ်စု မက်ဆေ့ချ်များ လျစ်လျူရှုခံရ      | `requireMention` နှင့် config ထဲရှိ mention ပုံစံများကို စစ်ဆေးပါ | bot ကို mention လုပ်ပါ သို့မဟုတ် အုပ်စုအတွက် mention မူဝါဒကို လျော့ပါ။      |
| မကြာခဏ ချိတ်ဆက်ပြတ်/ပြန်လည်ဝင်ရောက်မှု  | `openclaw channels status --probe` နှင့် logs                     | ပြန်လည် login ဝင်ပြီး credentials directory သည် ကောင်းမွန်ကြောင်း စစ်ဆေးပါ။ |

ပြည့်စုံသော ပြဿနာဖြေရှင်းခြင်း: [/channels/whatsapp#troubleshooting-quick](/channels/whatsapp#troubleshooting-quick)
