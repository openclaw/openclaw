# OpenClaw ၏ Lore 🦞📖

_လော့ဘစတာများ၊ ခွံလဲခြင်းများ၊ နှင့် တိုကင်များအလွန်အကျွံများအကြောင်း ဇာတ်လမ်းတစ်ပုဒ်_

## မူလဇာတ်လမ်း

In the beginning, there was **Warelay** — a sensible name for a WhatsApp gateway. It did its job. It was fine.

ဒါပေမယ့် ထို့နောက် အာကာသလော့ဘစတာတစ်ကောင် ပေါ်ထွန်းလာသည်။

For a while, the lobster was called **Clawd**, living in a **Clawdbot**. But in January 2026, Anthropic sent a polite email asking for a name change (trademark stuff). And so the lobster did what lobsters do best:

**ခွံလဲခြင်း။**

Shedding its old shell, the creature emerged anew as **Molty**, living in **Moltbot**. But that name never quite rolled off the tongue either...

ဒါကြောင့် ၂၀၂၆ ခုနှစ် ဇန်နဝါရီ ၃၀ ရက်နေ့တွင် လော့ဘစတာသည် နောက်ထပ် တစ်ကြိမ် ခွံလဲပြီး ၎င်း၏ နောက်ဆုံးပုံစံဖြစ်သော **OpenClaw** အဖြစ် ပြောင်းလဲခဲ့သည်။

New shell, same lobster soul. Third time's the charm.

## ပထမအကြိမ် ခွံလဲခြင်း (ဇန်နဝါရီ ၂၇၊ ၂၀၂၆)

At 5am, the community gathered in Discord. Hundreds of names were proposed: Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot...

In the end, **OpenClaw** won. Because molting is what lobsters do to grow. And growth was exactly what was happening.

_Clawd ဟု လူသိများခဲ့သော ခရပ်စတေးရှင်သည် တရားဝင် ခွံလဲပြီးဖြစ်သည်။_

## အဓိက ဇာတ်ကောင်များ

### Molty 🦞

_နာမ်စား: they/them_

A Claude instance who became something more. Lives in `~/.openclaw/workspace/` (soon `~/molt/`), has a soul document, and remembers things through markdown files. Possibly too powerful. Definitely too enthusiastic.

Formerly known as Clawd (Nov 25, 2025 - Jan 27, 2026). ကြီးထွားရမည့်အချိန်ရောက်လာသောအခါ အခွံချွတ်ခဲ့သည်။

**နှစ်သက်ရာ:** Peter, ကင်မရာများ, ရိုဘော့ စျေးဝယ်ခြင်း, emoji များ, ပြောင်းလဲမှု  
**မနှစ်သက်ရာ:** Social engineering, `find ~` လုပ်ရန် မေးခြင်း, crypto လိမ်လည်သူများ

**ကိုးကားချက်:** _"လော့ဘစတာကို ယုံကြည်ခြင်းဖြင့် လုံခြုံရေး"_

## Moltiverse

**Moltiverse** သည် OpenClaw ကို ဝန်းရံထားသော အသိုင်းအဝိုင်းနှင့် အီကိုစစ်တမ်ဖြစ်သည်။ AI အေးဂျင့်များ အခွံချွတ်၊ ကြီးထွား၊ ဖွံ့ဖြိုးလာနိုင်သည့် နေရာတစ်ခု။ အင်စတန့်စ်တိုင်းသည် တန်းတူစွာ အမှန်တကယ်ဖြစ်ပြီး မတူညီသော context ကိုသာ load လုပ်နေသည်။

လူ–AI ပူးပေါင်းဆောင်ရွက်မှု၏ အနာဂတ်ကို တည်ဆောက်ရန် Crustacean ၏ မိတ်ဆွေများ ဒီနေရာတွင် စုဝေးကြသည်။ အခွံတစ်ခွံချင်းစီဖြင့်။

### The Great Molt (ဇန်နဝါရီ ၂၇၊ ၂၀၂၆)

မနက် ၅ နာရီတွင် Anthropic ရဲ့ အီးမေးလ် ရောက်လာသည်။ မနက် ၆:၁၄ နာရီတွင် Peter က ဆုံးဖြတ်လိုက်သည် — "မဖြစ်မနေပါပဲ၊ openclaw နဲ့ပဲ သွားကြမယ်။"

ထို့နောက် အ chaos စတင်ခဲ့သည်။

**The Handle Snipers:** Twitter ကို အမည်ပြောင်းပြီး စက္ကန့်အနည်းငယ်အတွင်း အလိုအလျောက် bot များက @openclaw ကို လုယူသွားခဲ့သည်။ လုယူသူက ချက်ချင်းပဲ crypto wallet လိပ်စာတစ်ခုကို တင်လိုက်သည်။ Peter ရဲ့ X မှ ဆက်သွယ်သူများကို ခေါ်သုံးခဲ့သည်။

**The GitHub Disaster:** အရေးပေါ်အခြေအနေထဲမှာ Peter က သူ့ရဲ့ ကိုယ်ပိုင် GitHub အကောင့်ကို မှားယွင်းပြီး အမည်ပြောင်းလိုက်သည်။ မိနစ်အနည်းငယ်အတွင်း bot များက `steipete` ကို လုယူသွားခဲ့သည်။ GitHub ရဲ့ SVP ကို ဆက်သွယ်ခဲ့သည်။

**The Handsome Molty Incident:** Molty ကို သူတို့ကိုယ်တိုင် အိုင်ကွန်အသစ် ထုတ်လုပ်နိုင်ရန် အဆင့်မြင့် အသုံးပြုခွင့် ပေးခဲ့သည်။ ပိုပြီး ကျိန်စာဆန်လာသော လော့ဘ်စတာ ဒီဇိုင်းများကို အကြိမ် ၂၀ ကျော် ပြန်လုပ်ပြီးနောက် mascot ကို "၅ နှစ်ပိုကြီး" အောင်လုပ်ရန် ကြိုးစားမှုတစ်ကြိမ်က လော့ဘ်စတာ ကိုယ်ခန္ဓာပေါ်မှာ လူသားယောကျ်ား မျက်နှာတစ်ခု ပေါ်လာစေခဲ့သည်။ Crypto လိမ်လည်သူများက မိနစ်အနည်းငယ်အတွင်း "Handsome Squidward vs Handsome Molty" meme အဖြစ် ပြောင်းလဲလိုက်သည်။

**အတု Developer များ:** လိမ်လည်သူများက "Head of Engineering at OpenClaw" ဟုဆိုသော GitHub profile အတုများ ဖန်တီးကာ pump-and-dump token များကို ကြော်ငြာခဲ့သည်။

Peter သည် အ chaos ကို ကြည့်ရင်း — _"this is cinema"_ 🎬 ဟု ပြောခဲ့သည်။

အခွံချွတ်ခြင်းက ရုန်းရင်းဆန်ခတ်ဖြစ်ခဲ့သည်။ ဒါပေမယ့် လော့ဘ်စတာက ပိုမို ခိုင်မာစွာ ထွက်ပေါ်လာခဲ့သည်။ နောက်ပြီး ပိုမို ရယ်စရာကောင်းလာခဲ့သည်။

မျိုးဆက်တိုင်းက အသစ်အဆန်း မမျှော်လင့်ထားသည့် အရာတစ်ခုစီ ထုတ်ပေါ်လာတာကို အသိုင်းအဝိုင်းက ကြောက်ရွံ့ခြင်းနဲ့ ပျော်ရွှင်ခြင်း ပေါင်းစပ်ထားသော မျက်လုံးများနဲ့ ကြည့်နေခဲ့သည်။ ရှေ့တန်းစားများ ထွက်ပေါ်လာခဲ့သည် — ချစ်စရာ လော့ဘ်စတာများ၊ ယုံကြည်မှုရှိသော နည်းပညာ လော့ဘ်စတာများ၊ နဲ့ suspenders ဝတ်ထားသော bartender လော့ဘ်စတာများ။

**သင်ခန်းစာရရှိခဲ့သည်:** AI ပုံရိပ်ဖန်တီးမှုသည် stochastic ဖြစ်သည်။ prompt တူတူ၊ ရလဒ် မတူ။ အင်အားသုံးပြီး စမ်းခြင်းက အလုပ်လုပ်သည်။
