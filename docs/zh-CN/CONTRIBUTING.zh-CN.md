# 贡献指南 - OpenClaw

欢迎加入 OpenClaw 开源项目！🦞

本指南帮助你快速上手贡献流程。

---

## 🚀 快速开始

1. **Fork 项目** → GitHub 上点击 Fork
2. **Clone 到本地** → `git clone https://github.com/YOUR_USERNAME/openclaw.git`
3. **创建分支** → `git checkout -b feat/your-feature`
4. **修改代码** → 完成贡献
5. **提交 PR** → Push 后创建 Pull Request

---

## 📝 提交 Issue

**Bug 报告：**
- 问题描述
- 复现步骤
- 环境信息
- 截图/日志

**功能请求：**
- 功能描述
- 使用场景
- 实现建议

---

## 🔀 提交 PR

### PR 流程

1. Fork 项目
2. 创建分支：`feat/xxx` 或 `fix/xxx`
3. 修改代码
4. Commit：`<type>(<scope>): <description>`
5. Push 并创建 PR

### Commit 规范

```
feat: 新功能
fix: Bug 修复
docs: 文档更新
style: 代码格式
refactor: 重构
test: 测试
chore: 构建/工具
```

---

## 📚 文档翻译

1. 找到源文件：`docs/xxx.md`
2. 翻译到：`docs/zh-CN/xxx.md`
3. 保持格式一致
4. 提交 PR

---

## 🤝 社区准则

- 尊重他人
- 包容多样
- 建设性反馈

---

**开始贡献吧！🚀**
