# `openclaw pairing`

Godkänn eller inspektera DM-parningsförfrågningar (för kanaler som stöder parning).

Relaterat:

- Parningsflöde: [Parning](/channels/pairing)

## Kommandon

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
