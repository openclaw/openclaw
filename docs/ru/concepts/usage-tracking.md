# Отслеживание использования

## Что это такое

- Получает данные об использовании/квотах провайдеров напрямую из их эндпоинтов использования.
- Без оценочных затрат; используются только окна, сообщаемые провайдером.

## Где отображается

- `/status` в чатах: статусная карточка с эмодзи с токенами сеанса + оценочной стоимостью (только при использовании ключа API). Использование провайдера показывается для **текущего провайдера модели**, когда доступно.
- `/usage off|tokens|full` в чатах: подвал использования для каждого ответа (при OAuth показываются только токены).
- `/usage cost` в чатах: локальная сводка стоимости, агрегированная из журналов сеансов OpenClaw.
- CLI: `openclaw status --usage` выводит полный разбор по провайдерам.
- CLI: `openclaw channels list` выводит тот же снимок использования рядом с конфигурацией провайдера (используйте `--no-usage` для пропуска).
- строка меню macOS: раздел «Usage» в Context (только если доступно).

## Поставщики + учетные данные

- **Anthropic (Claude)**: токены OAuth в профилях аутентификации.
- **GitHub Copilot**: токены OAuth в профилях аутентификации.
- **Gemini CLI**: токены OAuth в профилях аутентификации.
- **Antigravity**: токены OAuth в профилях аутентификации.
- **OpenAI Codex**: токены OAuth в профилях аутентификации (используется accountId, если присутствует).
- **MiniMax**: ключ API (ключ тарифного плана coding; `MINIMAX_CODE_PLAN_KEY` или `MINIMAX_API_KEY`); используется 5‑часовое окно тарифного плана coding.
- **z.ai**: ключ API через env/config/auth store.

Использование скрыто, если отсутствуют соответствующие учётные данные OAuth/API.
