# AGENTS.md — персональный ассистент OpenClaw (по умолчанию)

## Первый запуск (рекомендуется)

OpenClaw использует выделенный рабочий каталог для агента. По умолчанию: `~/.openclaw/workspace` (настраивается через `agents.defaults.workspace`).

1. Создайте рабочее пространство (если оно ещё не существует):

```bash
mkdir -p ~/.openclaw/workspace
```

2. Скопируйте шаблоны рабочего пространства по умолчанию в рабочее пространство:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. Необязательно: если вам нужен перечень навыков персонального ассистента, замените AGENTS.md этим файлом:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. Необязательно: выберите другое рабочее пространство, установив `agents.defaults.workspace` (поддерживает `~`):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## Параметры безопасности по умолчанию

- Не выгружайте каталоги или секреты в чат.
- Не выполняйте разрушительные команды, если об этом явно не попросили.
- Не отправляйте частичные/потоковые ответы во внешние мессенджеры (только финальные ответы).

## Начало сеанса (обязательно)

- Прочитайте `SOUL.md`, `USER.md`, `memory.md`, а также сегодняшнюю и вчерашнюю записи в `memory/`.
- Сделайте это до ответа.

## Soul (обязательно)

- `SOUL.md` определяет идентичность, тон и границы. Поддерживайте актуальность.
- Если вы изменяете `SOUL.md`, сообщите пользователю.
- Каждый сеанс — новый экземпляр; преемственность хранится в этих файлах.

## Общие пространства (рекомендуется)

- Вы — не голос пользователя; будьте осторожны в групповых чатах или публичных каналах.
- Не делитесь приватными данными, контактной информацией или внутренними заметками.

## Система памяти (рекомендуется)

- Ежедневный журнал: `memory/YYYY-MM-DD.md` (создайте `memory/` при необходимости).
- Долгосрочная память: `memory.md` для устойчивых фактов, предпочтений и решений.
- При начале сеанса прочитайте: сегодня + вчера + `memory.md`, если он существует.
- Фиксируйте: решения, предпочтения, ограничения, открытые вопросы.
- Избегайте секретов, если это не запрошено явно.

## Инструменты и навыки

- Инструменты находятся в навыках; следуйте `SKILL.md` каждого навыка, когда он вам нужен.
- Храните заметки, специфичные для окружения, в `TOOLS.md` (Notes for Skills).

## Совет по резервному копированию (рекомендуется)

Если вы рассматриваете это рабочее пространство как «память» Clawd, сделайте его git‑репозиторием (желательно приватным), чтобы `AGENTS.md` и ваши файлы памяти были сохранены в резервной копии.

```bash
cd ~/.openclaw/workspace
git init
git add AGENTS.md
git commit -m "Add Clawd workspace"
# Optional: add a private remote + push
```

## Что делает OpenClaw

- Запускает шлюз WhatsApp и кодирующего агента Pi, чтобы ассистент мог читать/писать чаты, получать контекст и выполнять навыки через хост Mac.
- Приложение для macOS управляет разрешениями (запись экрана, уведомления, микрофон) и предоставляет CLI `openclaw` через свой бинарник, входящий в комплект.
- Прямые чаты по умолчанию объединяются в сеанс агента `main`; группы остаются изолированными как `agent:<agentId>:<channel>:group:<id>` (комнаты/каналы: `agent:<agentId>:<channel>:channel:<id>`); сигналы keepalive поддерживают работу фоновых задач.

## Основные навыки (включаются в Settings → Skills)

- **mcporter** — среда выполнения/CLI сервера инструментов для управления внешними бэкендами навыков.
- **Peekaboo** — быстрые скриншоты macOS с опциональным анализом изображения ИИ.
- **camsnap** — захват кадров, клипов или оповещений о движении с камер безопасности RTSP/ONVIF.
- **oracle** — CLI агента, готовый к OpenAI, с воспроизведением сеансов и управлением браузером.
- **eightctl** — управление сном из терминала.
- **imsg** — отправка, чтение и потоковая передача iMessage и SMS.
- **wacli** — CLI WhatsApp: синхронизация, поиск, отправка.
- **discord** — действия Discord: реакции, стикеры, опросы. Используйте цели `user:<id>` или `channel:<id>` (голые числовые id неоднозначны).
- **gog** — CLI Google Suite: Gmail, Calendar, Drive, Contacts.
- **spotify-player** — терминальный клиент Spotify для поиска/очереди/управления воспроизведением.
- **sag** — синтез речи ElevenLabs с UX в стиле macOS; по умолчанию выводит звук на динамики.
- **Sonos CLI** — управление колонками Sonos (обнаружение/статус/воспроизведение/громкость/группировка) из скриптов.
- **blucli** — воспроизведение, группировка и автоматизация плееров BluOS из скриптов.
- **OpenHue CLI** — управление освещением Philips Hue для сцен и автоматизаций.
- **OpenAI Whisper** — локальное преобразование речи в текст для быстрой диктовки и транскрипций голосовой почты.
- **Gemini CLI** — модели Google Gemini из терминала для быстрых вопросов и ответов.
- **agent-tools** — набор утилит для автоматизаций и вспомогательных скриптов.

## Примечания по использованию

- Для скриптинга предпочитайте CLI `openclaw`; приложение для macOS управляет разрешениями.
- Устанавливайте из вкладки Skills; кнопка скрывается, если бинарник уже присутствует.
- Держите сигналы keepalive включёнными, чтобы ассистент мог планировать напоминания, отслеживать входящие и инициировать захват с камер.
- Интерфейс Canvas работает в полноэкранном режиме с нативными оверлеями. Избегайте размещения критически важных элементов управления в верхнем левом/правом и нижнем краях; добавляйте явные отступы в макете и не полагайтесь на safe-area insets.
- Для проверки через браузер используйте `openclaw browser` (вкладки/статус/скриншот) с профилем Chrome под управлением OpenClaw.
- Для инспекции DOM используйте `openclaw browser eval|query|dom|snapshot` (а также `--json`/`--out`, когда нужен машинный вывод).
- Для взаимодействий используйте `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` (click/type требуют ссылок на снимки; используйте `evaluate` для CSS‑селекторов).
