# TOOLS.md — Пользовательские заметки по инструментам (редактируемо)

Этот файл предназначен для _ваших_ заметок о внешних инструментах и соглашениях.
Он не определяет, какие инструменты существуют; OpenClaw предоставляет встроенные инструменты внутри системы.

## Примеры

### imsg

- Отправка iMessage/SMS: опишите, кому и что, подтвердите перед отправкой.
- Предпочитайте короткие сообщения; избегайте отправки секретов.

### sag

- Преобразование текста в речь: укажите голос, целевого слушателя/комнату и нужно ли потоковое воспроизведение.

Добавьте всё, что вы хотите, чтобы ассистент знал о вашем локальном наборе инструментов.
