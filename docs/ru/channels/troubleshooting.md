# Устранение неполадок каналов

Используйте эту страницу, когда канал подключается, но поведение работает некорректно.

## Командная лестница

Сначала выполните их по порядку:

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

Здоровая базовая линия:

- `Runtime: running`
- `RPC probe: ok`
- Проверка канала показывает состояние connected/ready

## WhatsApp

### Признаки сбоев WhatsApp

| Симптом                            | Самая быстрая проверка                                    | Исправление                                                                      |
| ---------------------------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------- |
| Подключено, но нет ответов на ЛС   | `openclaw pairing list whatsapp`                          | Одобрите отправителя или переключите политику ЛС/allowlist.      |
| Групповые сообщения игнорируются   | Проверьте `requireMention` и шаблоны упоминаний в конфиге | Упомяните бота или ослабьте политику упоминаний для этой группы. |
| Случайные разрывы/циклы перелогина | `openclaw channels status --probe` и логи                 | Перелогиньтесь и проверьте, что каталог учётных данных исправен. |

Полное устранение неполадок: [/channels/whatsapp#troubleshooting-quick](/channels/whatsapp#troubleshooting-quick)

## Telegram

### Признаки сбоев Telegram

| Симптом                                | Самая быстрая проверка                                   | Исправление                                                                            |
| -------------------------------------- | -------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| `/start` без пригодного потока ответов | `openclaw pairing list telegram`                         | Одобрите сопряжение или измените политику ЛС.                          |
| Бот онлайн, но группа молчит           | Проверьте требование упоминаний и режим приватности бота | Отключите режим приватности для видимости в группе или упомяните бота. |
| Сбои отправки с сетевыми ошибками      | Изучите логи на ошибки вызовов API Telegram              | Исправьте маршрутизацию DNS/IPv6/прокси к `api.telegram.org`.          |

Полное устранение неполадок: [/channels/telegram#troubleshooting](/channels/telegram#troubleshooting)

## Discord

### Признаки сбоев Discord

| Симптом                              | Самая быстрая проверка                          | Исправление                                                                            |
| ------------------------------------ | ----------------------------------------------- | -------------------------------------------------------------------------------------- |
| Бот онлайн, но нет ответов в гильдии | `openclaw channels status --probe`              | Разрешите гильдию/канал и проверьте Intent «Message Content».          |
| Групповые сообщения игнорируются     | Проверьте логи на отбрасывания из‑за упоминаний | Упомяните бота или задайте для гильдии/канала `requireMention: false`. |
| Нет ответов в ЛС                     | `openclaw pairing list discord`                 | Одобрите сопряжение ЛС или скорректируйте политику ЛС.                 |

Полное устранение неполадок: [/channels/discord#troubleshooting](/channels/discord#troubleshooting)

## Slack

### Признаки сбоев Slack

| Симптом                               | Самая быстрая проверка                      | Исправление                                                                    |
| ------------------------------------- | ------------------------------------------- | ------------------------------------------------------------------------------ |
| Socket mode подключён, но нет ответов | `openclaw channels status --probe`          | Проверьте app token + bot token и необходимые области доступа. |
| ЛС заблокированы                      | `openclaw pairing list slack`               | Одобрите сопряжение или ослабьте политику ЛС.                  |
| Сообщение в канале игнорируется       | Проверьте `groupPolicy` и allowlist каналов | Разрешите канал или переключите политику на `open`.            |

Полное устранение неполадок: [/channels/slack#troubleshooting](/channels/slack#troubleshooting)

### Признаки сбоев iMessage и BlueBubbles

| Симптом                                   | Самая быстрая проверка                                                   | Исправление                                                                     |
| ----------------------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------------------------------------- |
| Нет входящих событий                      | Проверьте доступность webhook/сервера и разрешения приложения            | Исправьте URL webhook или состояние сервера BlueBubbles.        |

| Отправитель ЛС заблокирован               | `openclaw pairing list imessage` или `openclaw pairing list bluebubbles` | Одобрите сопряжение или обновите allowlist.                     |

## Signal

### Признаки сбоев Signal

| Симптом                         | Самая быстрая проверка                         | Исправление                                                                      |
| ------------------------------- | ---------------------------------------------- | -------------------------------------------------------------------------------- |
| Демон доступен, но бот молчит   | `openclaw channels status --probe`             | Проверьте URL/учётную запись демона `signal-cli` и режим приёма. |
| ЛС заблокированы                | `openclaw pairing list signal`                 | Одобрите отправителя или скорректируйте политику ЛС.             |
| Ответы в группах не срабатывают | Проверьте allowlist групп и шаблоны упоминаний | Добавьте отправителя/группу или ослабьте ограничения.            |

Полное устранение неполадок: [/channels/signal#troubleshooting](/channels/signal#troubleshooting)
