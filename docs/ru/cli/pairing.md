# `openclaw pairing`

Утверждение или просмотр запросов на сопряжение в личных сообщениях (для каналов, поддерживающих сопряжение).

Связанное:

- Процесс сопряжения: [Pairing](/channels/pairing)

## Команды

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
