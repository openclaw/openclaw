# Инструменты реакций

Общая семантика реакций для разных каналов:

- `emoji` обязателен при добавлении реакции.
- `emoji=""` удаляет реакцию(и) бота, если поддерживается.
- `remove: true` удаляет указанный эмодзи, если поддерживается (требует `emoji`).

Примечания по каналам:

- **Discord/Slack**: пустой `emoji` удаляет все реакции бота на сообщении; `remove: true` удаляет только этот эмодзи.
- **Google Chat**: пустой `emoji` удаляет реакции приложения на сообщении; `remove: true` удаляет только этот эмодзи.
- **Telegram**: пустой `emoji` удаляет реакции бота; `remove: true` также удаляет реакции, но всё равно требует непустой `emoji` для валидации инструмента.
- **WhatsApp**: пустой `emoji` удаляет реакцию бота; `remove: true` сопоставляется с пустым эмодзи (всё равно требует `emoji`).
- **Signal**: входящие уведомления о реакциях генерируют системные события, когда включён `channels.signal.reactionNotifications`.
