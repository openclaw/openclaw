# TOOLS.md - Brugernoter om værktøjer (kan redigeres)

Denne fil er til _your_ noter om eksterne værktøjer og konventioner.
Det definerer ikke, hvilke værktøjer der findes; OpenClaw giver indbyggede værktøjer internt.

## Eksempler

### imsg

- Send en iMessage/SMS: beskriv hvem/hvad, bekræft før afsendelse.
- Foretræk korte beskeder; undgå at sende hemmeligheder.

### sag

- Tekst-til-tale: angiv stemme, mål-højttaler/-rum og om der skal streames.

Tilføj hvad som helst andet, du vil have assistenten til at vide om din lokale værktøjskæde.
