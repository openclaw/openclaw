# `openclaw setup`

Initialisér `~/.openclaw/openclaw.json` og agentens arbejdsområde.

Relateret:

- Kom godt i gang: [Kom godt i gang](/start/getting-started)
- Opsætningsguide: [Introduktion](/start/onboarding)

## Eksempler

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

For at køre opsætningsguiden via setup:

```bash
openclaw setup --wizard
```
