# OpenClaw 中文入门指南

## 什么是 OpenClaw？

OpenClaw 是一个强大的 AI 助手框架，可以帮助你构建自己的 AI 代理。

## 快速开始

### 1. 安装

```bash
npm install -g openclaw
```

### 2. 配置

```bash
openclaw setup
```

### 3. 启动

```bash
openclaw start
```

## 主要特性

- 🤖 AI 代理 - 创建智能代理帮你完成任务
- 💬 多渠道支持 - Telegram、Discord、Signal
- 🔌 插件系统 - 丰富的插件生态
- 🎯 可定制 - 创建自己的 Skills 和 Tools
