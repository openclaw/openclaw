# `openclaw tui`

Ouvrez l'interface utilisateur terminale connectee a la Gateway (passerelle).

Liens connexes :

- Guide TUI : [TUI](/tui)

## Exemples

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
