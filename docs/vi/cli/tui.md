# `openclaw tui`

Mở UI terminal được kết nối với Gateway.

Liên quan:

- Hướng dẫn TUI: [TUI](/web/tui)

## Ví dụ

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
