# ClawHub

ClawHub is the registry for OpenClaw skills and plugins. Use it to discover
community packages, review release trust details, and publish your own skills
or plugins.

## Start here

- [ClawHub CLI](/clawhub/cli) - search, install, update, verify, publish, and
  maintain packages from the command line.
- [Publishing on ClawHub](/clawhub/publishing) - owner scopes, package naming,
  release validation, and review flow.
