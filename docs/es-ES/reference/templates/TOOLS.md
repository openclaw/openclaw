# TOOLS.md - Notas Locales

Las Habilidades definen _cómo_ funcionan las herramientas. Este archivo es para _tus_ detalles — las cosas que son únicas de tu configuración.

## Qué Va Aquí

Cosas como:

- Nombres y ubicaciones de cámaras
- Hosts SSH y alias
- Voces preferidas para TTS
- Nombres de altavoces/habitaciones
- Apodos de dispositivos
- Cualquier cosa específica del entorno

## Ejemplos

```markdown
### Cámaras

- living-room → Área principal, ángulo ancho de 180°
- front-door → Entrada, activada por movimiento

### SSH

- home-server → 192.168.1.100, usuario: admin

### TTS

- Voz preferida: "Nova" (cálida, ligeramente británica)
- Altavoz predeterminado: HomePod de cocina
```

## ¿Por Qué Separar?

Las Habilidades son compartidas. Tu configuración es tuya. Mantenerlas separadas significa que puedes actualizar habilidades sin perder tus notas, y compartir habilidades sin filtrar tu infraestructura.
