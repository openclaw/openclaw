# `openclaw dns`

Ayudantes DNS para descubrimiento de área amplia (Tailscale + CoreDNS). Actualmente enfocado en macOS + Homebrew CoreDNS.

Relacionado:

- Descubrimiento del Gateway: [Descubrimiento](/es-ES/gateway/discovery)
- Configuración de descubrimiento de área amplia: [Configuración](/es-ES/gateway/configuration)

## Configuración

```bash
openclaw dns setup
openclaw dns setup --apply
```
