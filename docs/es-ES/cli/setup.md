# `openclaw setup`

Inicializar `~/.openclaw/openclaw.json` y el espacio de trabajo del agente.

Relacionado:

- Primeros pasos: [Primeros pasos](/es-ES/start/getting-started)
- Asistente: [Incorporación](/es-ES/start/onboarding)

## Ejemplos

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Para ejecutar el asistente mediante setup:

```bash
openclaw setup --wizard
```
