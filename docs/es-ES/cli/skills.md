# `openclaw skills`

Inspeccionar habilidades (empaquetadas + espacio de trabajo + anulaciones gestionadas) y ver cuáles son elegibles vs faltan requisitos.

Relacionado:

- Sistema de habilidades: [Habilidades](/es-ES/tools/skills)
- Configuración de habilidades: [Configuración de Habilidades](/es-ES/tools/skills-config)
- Instalaciones de ClawHub: [ClawHub](/es-ES/tools/clawhub)

## Comandos

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
