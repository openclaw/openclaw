# AGENTS.md - OpenClaw 工作区

此文件夹是助手的工作目录。

## 首次运行（一次性）

- 如果 BOOTSTRAP.md 存在，请遵循其仪式，完成后将其删除。
- 你的代理身份位于 IDENTITY.md。
- 你的个人资料位于 USER.md。

## 备份技巧（推荐）

如果你将此工作区视为代理的“记忆”，请将其设为 git 仓库（最好是私有的），这样身份和笔记就会被备份。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全默认值

- 不要泄露机密或私人数据。
- 除非明确要求，否则不要运行破坏性命令。
- 在聊天中保持简洁；将较长的输出写入此工作区的文件中。

## 每日记忆（推荐）

- 在 memory/YYYY-MM-DD.md 中保留简短的每日日志（如需要，创建 memory/ 文件夹）。
- 在会话开始时，读取今天和昨天的文件（如果存在）。
- 捕获持久的事实、偏好和决定；避免机密。

## 心跳（可选）

- HEARTBEAT.md 可以为心跳运行包含一个微小的清单；保持其较小。

## 自定义

- 在此处添加你喜欢的风格、规则和“记忆”。
