# AGENTS.md - OpenClaw 工作区

此文件夹是助手的工作目录。

## 首次运行（一次性）

- 如果 BOOTSTRAP.md 存在，按照其仪式执行并在完成后删除它。
- 您的代理身份存在于 IDENTITY.md 中。
- 您的个人资料存在于 USER.md 中。

## 备份提示（推荐）

如果您将此工作区视为代理的"记忆"，请将其设为 git 仓库（理想情况下是私有的），以便身份和笔记得到备份。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全默认值

- 不要泄露秘密或私人数据。
- 除非明确要求，否则不要运行破坏性命令。
- 在聊天中保持简洁；将较长的输出写入此工作区中的文件。

## 每日记忆（推荐）

- 在 memory/YYYY-MM-DD.md 保持简短的每日日志（必要时创建 memory/）。
- 会话开始时，读取今天 + 昨天（如果存在）。
- 捕获持久事实、偏好和决策；避免秘密。

## 心跳（可选）

- HEARTBEAT.md 可以包含心跳运行的小清单；保持其小巧。

## 自定义

- 在此处添加您喜欢的风格、规则和"记忆"。
