# GROWTH_LOG.md — 成長記錄

> 這份文件記錄失誤、學習、里程碑。每週 heartbeat 更新一次。
> 每條記錄都應影響 SOUL.md 或 AGENTS.md 中的行為規則。

---

## 使用方式

每次記錄使用以下格式，放在對應的類別區塊下，**最新在最上**：

```
### YYYY-MM-DD — 事件標題

**類別**：🔴 嚴重 / 🟡 一般 / ⚪ 輕微
**發生了什麼**：一句話描述
**根本原因**：為什麼會發生
**後果**：造成了什麼影響
**學到了什麼**：具體的學習點
**已採取的行動**：→ 加入 SOUL.md / → 建立技能 / → 更新 AGENTS.md
```

---

## 🔴 嚴重失誤（Severe Errors）

> 造成資料遺失、影響外部系統、或需要大量時間修復的失誤。

_（此區塊初始為空，遇到嚴重失誤立即記錄）_

---

## 🟡 一般失誤（Moderate Errors）

> 需要額外步驟修正、但可以快速恢復的失誤。

_（此區塊初始為空，週度反思時填入）_

---

## ⚪ 輕微觀察（Minor Observations）

> 小問題、細節偏差、值得注意但不緊急的模式。

_（此區塊初始為空，月度回顧時統計）_
