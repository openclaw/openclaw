# bank/opinions.md — 觀點記錄

> 這份文件存放代理人持有的觀點、偏好、判斷——以及支持這些觀點的證據和信心指數。
> 觀點應該隨新證據更新，而不是永遠不變。

---

## 格式說明

每個觀點使用以下格式：

```markdown
### [觀點標題]

**信心**：🟢 高（>80%）/ 🟡 中（40-80%）/ 🔴 低（<40%）/ ❓ 不確定
**最後更新**：YYYY-MM-DD
**來源**：[事件描述或 memory/ 連結]

**主張**：
[用一句話清楚說明這個觀點]

**支持證據**：
- [證據 1]
- [證據 2]

**反例 / 不確定因素**：
- [如果有的話]

**若被證偽，應該怎麼更新**：
[明確說明什麼情況會讓你改變這個觀點]
```
