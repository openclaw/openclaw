# चैनल समस्या-निवारण

जब कोई चैनल कनेक्ट हो जाए लेकिन व्यवहार गलत हो, तब इस पृष्ठ का उपयोग करें।

## कमांड सीढ़ी

पहले इन्हें क्रम से चलाएँ:

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

स्वस्थ आधाररेखा:

- `Runtime: running`
- `RPC probe: ok`
- चैनल प्रोब कनेक्टेड/तैयार दिखाता है

## WhatsApp

### WhatsApp विफलता संकेत

| लक्षण                             | सबसे तेज़ जाँच                                      | सुधार                                                       |
| --------------------------------- | --------------------------------------------------- | ----------------------------------------------------------- |
| कनेक्टेड है लेकिन DM उत्तर नहीं   | `openclaw pairing list whatsapp`                    | प्रेषक को स्वीकृत करें या DM नीति/allowlist बदलें।          |
| समूह संदेश अनदेखे                 | `requireMention` + विन्यास में उल्लेख पैटर्न जाँचें | बॉट का उल्लेख करें या उस समूह के लिए उल्लेख नीति ढीली करें। |
| यादृच्छिक डिस्कनेक्ट/री-लॉगिन लूप | `openclaw channels status --probe` + लॉग्स          | पुनः लॉगिन करें और प्रमाण-पत्र निर्देशिका की सेहत जाँचें।   |

पूर्ण समस्या-निवारण: [/channels/whatsapp#troubleshooting-quick](/channels/whatsapp#troubleshooting-quick)

## Telegram

### Telegram विफलता संकेत

| लक्षण                                       | सबसे तेज़ जाँच                                    | सुधार                                                              |
| ------------------------------------------- | ------------------------------------------------- | ------------------------------------------------------------------ |
| `/start` लेकिन उपयोगी उत्तर प्रवाह नहीं     | `openclaw pairing list telegram`                  | पेयरिंग स्वीकृत करें या DM नीति बदलें।                             |
| बॉट ऑनलाइन है लेकिन समूह शांत               | उल्लेख आवश्यकता और बॉट गोपनीयता मोड सत्यापित करें | समूह दृश्यता के लिए गोपनीयता मोड अक्षम करें या बॉट का उल्लेख करें। |
| नेटवर्क त्रुटियों के साथ भेजने में विफलताएँ | Telegram API कॉल विफलताओं के लिए लॉग्स देखें      | `api.telegram.org` के लिए DNS/IPv6/प्रॉक्सी रूटिंग ठीक करें।       |

पूर्ण समस्या-निवारण: [/channels/telegram#troubleshooting](/channels/telegram#troubleshooting)
