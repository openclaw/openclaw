# `openclaw reset`

स्थानीय विन्यास/स्थिति रीसेट करें (CLI स्थापित रहता है)।

```bash
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```
