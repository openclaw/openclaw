# `openclaw security`

सुरक्षा टूल्स (ऑडिट + वैकल्पिक सुधार)।

संबंधित:

- सुरक्षा मार्गदर्शिका: [सुरक्षा](/gateway/security)

## ऑडिट

```bash
openclaw security audit
openclaw security audit --deep
openclaw security audit --fix
```
