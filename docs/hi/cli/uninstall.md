# `openclaw uninstall`

Gateway सेवा और स्थानीय डेटा को अनइंस्टॉल करें (CLI बना रहता है)।

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```
