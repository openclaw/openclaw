# `openclaw tui`

Gateway से जुड़े टर्मिनल UI को खोलें।

संबंधित:

- TUI मार्गदर्शिका: [TUI](/web/tui)

## उदाहरण

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
