# स्क्रिप्ट्स

`scripts/` डायरेक्टरी में स्थानीय वर्कफ़्लो और ऑप्स कार्यों के लिए सहायक स्क्रिप्ट्स शामिल हैं।
जब कोई कार्य स्पष्ट रूप से किसी स्क्रिप्ट से जुड़ा हो तब इन्हें उपयोग करें; अन्यथा CLI को प्राथमिकता दें।

## परंपराएँ

- स्क्रिप्ट्स **वैकल्पिक** हैं, जब तक कि उनका उल्लेख दस्तावेज़ों या रिलीज़ चेकलिस्ट में न हो।
- जहाँ उपलब्ध हों, वहाँ CLI सतहों को प्राथमिकता दें (उदाहरण: प्रमाणीकरण मॉनिटरिंग में `openclaw models status --check` का उपयोग होता है)।
- स्क्रिप्ट्स को होस्ट‑विशिष्ट मानें; किसी नई मशीन पर चलाने से पहले उन्हें पढ़ें।
