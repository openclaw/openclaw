# स्क्रिप्ट्स

The `scripts/` directory contains helper scripts for local workflows and ops tasks.
Use these when a task is clearly tied to a script; otherwise prefer the CLI.

## परंपराएँ

- स्क्रिप्ट्स **वैकल्पिक** हैं, जब तक कि उनका उल्लेख दस्तावेज़ों या रिलीज़ चेकलिस्ट में न हो।
- जहाँ उपलब्ध हों, वहाँ CLI सतहों को प्राथमिकता दें (उदाहरण: प्रमाणीकरण मॉनिटरिंग में `openclaw models status --check` का उपयोग होता है)।
- स्क्रिप्ट्स को होस्ट‑विशिष्ट मानें; किसी नई मशीन पर चलाने से पहले उन्हें पढ़ें।

## स्क्रिप्ट्स जोड़ते समय

- स्क्रिप्ट्स को केंद्रित रखें और उनका दस्तावेज़ीकरण करें।
- संबंधित दस्तावेज़ में एक संक्षिप्त प्रविष्टि जोड़ें (या यदि अनुपस्थित हो तो नया बनाएँ)।
