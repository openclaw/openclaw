# `openclaw tui`

Gateway’e bağlı terminal UI’yi açar.

İlgili:

- TUI kılavuzu: [TUI](/web/tui)

## Örnekler

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
