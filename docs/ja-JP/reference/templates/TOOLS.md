# TOOLS.md - ローカルメモ

スキルはツールが_どのように_機能するかを定義します。このファイルは_あなた_の固有の情報のためのものです — あなたのセットアップに固有のもの。

## ここに入れるもの

次のようなもの:

- カメラ名と場所
- SSH ホストとエイリアス
- TTS の好みの音声
- スピーカー/ルーム名
- デバイスのニックネーム
- 環境固有のもの

## 例

```markdown
### カメラ

- living-room → メインエリア、180° ワイドアングル
- front-door → 入口、モーション検出

### SSH

- home-server → 192.168.1.100、ユーザー: admin

### TTS

- 好みの音声: "Nova"（温かい、少しイギリス風）
- デフォルトスピーカー: キッチン HomePod
```

## なぜ分けるのか？

スキルは共有されます。あなたのセットアップはあなたのものです。それらを分けておくことで、あなたのメモを失わずにスキルを更新でき、インフラを漏らさずにスキルを共有できます。
