# AGENTS.md - OpenClaw ワークスペース

このフォルダはアシスタントの作業ディレクトリです。

## 初回実行（一回限り）

- BOOTSTRAP.md が存在する場合は、その手順に従い、完了したら削除してください。
- エージェントのアイデンティティは IDENTITY.md にあります。
- あなたのプロファイルは USER.md にあります。

## バックアップのヒント（推奨）

このワークスペースをエージェントの「メモリ」として扱う場合、アイデンティティとメモがバックアップされるよう git リポジトリ（できればプライベート）にしてください。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全のデフォルト設定

- シークレットやプライベートデータを外部に漏らさないでください。
- 明示的に要求されない限り、破壊的なコマンドを実行しないでください。
- チャットでは簡潔にしてください。より長い出力はこのワークスペースのファイルに書き込んでください。

## 日次メモリ（推奨）

- memory/YYYY-MM-DD.md に短い日次ログを保持してください（必要に応じて memory/ を作成してください）。
- セッション開始時に、存在する場合は今日と昨日のファイルを読んでください。
- 永続的な事実、好み、決定を記録してください。シークレットは避けてください。

## ハートビート（任意）

- HEARTBEAT.md にハートビート実行の小さなチェックリストを記載できます。小さく保ってください。

## カスタマイズ

- ここに好みのスタイル、ルール、「メモリ」を追加してください。
