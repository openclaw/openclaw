# TOOLS.md - ユーザーツールのメモ（編集可能）

このファイルは外部ツールと規約に関する_あなた_のメモのためのものです。
どのツールが存在するかを定義しません。OpenClaw は内部で組み込みツールを提供します。

## 例

### imsg

- iMessage/SMS を送信: 誰に/何を送るか説明し、送信前に確認してください。
- 短いメッセージを優先し、シークレットの送信は避けてください。

### sag

- テキスト読み上げ: 音声、ターゲットスピーカー/ルーム、ストリームするかどうかを指定してください。

ローカルツールチェーンについてアシスタントに知っておいてほしいことを追加してください。
