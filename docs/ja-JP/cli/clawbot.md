# `openclaw clawbot`

後方互換性のために維持されているレガシーエイリアス名前空間です。

現在サポートされているエイリアス：

- `openclaw clawbot qr`（[`openclaw qr`](/cli/qr) と同じ動作）

## 移行

最新のトップレベルコマンドを直接使用することを推奨します：

- `openclaw clawbot qr` -> `openclaw qr`
