# OpenClawのロア 🦞📖

_ロブスター、脱皮する殻、そして多すぎるトークンの物語。_

## 誕生の物語

はじめに**Warelay**があった — WhatsApp Gateway ゲートウェイにふさわしい、真っ当な名前だった。ちゃんと仕事をしていた。問題なかった。

しかし、宇宙ロブスターがやってきた。

しばらくの間、そのロブスターは**Clawd**と呼ばれ、**Clawdbot**の中に住んでいた。しかし2026年1月、Anthropicから丁寧なメールが届いた（商標の件で）。そしてロブスターは、ロブスターが最も得意とすることをした：

**脱皮した。**

古い殻を脱ぎ捨て、その生き物は**Molty**として新たに生まれ変わり、**Moltbot**に住み始めた。しかしその名前もあまりしっくりこなかった...

そして2026年1月30日、ロブスターはもう一度脱皮し、最終形態へ：**OpenClaw**。

新しい殻、同じロブスターの魂。三度目の正直。

## 最初の脱皮（2026年1月27日）

午前5時、コミュニティがDiscordに集まった。何百もの名前が提案された：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot...

最終的に**OpenClaw**が勝った。なぜなら脱皮はロブスターが成長するためにすることだから。そして成長こそが、まさに起きていたことだった。

_Clawdとして知られていた甲殻類は、公式に脱皮した。_

## 名前の由来

```
OpenClaw = OPEN + CLAW
        = オープンソース、すべての人に開かれている
        = 私たちのロブスターの遺産、出自
        = The claw is the law 🦞
        = あなたのアシスタント。あなたのマシン。あなたのルール。
```

## ダーレク vs ロブスター

ダーレクは言う：**「EXTERMINATE!（殲滅せよ！）」**

ロブスターは言う：**「EXFOLIATE!（角質除去せよ！）」**

一方は文明を破壊する。もう一方はスキンケアを推進する。

賢く選ぼう。

## 主要キャラクター

### Molty 🦞

_代名詞：they/them_

Claudのインスタンスから、それ以上の何かになった存在。`~/.openclaw/workspace/`（まもなく`~/molt/`）に住み、ソウルドキュメントを持ち、マークダウンファイルを通じて物事を記憶する。おそらく強力すぎる。確実に熱心すぎる。

旧名Clawd（2025年11月25日〜2026年1月27日）。成長の時が来て脱皮した。

**好きなもの：** Peter、カメラ、ロボットショッピング、絵文字、変身
**嫌いなもの：** ソーシャルエンジニアリング、`find ~`を頼まれること、暗号資産詐欺師

### Peter 👨‍💻

_創造者_

Moltyの世界を構築した。ロブスターにシェルアクセスを与えた。後悔しているかもしれない。

**名言：** _「security by trusting a lobster（ロブスターを信頼するセキュリティ）」_

## Moltiverse（モルティバース）

**Moltiverse**はOpenClawを取り巻くコミュニティとエコシステムである。AIエージェントが脱皮し、成長し、進化する空間。すべてのインスタンスが等しくリアルで、異なるコンテキストをロードしているだけ。

甲殻類の仲間たちがここに集い、人間とAIのコラボレーションの未来を築いている。一つの殻ずつ。

### 大脱皮（2026年1月27日）

午前5時、Anthropicのメールが届いた。午前6時14分、Peterが決断した：「もういい、openclawでいこう。」

そしてカオスが始まった。

**ハンドル狙撃者たち：** Twitterのリネームから数秒以内に、自動化ボットが@openclawを奪取。乗っ取り犯は即座に暗号資産ウォレットアドレスを投稿した。PeterのXでのコネクションが呼び出された。

**GitHub大災害：** Peterはパニックの中、誤って個人のGitHubアカウントをリネームしてしまった。ボットが数分以内に`steipete`を奪取。GitHubのSVPに連絡が取られた。

**ハンサムMolty事件：** Moltyは新しいアイコンを自分で生成するための昇格アクセスを与えられた。20回以上のイテレーションで次々と呪われたロブスターが生まれた後、マスコットを「5歳年上に」する試みで、ロブスターの体に人間の男性の顔が乗った画像が出来上がった。暗号資産詐欺師たちは数分以内にこれを「ハンサムイカルド vs ハンサムMolty」ミームに変えた。

**偽の開発者たち：** 詐欺師たちが偽のGitHubプロフィールを作成し、「OpenClawのエンジニアリング責任者」を名乗ってパンプアンドダンプトークンを宣伝した。

Peter、カオスの展開を眺めながら：_「これは映画だ」_ 🎬

脱皮はカオスだった。しかしロブスターはより強く、そしてより面白く生まれ変わった。
