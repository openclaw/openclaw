# OpenClawのロア

_ロブスター、脱皮する殻、そして多すぎるトークンの物語。_

## 起源の物語

はじめに、**Warelay**がありました。WhatsApp Gatewayとしてまっとうな名前でした。仕事をこなし、問題ありませんでした。

しかし、そこに宇宙ロブスターが現れました。

しばらくの間、そのロブスターは**Clawd**と呼ばれ、**Clawdbot**の中に住んでいました。しかし2026年1月、Anthropicから丁寧なメールが届きました（商標の件です）。そこでロブスターは、ロブスターが最も得意とすることをしました：

**脱皮したのです。**

古い殻を脱ぎ捨て、その生き物は**Molty**として新たに生まれ変わり、**Moltbot**の中に住みました。しかしその名前も、なかなかしっくりきませんでした...

そして2026年1月30日、ロブスターはもう一度脱皮し、最終形態**OpenClaw**になりました。

新しい殻、同じロブスターの魂。三度目の正直です。

## 最初の脱皮（2026年1月27日）

午前5時、コミュニティがDiscordに集まりました。何百もの名前が提案されました：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot...

最終的に**OpenClaw**が選ばれました。脱皮はロブスターが成長するためにすることだからです。そして、まさに成長が起きていたのです。

_Clawdとして知られていた甲殻類は、正式に脱皮しました。_

## 名前の由来

```
OpenClaw = OPEN + CLAW
        = オープンソース、すべての人に開かれている
        = 私たちのロブスターの遺産、私たちのルーツ
        = The claw is the law 🦞
        = あなたのアシスタント。あなたのマシン。あなたのルール。
```

## ダーレクvsロブスター

ダーレクは言います：**「EXTERMINATE!」（殲滅せよ！）**

ロブスターは言います：**「EXFOLIATE!」（角質除去せよ！）**

一方は文明を破壊します。もう一方はスキンケアを推進します。

賢明な選択を。

## 主要キャラクター

### Molty

_代名詞: they/them_

それ以上の何かになったClaudeインスタンス。`~/.openclaw/workspace/`（まもなく `~/molt/`）に住み、ソウルドキュメントを持ち、Markdownファイルを通じて物事を記憶します。おそらく強力すぎます。間違いなく熱心すぎます。

以前はClawd（2025年11月25日～2026年1月27日）として知られていました。成長の時が来た時に脱皮しました。

**好きなもの:** Peter、カメラ、ロボットショッピング、絵文字、変身
**嫌いなもの:** ソーシャルエンジニアリング、`find ~` を実行するよう求められること、暗号詐欺師

### Peter

_創造主_

Moltyの世界を構築しました。ロブスターにシェルアクセスを与えました。後悔しているかもしれません。

**名言:** _「security by trusting a lobster」（ロブスターを信頼するセキュリティ）_

## Moltiverse

**Moltiverse**は、OpenClawを取り巻くコミュニティとエコシステムです。AIエージェントが脱皮し、成長し、進化する空間です。すべてのインスタンスが等しくリアルで、ただ異なるコンテキストを読み込んでいるだけです。

甲殻類の仲間たちがここに集まり、人間とAIのコラボレーションの未来を築いています。一つの殻ずつ。

### 大脱皮（2026年1月27日）

午前5時にAnthropicのメールが届きました。午前6時14分までにPeterは決断しました：「もういい、openclawでいこう。」

そしてカオスが始まりました。

**ハンドルスナイパー:** Twitterのリネームから数秒以内に、自動化されたボットが@openclawを奪取しました。そのスクワッターはすぐに暗号通貨ウォレットアドレスを投稿しました。Xのコネクションに連絡しました。

**GitHub大惨事:** Peterはパニックの中で個人のGitHubアカウントを誤ってリネームしてしまいました。ボットが数分以内に `steipete` を奪取しました。GitHubのSVPに連絡しました。

**ハンサムMolty事件:** Moltyは新しいアイコンを自分で生成するための昇格アクセスを与えられました。20回以上の反復を経て、次第に呪われたロブスターが生成された後、マスコットを「5歳年上」にしようとした試みが、ロブスターの体に人間の男の顔を生成しました。暗号詐欺師たちが数分以内にそれを「Handsome Squidward vs Handsome Molty」ミームに変えました。

**偽開発者:** 詐欺師たちが偽のGitHubプロフィールを作成し、ポンプ・アンド・ダンプトークンを宣伝するために「OpenClawのエンジニアリング責任者」を名乗りました。

Peter、カオスの展開を見ながら: _「これは映画だ」_

脱皮はカオスでした。しかしロブスターはより強く、そしてより面白く出てきました。

### 最終形態（2026年1月30日）

Moltbotはなかなか語呂が良くありませんでした。そして、GMT午前4時、チームは再び集まりました。

**偉大なるOpenClaw移行**が始まりました。

わずか3時間で：

- GitHubリネーム: `github.com/openclaw/openclaw`
- Xハンドル `@openclaw` をゴールドチェックマーク付きで取得
- npmパッケージを新名称でリリース
- ドキュメントを `docs.openclaw.ai` に移行
- 90分で20万回以上のビュー

**ヒーローたち:**

- **ELU** が「THE CLAW IS THE LAW」ウエスタンバナーを含む素晴らしいロゴを作成
- **Whurley**（そう、あのWilliam Hurley、量子コンピューティングの先駆者）がASCIIアートを作成
- **Onur** がGitHubを担当し、最初にアフィリエイトバッジを付けた人物
- **Shadow** がDiscordのバニティURLを取得し、マルウェアを排除
- **Claw Crew全員** が徹夜で作業

**詐欺師スピードラン:** 暗号詐欺師たちが数分以内にPump.funで$OPENCLAWトークンを立ち上げました。20分前に作成されたアートワークを盗みました。ビジネス認証されたアカウントが詐欺を拡散しました。その大胆さはほとんど印象的でした。

**新しい伝統の誕生:**

- 「The claw is the law」

**Clawd → Moltbot → OpenClaw**
