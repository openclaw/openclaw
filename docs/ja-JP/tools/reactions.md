# リアクションツール

チャンネル間で共有されるリアクションのセマンティクス:

- リアクションを追加する際は `emoji` が必須です。
- サポートされている場合、`emoji=""` はボットのリアクションを削除します。
- サポートされている場合、`remove: true` は指定された絵文字を削除します（`emoji` が必要）。

チャンネル別の注意事項:

- **Discord/Slack**: 空の `emoji` はメッセージ上のボットのすべてのリアクションを削除します。`remove: true` はその絵文字のみを削除します。
- **Google Chat**: 空の `emoji` はメッセージ上のアプリのリアクションを削除します。`remove: true` はその絵文字のみを削除します。
- **Telegram**: 空の `emoji` はボットのリアクションを削除します。`remove: true` もリアクションを削除しますが、ツールの検証のために空でない `emoji` が引き続き必要です。
- **WhatsApp**: 空の `emoji` はボットのリアクションを削除します。`remove: true` は空の絵文字にマッピングされます（それでも `emoji` が必要）。
- **Signal**: `channels.signal.reactionNotifications` が有効な場合、受信リアクション通知はシステムイベントを発行します。
