# スクリプト

`scripts/`ディレクトリには、ローカルワークフローやオペレーションタスク用のヘルパースクリプトが含まれています。
タスクが明確にスクリプトに紐づいている場合にこれらを使用してください。それ以外の場合はCLIを優先してください。

## 規約

- スクリプトはドキュメントやリリースチェックリストで参照されていない限り**任意**です。
- CLI操作が存在する場合はそちらを優先してください（例：認証モニタリングには`openclaw models status --check`を使用）。
- スクリプトはホスト固有であることを前提とし、新しいマシンで実行する前に内容を確認してください。

## スクリプトを追加する場合

- スクリプトは目的を絞り、ドキュメントを記載してください。
- 関連するドキュメントに短い項目を追加してください（存在しない場合は作成してください）。
