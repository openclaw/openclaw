# スクリプト

`scripts/` ディレクトリには、ローカルワークフローと運用タスク向けのヘルパースクリプトが含まれています。
タスクが明らかにスクリプトに結びついている場合はこれらを使用し、そうでない場合は CLI を優先してください。

## 規約

- スクリプトはドキュメントやリリースチェックリストで参照されない限り**オプション**です。
- CLI の操作が存在する場合はそちらを優先してください（例: 認証モニタリングは `openclaw models status --check` を使用します）。
- スクリプトはホスト固有であると想定してください。新しいマシンで実行する前に内容を確認してください。

## スクリプトを追加する際の注意

- スクリプトは目的を絞り、ドキュメント化してください。
- 関連ドキュメントに短いエントリを追加してください（ない場合は作成してください）。
