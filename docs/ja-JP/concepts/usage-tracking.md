# 使用状況トラッキング

## 概要

- プロバイダーの使用状況/クォータを使用エンドポイントから直接取得します。
- コストの推定は行いません。プロバイダーが報告するウィンドウのみです。

## 表示される場所

- チャットの `/status`: セッショントークンと推定コストを含む絵文字豊かなステータスカード（API キーのみ）。利用可能な場合は**現在のモデルプロバイダー**のプロバイダー使用状況が表示されます。
- チャットの `/usage off|tokens|full`: レスポンスごとの使用状況フッター（OAuth ではトークンのみ表示）。
- チャットの `/usage cost`: OpenClaw セッションログから集計されたローカルコストサマリー。
- CLI: `openclaw status --usage` はプロバイダーごとの完全な内訳を出力します。
- CLI: `openclaw channels list` はプロバイダー設定と並んで同じ使用状況スナップショットを出力します（スキップするには `--no-usage` を使用）。
- macOS メニューバー: コンテキスト下の「使用状況」セクション（利用可能な場合のみ）。

## プロバイダーと認証情報

- **Anthropic（Claude）**: 認証プロフィールの OAuth トークン。
- **GitHub Copilot**: 認証プロフィールの OAuth トークン。
- **Gemini CLI**: 認証プロフィールの OAuth トークン。
- **Antigravity**: 認証プロフィールの OAuth トークン。
- **OpenAI Codex**: 認証プロフィールの OAuth トークン（存在する場合は accountId を使用）。
- **MiniMax**: API キー（コーディングプランキー; `MINIMAX_CODE_PLAN_KEY` または `MINIMAX_API_KEY`）; 5 時間のコーディングプランウィンドウを使用します。
- **z.ai**: env/設定/認証ストアを介した API キー。

一致する OAuth/API 認証情報が存在しない場合、使用状況は非表示になります。
