# セッションツール

OpenClaw はエージェントにセッション間で作業するためのツールを提供します -- 会話の一覧表示、履歴の読み取り、他のセッションへのメッセージ送信、サブエージェントの生成が可能です。

## 利用可能なツール

| ツール               | 機能                                                          |
| -------------------- | ------------------------------------------------------------- |
| `sessions_list`      | オプションのフィルター（種類、最近のもの）でセッションを一覧表示 |
| `sessions_history`   | 特定のセッションのトランスクリプトを読み取る                  |
| `sessions_send`      | 別のセッションにメッセージを送信し、オプションで応答を待つ    |
| `sessions_spawn`     | バックグラウンド作業用に分離されたサブエージェントセッションを生成する |

## セッションの一覧表示と読み取り

`sessions_list` はセッションのキー、種類、チャネル、モデル、トークン数、タイムスタンプとともにセッションを返します。種類（`main`、`group`、`cron`、`hook`、`node`）や最近のもの（`activeMinutes`）でフィルターできます。

`sessions_history` は特定のセッションの会話トランスクリプトを取得します。デフォルトではツール結果は除外されます -- 表示するには `includeTools: true` を渡してください。

どちらのツールも**セッションキー**（`"main"` など）または以前のリスト呼び出しからの**セッション ID** を受け付けます。

## クロスセッションメッセージの送信

`sessions_send` は別のセッションにメッセージを配信し、オプションで応答を待ちます:

- **ファイア・アンド・フォーゲット:** `timeoutSeconds: 0` を設定するとキューに入れて即座に返します。
- **応答を待つ:** タイムアウトを設定するとインラインで応答を取得します。

ターゲットが応答した後、OpenClaw は**リプライバックループ**を実行でき、エージェント同士が交互にメッセージをやり取りします（最大5ターン）。ターゲットエージェントは `REPLY_SKIP` を返して早期に停止できます。

## サブエージェントの生成

`sessions_spawn` はバックグラウンドタスク用に分離されたセッションを作成します。常にノンブロッキングで、`runId` と `childSessionKey` を即座に返します。

主要なオプション:

- `runtime: "subagent"`（デフォルト）または外部ハーネスエージェント用の `"acp"`。
- 子セッション用の `model` と `thinking` のオーバーライド。
- `thread: true` でスポーンをチャットスレッド（Discord、Slack など）にバインド。
- `sandbox: "require"` で子のサンドボックス化を強制。

サブエージェントはセッションツールを除くフルツールセットを取得します（再帰的なスポーンはできません）。完了後、アナウンスステップがリクエスター元のチャネルに結果を投稿します。

ACP 固有の動作については、[ACP エージェント](/tools/acp-agents)を参照してください。

## 可視性

セッションツールはエージェントが参照できる範囲を制限するようにスコープされています:

| レベル   | スコープ                                      |
| -------- | --------------------------------------------- |
| `self`   | 現在のセッションのみ                          |
| `tree`   | 現在のセッション + 生成されたサブエージェント  |
| `agent`  | このエージェントのすべてのセッション          |
| `all`    | すべてのセッション（設定済みの場合はクロスエージェント） |

デフォルトは `tree` です。サンドボックス化されたセッションは設定に関係なく `tree` に制限されます。
