# Telegram Guide

## What is Telegram Control?

Telegram is your mobile control layer. You get alerts, approve actions,
and run prompts — all from your phone. It has the same capabilities as
the Command Center, just in a chat interface.

## What can you do here?

- **Receive alerts** — get notified about issues, deadlines, and actions
- **Approve or reject** — tap buttons to authorize sensitive changes
- **Send prompts** — ask anything, just like in the Command Center
- **Check status** — quick health checks on the go
- **Run guided commands** — use /howto and /guide for step-by-step help

## Commands

| Command | What it does |
|---------|-------------|
| `/help` | Show available commands |
| `/guide` | Start the system walkthrough |
| `/guide <number>` | Jump to a specific walkthrough step |
| `/howto <topic>` | Get a step-by-step guide |
| `/whatcanido <section>` | See what's possible in a section |
| `/explain <action>` | Learn what happens if you do something |

## Plain English

You don't have to use commands. Just type what you want:

- "What should I focus on today?"
- "Check cluster health."
- "Find grants for Full Digital."
- "Why are ads underperforming?"

OpenClaw understands plain English and responds the same way.

## Approvals

When OpenClaw needs your approval, it sends a message with inline buttons:

- **Approve** — execute the action
- **Reject** — cancel and explain why
- **Details** — see more info before deciding

Tap once. That's it.
