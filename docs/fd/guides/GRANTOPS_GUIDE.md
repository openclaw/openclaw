# GrantOps Guide

## What is GrantOps?

GrantOps automates the tedious parts of grant work. It finds opportunities,
scores them by fit, drafts applications, and tracks submissions. You review
and approve — GrantOps handles the heavy lifting.

## What can you do here?

- **Scan for grants** — automated discovery from multiple sources
- **Score opportunities** — AI-powered fit scoring
- **Prepare drafts** — auto-generated application packages
- **Track submissions** — deadlines, status, follow-ups
- **Manage deadlines** — never miss a due date
- **Generate application packages** — complete, ready-to-submit docs

## Common prompts

Try asking OpenClaw:

- "Find grants for Full Digital."
- "Draft the top two grants."
- "Show me grants needing review."
- "What deadlines are coming up?"
- "Score this opportunity."

## What requires approval?

- Submitting applications
- Committing to deliverables

Scanning, scoring, and drafting is automated and safe.

## How to run a grant scan

1. Go to Finance > GrantOps
2. Review the GrantOps summary panel
3. Ask OpenClaw to run the daily grant scan
4. Review the top scored opportunities
5. Approve draft generation for strong matches
6. Review packages before submission

## How to prepare a grant application

1. Ask OpenClaw to draft a specific grant
2. Review the generated application package
3. Edit or request revisions as needed
4. When ready, approve for submission
5. OpenClaw tracks the submission and reminds you of follow-ups
