# `openclaw tui`

Abra la UI de terminal conectada al Gateway.

Relacionado:

- Guía de TUI: [TUI](/web/tui)

## Ejemplos

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
