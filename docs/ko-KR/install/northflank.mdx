---
title: Northflank에 배포하기
---

Northflank의 원클릭 템플릿을 사용하여 OpenClaw를 배포하고 브라우저에서 설정을 완료하세요. 이는 "서버에서 터미널을 사용하지 않는" 가장 쉬운 경로입니다. Northflank가 게이트웨이를 실행하고, `/setup` 웹 마법사를 통해 모든 것을 구성합니다.

## 시작하기

1. [Deploy OpenClaw](https://northflank.com/stacks/deploy-openclaw)를 클릭하여 템플릿을 엽니다.
2. [Northflank 계정을 생성](https://app.northflank.com/signup)합니다 (계정이 없을 경우).
3. **Deploy OpenClaw now**를 클릭합니다.
4. 필수 환경 변수: `SETUP_PASSWORD`를 설정합니다.
5. **Deploy stack**을 클릭하여 OpenClaw 템플릿을 빌드하고 실행합니다.
6. 배포가 완료될 때까지 기다린 후 **View resources**를 클릭합니다.
7. OpenClaw 서비스를 엽니다.
8. 공용 OpenClaw URL을 열고 `/setup`에서 설정을 완료합니다.
9. `/openclaw`에서 Control UI를 엽니다.

## 얻을 수 있는 것

- 호스팅된 OpenClaw 게이트웨이 + Control UI
- `/setup`의 웹 설정 마법사 (터미널 명령어 없음)
- Northflank Volume (`/data`)을 통한 영구 저장소로 설정/자격 증명/작업 공간이 재배포에도 살아남음

## 설정 흐름

1. `https://<your-northflank-domain>/setup`에 방문하고 `SETUP_PASSWORD`를 입력합니다.
2. 모델/인증 프로바이더를 선택하고 키를 붙여 넣습니다.
3. (선택 사항) Telegram/Discord/Slack 토큰을 추가합니다.
4. **Run setup**을 클릭합니다.
5. `https://<your-northflank-domain>/openclaw`에서 Control UI를 엽니다.

Telegram 다이렉트 메시지가 페어링으로 설정되어 있으면, 설정 마법사가 페어링 코드를 승인할 수 있습니다.

## 채팅 토큰 얻기

### Telegram 봇 토큰

1. Telegram에서 `@BotFather`에게 메시지를 보냅니다.
2. `/newbot`을 실행합니다.
3. 토큰을 복사합니다 (예: `123456789:AA...`).
4. `/setup`에 붙여 넣습니다.

### Discord 봇 토큰

1. [https://discord.com/developers/applications](https://discord.com/developers/applications)로 이동합니다.
2. **New Application** → 이름 선택
3. **Bot** → **Add Bot**
4. Bot → Privileged Gateway Intents 아래에서 **MESSAGE CONTENT INTENT**를 활성화합니다 (필수, 활성화하지 않으면 봇이 시작 시 충돌).
5. **Bot Token**을 복사하여 `/setup`에 붙여 넣습니다.
6. 봇을 서버에 초대합니다 (OAuth2 URL Generator; 범위: `bot`, `applications.commands`).
