# Tlon (플러그인)

Tlon은 Urbit 위에 구축된 분산형 메신저입니다. OpenClaw는 Urbit ship에 연결되고
DM 및 그룹 채팅 메시지에 응답할 수 있습니다. 그룹 회신은 기본적으로 @ 멘션이 필요하며 허용 목록으로 추가로 제한될 수 있습니다.

상태: 플러그인을 통해 지원됨. DM, 그룹 멘션, 스레드 회신, 텍스트 전용 미디어 폴백 (캡션에 URL 추가). 반응, 폴, 네이티브 미디어 업로드는 지원되지 않습니다.
