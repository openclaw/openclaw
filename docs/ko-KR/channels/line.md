# LINE (플러그인)

LINE은 OpenClaw에 LINE Messaging API를 통해 연결됩니다. 플러그인은 게이트웨이에서 웹훅
리시버로 실행되고 채널 액세스 토큰 + 채널 비밀을 인증에 사용합니다.

상태: 플러그인을 통해 지원. 직접 메시지, 그룹 채팅, 미디어, 위치, Flex 메시지, 템플릿 메시지, 빠른 회신이 지원됨. 반응과 스레드는 지원되지 않음.
