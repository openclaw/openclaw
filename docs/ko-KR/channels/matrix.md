# Matrix (플러그인)

Matrix는 개방형, 분산형 메시징 프로토콜입니다. OpenClaw는 Matrix **사용자**로 모든 홈서버에 연결되므로 봇을 위해 Matrix 계정이 필요합니다. 로그인하면 봇에 직접 메시지를 보내거나 방에 초대할 수 있습니다 (Matrix "그룹"). Beeper도 유효한 클라이언트 옵션이지만 E2EE를 활성화해야 합니다.

상태: 플러그인을 통해 지원 (@vector-im/matrix-bot-sdk). 직접 메시지, 방, 스레드, 미디어, 반응, 폴 (전송 + 폴 시작을 텍스트로), 위치, 그리고 E2EE (암호 지원).
