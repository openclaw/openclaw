# TOOLS.md - Local Notes

스킬은 도구가 작동하는 _방법_을 정의합니다. 이 파일은 _당신의_ 설정에만 고유한 내용을 위한 것입니다.

## What Goes Here

다음과 같은 것들:

- 카메라 이름과 위치
- SSH 호스트와 별칭
- TTS를 위한 선호 음성
- 스피커/방 이름
- 디바이스 별명
- 환경 특수 사항

## Examples

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## Why Separate?

스킬은 공유됩니다. 당신의 설정은 당신만의 것입니다. 이를 분리하여 유지하면 스킬을 업데이트하면서 노트를 잃지 않고, 인프라를 노출하지 않으면서 스킬을 공유할 수 있습니다.

당신의 업무에 도움이 되는 내용을 추가하세요. 이것은 당신의 치트 시트입니다.
