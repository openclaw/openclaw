# BTW 부가 질문

`/btw`를 사용하면 질문을 일반 대화 히스토리에 포함시키지 않고 **현재 세션**에 대한 빠른 부가 질문을 할 수 있습니다.

Claude Code 의 `/btw` 동작을 기반으로 하지만, OpenClaw 의 Gateway 및 멀티 채널 아키텍처에 맞게 조정되었습니다.

## 작동 방식

다음과 같이 보내면:

```text
/btw what changed?
```

OpenClaw 은:

1. 현재 세션 컨텍스트를 스냅샷하고,
2. 별도의 **도구 없는** 모델 호출을 실행하고,
3. 부가 질문에만 답변하고,
4. 메인 실행은 그대로 두고,
5. BTW 질문이나 답변을 세션 히스토리에 기록하지 **않으며**,
6. 일반 어시스턴트 메시지가 아닌 **라이브 부가 결과**로 답변을 출력합니다.

중요한 멘탈 모델:

- 동일한 세션 컨텍스트
- 별도의 원샷 부가 쿼리
- 도구 호출 없음
- 향후 컨텍스트 오염 없음
- 트랜스크립트 지속성 없음

## 하지 않는 것

`/btw`는 다음을 하지 **않습니다**:

- 새로운 영구 세션 생성,
- 미완료된 메인 작업 계속,
- 도구 또는 에이전트 도구 루프 실행,
- BTW 질문/답변 데이터를 트랜스크립트 히스토리에 기록,
- `chat.history`에 표시,
- 새로고침 후 유지.

의도적으로 **임시적**입니다.

## 컨텍스트 작동 방식

BTW 는 현재 세션을 **배경 컨텍스트로만** 사용합니다.

메인 실행이 현재 활성 상태인 경우, OpenClaw 은 현재 메시지 상태를 스냅샷하고 진행 중인 메인 프롬프트를 배경 컨텍스트로 포함하면서 모델에 다음을 명시적으로 전달합니다:

- 부가 질문에만 답변할 것,
- 미완료된 메인 작업을 재개하거나 완료하지 말 것,
- 도구 호출이나 의사 도구 호출을 출력하지 말 것.

이를 통해 BTW 가 메인 실행과 격리되면서도 세션이 무엇에 관한 것인지 인식할 수 있습니다.

## 전달 모델

BTW 는 일반 어시스턴트 트랜스크립트 메시지로 전달되지 **않습니다**.

Gateway 프로토콜 수준에서:

- 일반 어시스턴트 채팅은 `chat` 이벤트를 사용합니다
- BTW 는 `chat.side_result` 이벤트를 사용합니다

이 분리는 의도적입니다. BTW 가 일반 `chat` 이벤트 경로를 재사용하면 클라이언트가 이를 일반 대화 히스토리처럼 취급합니다.

BTW 는 별도의 라이브 이벤트를 사용하고 `chat.history`에서 재생되지 않기 때문에 새로고침 후 사라집니다.

## 표면 동작

### TUI

TUI 에서 BTW 는 현재 세션 뷰에 인라인으로 렌더링되지만 임시적입니다:

- 일반 어시스턴트 답변과 시각적으로 구분됩니다
- `Enter` 또는 `Esc`로 닫을 수 있습니다
- 새로고침 시 재생되지 않습니다

### 외부 채널

Telegram, WhatsApp, Discord 와 같은 채널에서 BTW 는 명확하게 레이블된 일회성 답변으로 전달됩니다. 해당 표면에는 로컬 임시 오버레이 개념이 없기 때문입니다.

답변은 여전히 일반 세션 히스토리가 아닌 부가 결과로 취급됩니다.

### Control UI / 웹

Gateway 는 BTW 를 `chat.side_result`로 올바르게 출력하며, BTW 는 `chat.history`에 포함되지 않으므로 지속성 계약은 이미 웹에 맞게 올바릅니다.

현재 Control UI 는 브라우저에서 BTW 를 라이브로 렌더링하기 위한 전용 `chat.side_result` 소비자가 아직 필요합니다. 해당 클라이언트 측 지원이 적용될 때까지 BTW 는 완전한 TUI 및 외부 채널 동작을 갖춘 Gateway 수준 기능이지만 아직 완전한 브라우저 UX 는 아닙니다.

## BTW 를 사용할 때

다음을 원할 때 `/btw`를 사용합니다:

- 현재 작업에 대한 빠른 설명,
- 긴 실행이 아직 진행 중일 때 사실적인 부가 답변,
- 향후 세션 컨텍스트의 일부가 되지 않아야 하는 임시 답변.

예시:

```text
/btw what file are we editing?
/btw what does this error mean?
/btw summarize the current task in one sentence
/btw what is 17 * 19?
```

## BTW 를 사용하지 않을 때

답변이 세션의 향후 작업 컨텍스트의 일부가 되기를 원하는 경우 BTW 를 사용하지 마세요.

이 경우 BTW 를 사용하는 대신 메인 세션에서 정상적으로 질문하세요.
