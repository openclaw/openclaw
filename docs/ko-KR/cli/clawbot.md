# `openclaw clawbot`

하위 호환성을 위해 유지되는 레거시 별칭 네임스페이스입니다.

현재 지원되는 별칭:

- `openclaw clawbot qr` ([`openclaw qr`](/cli/qr)과 동일한 동작)

## 마이그레이션

최신 최상위 명령어를 직접 사용하는 것을 권장합니다:

- `openclaw clawbot qr` -> `openclaw qr`
