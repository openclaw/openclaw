# `openclaw pairing`

다이렉트 메시지 페어링 요청 승인 또는 검사 (페어링을 지원하는 채널용).

관련 항목:

- 페어링 흐름: [페어링](/channels/pairing)

## 명령어

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
