# 커뮤니티 플러그인

이 페이지는 OpenClaw를 위한 고품질 **커뮤니티 관리 플러그인**을 추적합니다.

품질 기준을 충족하는 커뮤니티 플러그인을 추가하는 PR을 수락합니다.

## 목록 등재 요건

- 플러그인 패키지가 npmjs에 게시되어 있어야 합니다 (`openclaw plugins install <npm-spec>`을 통해 설치 가능).
- 소스 코드가 GitHub에 호스팅되어 있어야 합니다 (공개 저장소).
- 저장소에 설치/사용 문서와 이슈 트래커가 포함되어 있어야 합니다.
- 플러그인에 명확한 유지 관리 신호가 있어야 합니다 (활성 유지 관리자, 최근 업데이트 또는 응답형 이슈 처리).

## 제출 방법

다음을 포함하여 이 페이지에 플러그인을 추가하는 PR을 엽니다:

- 플러그인 이름
- npm 패키지 이름
- GitHub 저장소 URL
- 한 줄 설명
- 설치 명령

## 검토 기준

유용하고, 문서화되어 있으며, 안전하게 작동할 수 있는 플러그인을 선호합니다.
노력이 적은 래퍼, 불명확한 소유권 또는 유지 관리되지 않는 패키지는 거부될 수 있습니다.

## 후보 형식

항목을 추가할 때 이 형식을 사용하세요:

- **플러그인 이름** — 간단한 설명
  npm: `@scope/package`
  repo: `https://github.com/org/repo`
  install: `openclaw plugins install @scope/package`
