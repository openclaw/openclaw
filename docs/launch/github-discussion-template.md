## Default pinned discussion body

```md
# VeriClaw

VeriClaw is the public-facing product brand for a self-hosted AI assistant, MCP-compatible gateway, and multi-channel bot runtime for Discord, Telegram, WhatsApp, Slack, browser automation, and multi-agent workflows.

## Why this repo is worth forking

- one runtime spans channels, tools, browser automation, and multi-agent execution
- the correction workflow focuses on runtime truth and delivery integrity
- the repo is useful as both a deployment base and a reference architecture

## Start here

- Getting started: https://docs.openclaw.ai/start/getting-started
- Docs: https://docs.openclaw.ai
- Repository: https://github.com/openclaw/openclaw

## Note on naming

The public-facing product name is VeriClaw.
The current repository, runtime, and CLI still use `OpenClaw` / `openclaw` naming for compatibility.
```
