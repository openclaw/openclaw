# `activi setup`

Initialize `~/.activi/activi.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
activi setup
activi setup --workspace ~/.activi/workspace
```

To run the wizard via setup:

```bash
activi setup --wizard
```
