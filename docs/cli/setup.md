# `dna setup`

Initialize `~/.dna/dna.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
dna setup
dna setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
dna setup --wizard
```
