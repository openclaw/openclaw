# `activi pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
activi pairing list whatsapp
activi pairing approve whatsapp <code> --notify
```
