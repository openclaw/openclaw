# `dna tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
dna tui
dna tui --url ws://127.0.0.1:18789 --token <token>
dna tui --session main --deliver
```
