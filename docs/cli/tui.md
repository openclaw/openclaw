# `activi tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
activi tui
activi tui --url ws://127.0.0.1:18789 --token <token>
activi tui --session main --deliver
```
