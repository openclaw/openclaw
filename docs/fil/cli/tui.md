# `openclaw tui`

Buksan ang terminal UI na konektado sa Gateway.

Kaugnay:

- Gabay sa TUI: [TUI](/web/tui)

## Mga halimbawa

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
