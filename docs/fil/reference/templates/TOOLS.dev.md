# TOOLS.md - Mga Tala ng Tool ng User (nae-edit)

Ang file na ito ay para sa _iyong_ mga tala tungkol sa mga external na tool at kombensyon.
Hindi nito tinutukoy kung aling mga tool ang umiiral; nagbibigay ang OpenClaw ng mga built-in na tool sa loob.

## Mga halimbawa

### imsg

- Magpadala ng iMessage/SMS: ilarawan kung kanino/ano, kumpirmahin bago magpadala.
- Mas piliin ang maiikling mensahe; iwasang magpadala ng mga lihim.

### sag

- Text-to-speech: tukuyin ang boses, target na speaker/kuwarto, at kung mag-i-stream.

Idagdag ang anumang gusto mong malaman ng assistant tungkol sa iyong lokal na toolchain.
