# TOOLS.md - บันทึกเครื่องมือผู้ใช้ (แก้ไขได้)

1. ไฟล์นี้มีไว้สำหรับบันทึกของ _คุณ_ เกี่ยวกับเครื่องมือและธรรมเนียมภายนอก
   ไฟล์นี้มีไว้สำหรับบันทึกของ _คุณ_ เกี่ยวกับเครื่องมือภายนอกและธรรมเนียมการใช้งาน
   ไฟล์นี้ไม่ได้กำหนดว่ามีเครื่องมือใดบ้าง; OpenClaw มีเครื่องมือที่สร้างมาให้ภายในอยู่แล้ว

## ตัวอย่าง

### imsg

- ส่ง iMessage/SMS: อธิบายว่าใคร/อะไร และยืนยันก่อนส่ง
- ควรใช้ข้อความสั้นๆ; หลีกเลี่ยงการส่งความลับ

### sag

- แปลงข้อความเป็นเสียงพูด: ระบุเสียง ผู้พูด/ห้องเป้าหมาย และระบุว่าจะสตรีมหรือไม่

เพิ่มสิ่งอื่นใดก็ได้ที่คุณต้องการให้ผู้ช่วยทราบเกี่ยวกับชุดเครื่องมือในเครื่องของคุณ
