# สคริปต์

ไดเรกทอรี `scripts/` มีสคริปต์ตัวช่วยสำหรับเวิร์กโฟลว์ในเครื่องและงานปฏิบัติการ
ไดเรกทอรี `scripts/` มีสคริปต์ช่วยสำหรับเวิร์กโฟลว์ภายในเครื่องและงานปฏิบัติการ
ให้ใช้เมื่อภารกิจผูกกับสคริปต์อย่างชัดเจน มิฉะนั้นควรใช้ CLI เป็นหลัก

## แนวปฏิบัติ

- สคริปต์เป็น **ทางเลือก** เว้นแต่จะถูกอ้างอิงในเอกสารหรือเช็กลิสต์การรีลีส
- ควรใช้พื้นผิวของ CLI เมื่อมีให้ใช้ (ตัวอย่าง: การมอนิเตอร์การยืนยันตัวตนใช้ `openclaw models status --check`)
- ถือว่าสคริปต์ขึ้นกับโฮสต์ โปรดอ่านก่อนรันบนเครื่องใหม่

## เมื่อเพิ่มสคริปต์

- ทำให้สคริปต์มีขอบเขตชัดเจนและมีเอกสารประกอบ
- เพิ่มรายการสั้นๆในเอกสารที่เกี่ยวข้อง (หรือสร้างเอกสารใหม่หากยังไม่มี)
