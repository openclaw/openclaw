# `openclaw tui`

เปิดเทอร์มินัลUIที่เชื่อมต่อกับGateway

เกี่ยวข้อง:

- คู่มือTUI: [TUI](/web/tui)

## ตัวอย่าง

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
