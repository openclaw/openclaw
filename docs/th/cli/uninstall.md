# `openclaw uninstall`

ถอนการติดตั้งบริการGateway（เกตเวย์）+ข้อมูลภายในเครื่อง (CLIยังคงอยู่)

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```
