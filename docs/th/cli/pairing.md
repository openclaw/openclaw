# `openclaw pairing`

อนุมัติหรือตรวจสอบคำขอการจับคู่DM(สำหรับช่องทางที่รองรับการจับคู่)

เกี่ยวข้อง:

- โฟลว์การจับคู่: [การจับคู่](/channels/pairing)

## คำสั่ง

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
