# چینل کی خرابیوں کا ازالہ

جب کوئی چینل کنیکٹ ہو جائے مگر رویہ درست نہ ہو تو یہ صفحہ استعمال کریں۔

## کمانڈ سیڑھی

سب سے پہلے ان کو ترتیب وار چلائیں:

```bash
openclaw status
openclaw gateway status
openclaw logs --follow
openclaw doctor
openclaw channels status --probe
```

صحت مند بنیادی حالت:

- `Runtime: running`
- `RPC probe: ok`
- چینل پروب منسلک/تیار دکھاتا ہے

## WhatsApp

### WhatsApp ناکامی کی علامات

| علامت                                 | تیز ترین جانچ                                       | حل                                                                |
| ------------------------------------- | --------------------------------------------------- | ----------------------------------------------------------------- |
| منسلک ہے مگر DMs کے جوابات نہیں       | `openclaw pairing list whatsapp`                    | ارسال کنندہ کی منظوری دیں یا DM پالیسی/اجازت فہرست تبدیل کریں۔    |
| گروپ پیغامات نظرانداز                 | `requireMention` + کنفیگ میں ذکر کے پیٹرنز چیک کریں | بوٹ کو منشن کریں یا اس گروپ کے لیے ذکر پالیسی نرم کریں۔           |
| بے ترتیب ڈس کنیکٹ/دوبارہ لاگ اِن لوپس | `openclaw channels status --probe` + لاگز           | دوبارہ لاگ اِن کریں اور تصدیق کریں کہ اسناد کی ڈائریکٹری درست ہے۔ |

مکمل خرابیوں کا ازالہ: [/channels/whatsapp#troubleshooting-quick](/channels/whatsapp#troubleshooting-quick)

## Telegram

### Telegram ناکامی کی علامات

| علامت                                        | تیز ترین جانچ                                  | حل                                                                    |
| -------------------------------------------- | ---------------------------------------------- | --------------------------------------------------------------------- |
| `/start` مگر قابلِ استعمال جواب کا بہاؤ نہیں | `openclaw pairing list telegram`               | جوڑی بنانے کی منظوری دیں یا DM پالیسی تبدیل کریں۔                     |
| بوٹ آن لائن مگر گروپ خاموش                   | ذکر کی شرط اور بوٹ پرائیویسی موڈ کی تصدیق کریں | گروپ کی مرئیت کے لیے پرائیویسی موڈ غیر فعال کریں یا بوٹ کو منشن کریں۔ |
| نیٹ ورک غلطیوں کے ساتھ بھیجنے میں ناکامی     | Telegram API کال ناکامیوں کے لیے لاگز دیکھیں   | `api.telegram.org` تک DNS/IPv6/پراکسی روٹنگ درست کریں۔                |

مکمل خرابیوں کا ازالہ: [/channels/telegram#troubleshooting](/channels/telegram#troubleshooting)
