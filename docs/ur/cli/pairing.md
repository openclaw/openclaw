# `openclaw pairing`

ان چینلز کے لیے DM پیئرنگ درخواستوں کی منظوری دیں یا ان کا معائنہ کریں جو پیئرنگ کو سپورٹ کرتے ہیں۔

متعلقہ:

- پیئرنگ فلو: [Pairing](/channels/pairing)

## کمانڈز

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
