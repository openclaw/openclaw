# `openclaw logs`

RPC کے ذریعے Gateway فائل لاگز کی ٹیلنگ کریں (ریموٹ موڈ میں کام کرتا ہے)۔

متعلقہ:

- لاگنگ کا جائزہ: [Logging](/logging)

## مثالیں

```bash
openclaw logs
openclaw logs --follow
openclaw logs --json
openclaw logs --limit 500
```
