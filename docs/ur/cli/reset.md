# `openclaw reset`

مقامی کنفیگ/اسٹیٹ کو ری سیٹ کریں (CLI انسٹال ہی رہتا ہے)۔

```bash
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```
