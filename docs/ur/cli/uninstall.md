# `openclaw uninstall`

گیٹ وے سروس اور مقامی ڈیٹا ان انسٹال کریں (CLI برقرار رہتا ہے)۔

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```
