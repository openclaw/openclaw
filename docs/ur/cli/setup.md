# `openclaw setup`

`~/.openclaw/openclaw.json` اور ایجنٹ ورک اسپیس کو ابتدائی حالت میں لائیں۔

متعلقہ:

- ابتدائی رہنمائی: [Getting started](/start/getting-started)
- وزرڈ: [Onboarding](/start/onboarding)

## مثالیں

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

سیٹ اپ کے ذریعے وزرڈ چلانے کے لیے:

```bash
openclaw setup --wizard
```
