# macOS پر ہیلتھ چیکس

مینو بار ایپ سے یہ دیکھنے کا طریقہ کہ منسلک چینل صحت مند ہے یا نہیں۔

## مینو بار

- اسٹیٹس ڈاٹ اب Baileys کی صحت کو ظاہر کرتا ہے:
  - سبز: منسلک + حال ہی میں ساکٹ کھلا ہوا۔
  - نارنجی: کنیکٹ ہو رہا ہے/دوبارہ کوشش۔
  - سرخ: لاگ آؤٹ یا پروب ناکام۔
- ثانوی سطر "linked · auth 12m" دکھاتی ہے یا ناکامی کی وجہ۔
- "Run Health Check" مینو آئٹم آن ڈیمانڈ پروب چلاتا ہے۔

## سیٹنگز

- جنرل ٹیب میں ایک ہیلتھ کارڈ شامل ہوتا ہے جو یہ دکھاتا ہے: لنکڈ آتھ کی عمر، session-store کا راستہ/گنتی، آخری چیک کا وقت، آخری خرابی/اسٹیٹس کوڈ، اور Run Health Check / Reveal Logs کے بٹن۔
- ایک کیشڈ اسنیپ شاٹ استعمال کرتا ہے تاکہ UI فوراً لوڈ ہو اور آف لائن ہونے پر خوش اسلوبی سے فال بیک کرے۔
- **Channels ٹیب** WhatsApp/Telegram کے لیے چینل اسٹیٹس + کنٹرولز ظاہر کرتا ہے (لاگ اِن QR، لاگ آؤٹ، پروب، آخری ڈسکنیکٹ/خرابی)۔

## پروب کیسے کام کرتا ہے

- ایپ ہر تقریباً 60 سیکنڈ بعد اور ضرورت پر `ShellExecutor` کے ذریعے `openclaw health --json` چلاتی ہے۔ یہ پروب اسناد لوڈ کرتا ہے اور پیغامات بھیجے بغیر اسٹیٹس رپورٹ کرتا ہے۔
- فِلکر سے بچنے کے لیے آخری درست اسنیپ شاٹ اور آخری خرابی کو الگ الگ کیش کریں؛ ہر ایک کا ٹائم اسٹیمپ دکھائیں۔

## جب شک ہو

- آپ اب بھی [Gateway health](/gateway/health) میں CLI فلو (`openclaw status`, `openclaw status --deep`, `openclaw health --json`) استعمال کر سکتے ہیں اور `web-heartbeat` / `web-reconnect` کے لیے `/tmp/openclaw/openclaw-*.log` کو ٹیل کر سکتے ہیں۔
