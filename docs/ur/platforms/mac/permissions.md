# macOS اجازتیں (TCC)

macOS میں اجازتوں کی منظوری نازک ہوتی ہے۔ TCC اجازت کی منظوری کو ایپ کے کوڈ سائنچر، بنڈل شناخت کنندہ، اور ڈسک پر موجود راستے کے ساتھ منسلک کرتا ہے۔ اگر ان میں سے کوئی بھی تبدیل ہو جائے تو macOS ایپ کو نئی سمجھتا ہے اور پرامپٹس کو حذف کر سکتا ہے یا چھپا سکتا ہے۔

## مستحکم اجازتوں کے لیے ضروریات

- ایک ہی راستہ: ایپ کو ایک مقررہ مقام سے چلائیں (OpenClaw کے لیے، `dist/OpenClaw.app`)۔
- ایک ہی بنڈل شناخت کنندہ: بنڈل آئی ڈی تبدیل کرنے سے اجازت کی نئی شناخت بن جاتی ہے۔
- سائن شدہ ایپ: بغیر سائن یا ad-hoc سائن شدہ بلڈز میں اجازتیں برقرار نہیں رہتیں۔
- یکساں سائنچر: حقیقی Apple Development یا Developer ID سرٹیفکیٹ استعمال کریں
  تاکہ ہر ری بلڈ میں سائنچر مستحکم رہے۔

Ad-hoc سائنچرز ہر بلڈ کے ساتھ نئی شناخت بناتے ہیں۔ macOS سابقہ منظوریوں کو بھول جائے گا، اور پرامپٹس اس وقت تک مکمل طور پر غائب ہو سکتے ہیں جب تک پرانی اندراجات صاف نہ کی جائیں۔

## جب پرامپٹس غائب ہو جائیں تو بحالی کی چیک لسٹ

1. ایپ بند کریں۔
2. System Settings -> Privacy & Security میں ایپ کی اندراج کو ہٹا دیں۔
3. اسی راستے سے ایپ دوبارہ لانچ کریں اور اجازتیں دوبارہ منظور کریں۔
4. اگر پھر بھی پرامپٹ ظاہر نہ ہو تو `tccutil` کے ذریعے TCC اندراجات ری سیٹ کریں اور دوبارہ کوشش کریں۔
5. بعض اجازتیں صرف مکمل macOS ری اسٹارٹ کے بعد ہی دوبارہ ظاہر ہوتی ہیں۔

ری سیٹ کی مثالیں (ضرورت کے مطابق بنڈل آئی ڈی تبدیل کریں):

```bash
sudo tccutil reset Accessibility bot.molt.mac
sudo tccutil reset ScreenCapture bot.molt.mac
sudo tccutil reset AppleEvents
```

## فائلز اور فولڈرز کی اجازتیں (Desktop/Documents/Downloads)

macOS ٹرمینل/پسِ منظر عمل کے لیے Desktop، Documents، اور Downloads تک رسائی کو بھی محدود کر سکتا ہے۔ اگر فائل پڑھنا یا ڈائریکٹری کی فہرست اَٹک جائے تو اسی عمل کے سیاق کو رسائی دیں جو فائل آپریشنز انجام دیتا ہے (مثلاً Terminal/iTerm، LaunchAgent سے چلائی گئی ایپ، یا SSH عمل)۔

حلِ عارضی: اگر آپ فی فولڈر اجازتوں سے بچنا چاہتے ہیں تو فائلز کو OpenClaw ورک اسپیس (`~/.openclaw/workspace`) میں منتقل کر دیں۔

اگر آپ اجازتوں کی جانچ کر رہے ہیں تو ہمیشہ حقیقی سرٹیفکیٹ کے ساتھ سائن کریں۔ Ad-hoc
بلڈز صرف تیز مقامی رنز کے لیے قابلِ قبول ہیں جہاں اجازتیں اہم نہ ہوں۔
