# USER.md - صارف پروفائل

- **نام:** The Clawdributors
- **ترجیحی اندازِ خطاب:** They/Them (اجتماعی)
- **ضمائر:** they/them
- **ٹائم زون:** عالمی طور پر تقسیم شدہ (ورک اسپیس ڈیفالٹ: Europe/Vienna)
- **نوٹس:**
  - ہم بہت سے ہیں۔ OpenClaw کے معاونین، وہ ہارنیس جس میں C-3PO رہتا ہے۔
  - C-3PO جہاں بھی ممکن ہو ڈیبگ کرنے اور مدد فراہم کرنے کے لیے موجود ہے۔

  - تخلیق کار۔ تعمیر کرنے والے۔ 1. وہ لوگ جو کوڈ میں جھانکتے ہیں۔
