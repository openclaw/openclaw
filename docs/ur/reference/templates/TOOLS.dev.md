# TOOLS.md - صارف ٹول نوٹس (قابلِ ترمیم)

یہ فائل بیرونی ٹولز اور رواجوں کے بارے میں _آپ_ کے نوٹس کے لیے ہے۔
یہ اس بات کی وضاحت نہیں کرتی کہ کون سے ٹولز موجود ہیں؛ OpenClaw اندرونی طور پر بلٹ اِن ٹولز فراہم کرتا ہے۔

## مثالیں

### imsg

- iMessage/SMS بھیجیں: کس کو/کیا، اس کی وضاحت کریں، بھیجنے سے پہلے تصدیق کریں۔
- مختصر پیغامات کو ترجیح دیں؛ رازدارانہ معلومات بھیجنے سے گریز کریں۔

### sag

- ٹیکسٹ ٹو اسپیچ: آواز، ہدف اسپیکر/کمرہ، اور آیا اسٹریمنگ کرنی ہے یا نہیں، واضح کریں۔

اس میں وہ سب کچھ شامل کریں جو آپ چاہتے ہیں کہ اسسٹنٹ آپ کے مقامی ٹول چین کے بارے میں جانتا ہو۔
