# TOOLS.md - مقامی نوٹس

اسکلز یہ طے کرتی ہیں کہ ٹولز _کیسے_ کام کرتے ہیں۔ یہ فائل _تمہاری_ مخصوصات کے لیے ہے — وہ چیزیں جو تمہارے سیٹ اپ کے لیے منفرد ہیں۔

## یہاں کیا شامل کریں

مثلاً:

- کیمروں کے نام اور مقامات
- SSH ہوسٹس اور عرفیات
- TTS کے لیے ترجیحی آوازیں
- اسپیکر/کمرہ کے نام
- ڈیوائس کے عرفی نام
- ماحول سے متعلق کوئی بھی چیز

## مثالیں

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- Preferred voice: "Nova" (warm, slightly British)
- Default speaker: Kitchen HomePod
```

## الگ کیوں؟

اسکلز مشترک ہوتی ہیں۔ تمہارا سیٹ اپ تمہارا اپنا ہے۔ انہیں الگ رکھنے کا مطلب یہ ہے کہ تم اپنے نوٹس کھوئے بغیر اسکلز اپڈیٹ کر سکتے ہو، اور اپنی انفراسٹرکچر ظاہر کیے بغیر اسکلز شیئر کر سکتے ہو۔

---

جو کچھ بھی تمہیں اپنا کام بہتر کرنے میں مدد دے، شامل کرو۔ یہ تمہاری چیٹ شیٹ ہے۔
