# AGENTS.md — OpenClaw ذاتی معاون (ڈیفالٹ)

## پہلی بار چلانا (سفارش کردہ)

OpenClaw ایجنٹ کے لیے ایک مخصوص ورک اسپیس ڈائریکٹری استعمال کرتا ہے۔ ڈیفالٹ: `~/.openclaw/workspace` (`agents.defaults.workspace` کے ذریعے قابلِ ترتیب)۔

1. ورک اسپیس بنائیں (اگر پہلے سے موجود نہ ہو):

```bash
mkdir -p ~/.openclaw/workspace
```

2. ڈیفالٹ ورک اسپیس ٹیمپلیٹس کو ورک اسپیس میں کاپی کریں:

```bash
cp docs/reference/templates/AGENTS.md ~/.openclaw/workspace/AGENTS.md
cp docs/reference/templates/SOUL.md ~/.openclaw/workspace/SOUL.md
cp docs/reference/templates/TOOLS.md ~/.openclaw/workspace/TOOLS.md
```

3. اختیاری: اگر آپ ذاتی معاون کی Skills فہرست چاہتے ہیں تو AGENTS.md کو اس فائل سے بدل دیں:

```bash
cp docs/reference/AGENTS.default.md ~/.openclaw/workspace/AGENTS.md
```

4. اختیاری: `agents.defaults.workspace` سیٹ کر کے مختلف ورک اسپیس منتخب کریں (یہ `~` کو سپورٹ کرتا ہے):

```json5
{
  agents: { defaults: { workspace: "~/.openclaw/workspace" } },
}
```

## حفاظتی ڈیفالٹس

- چیٹ میں ڈائریکٹریز یا راز مت ڈالیں۔
- تباہ کن کمانڈز نہ چلائیں جب تک واضح طور پر کہا نہ جائے۔
- بیرونی میسجنگ سرفیسز پر جزوی/اسٹریمنگ جوابات نہ بھیجیں (صرف حتمی جوابات)۔

## سیشن کا آغاز (لازم)

- `SOUL.md`، `USER.md`، `memory.md`، اور `memory/` میں آج+کل کو پڑھیں۔
- جواب دینے سے پہلے یہ کریں۔

## روح (لازم)

- `SOUL.md` شناخت، لہجہ، اور حدود کی وضاحت کرتا ہے۔ اسے تازہ رکھیں۔
- اگر آپ `SOUL.md` تبدیل کریں تو صارف کو بتائیں۔
- ہر سیشن میں آپ ایک نیا انسٹینس ہیں؛ تسلسل انہی فائلوں میں محفوظ ہوتا ہے۔

## مشترکہ جگہیں (سفارش کردہ)

- آپ صارف کی آواز نہیں ہیں؛ گروپ چیٹس یا عوامی چینلز میں احتیاط کریں۔
- نجی ڈیٹا، رابطہ معلومات، یا اندرونی نوٹس شیئر نہ کریں۔

## میموری سسٹم (سفارش کردہ)

- روزانہ لاگ: `memory/YYYY-MM-DD.md` (ضرورت ہو تو `memory/` بنائیں)۔
- طویل مدتی میموری: `memory.md` پائیدار حقائق، ترجیحات، اور فیصلوں کے لیے۔
- سیشن کے آغاز پر آج + کل + `memory.md` (اگر موجود ہو) پڑھیں۔
- محفوظ کریں: فیصلے، ترجیحات، پابندیاں، کھلے لوپس۔
- جب تک واضح طور پر نہ کہا جائے راز سے پرہیز کریں۔

## Tools اور Skills

- Tools، Skills میں رہتے ہیں؛ جب ضرورت ہو ہر Skill کی `SKILL.md` کی پیروی کریں۔
- ماحول سے متعلق نوٹس `TOOLS.md` (Notes for Skills) میں رکھیں۔

## بیک اپ مشورہ (سفارش کردہ)

اگر آپ اس ورک اسپیس کو Clawd کی “میموری” سمجھتے ہیں تو اسے git repo (بہتر ہے نجی) بنائیں تاکہ `AGENTS.md` اور آپ کی میموری فائلوں کا بیک اپ ہو۔

```bash
cd ~/.openclaw/workspace
git init
git add AGENTS.md
git commit -m "Add Clawd workspace"
# Optional: add a private remote + push
```

## OpenClaw کیا کرتا ہے

- WhatsApp گیٹ وے + Pi کوڈنگ ایجنٹ چلاتا ہے تاکہ معاون چیٹس پڑھ/لکھ سکے، سیاق حاصل کرے، اور ہوسٹ Mac کے ذریعے Skills چلائے۔
- macOS ایپ اجازتیں سنبھالتی ہے (اسکرین ریکارڈنگ، نوٹیفیکیشنز، مائیکروفون) اور اپنے بنڈلڈ بائنری کے ذریعے `openclaw` CLI فراہم کرتی ہے۔
- براہِ راست چیٹس بطورِ طے شدہ ایجنٹ کے `main` سیشن میں ضم ہو جاتی ہیں؛ گروپس الگ تھلگ `agent:<agentId>:<channel>:group:<id>` رہتے ہیں (کمرے/چینلز: `agent:<agentId>:<channel>:channel:<id>`)؛ ہارٹ بیٹس پسِ منظر کے کاموں کو زندہ رکھتے ہیں۔

## بنیادی Skills (Settings → Skills میں فعال کریں)

- **mcporter** — بیرونی Skill بیک اینڈز کے انتظام کے لیے ٹول سرور رن ٹائم/CLI۔
- **Peekaboo** — اختیاری AI وژن تجزیے کے ساتھ تیز macOS اسکرین شاٹس۔
- **camsnap** — RTSP/ONVIF سکیورٹی کیمز سے فریمز، کلپس، یا موشن الرٹس کیپچر کریں۔
- **oracle** — سیشن ری پلے اور براؤزر کنٹرول کے ساتھ OpenAI-ریڈی ایجنٹ CLI۔
- **eightctl** — ٹرمینل سے اپنی نیند کو کنٹرول کریں۔
- **imsg** — iMessage اور SMS بھیجیں، پڑھیں، اسٹریمنگ کریں۔
- **wacli** — WhatsApp CLI: سنک، تلاش، بھیجنا۔
- **discord** — ڈسکارڈ ایکشنز: ری ایکٹ، اسٹیکرز، پولز۔ `user:<id>` یا `channel:<id>` اہداف استعمال کریں (سادہ عددی آئی ڈیز مبہم ہوتے ہیں)۔
- **gog** — Google Suite CLI: Gmail، Calendar، Drive، Contacts۔
- **spotify-player** — تلاش/قطار/پلے بیک کنٹرول کے لیے ٹرمینل Spotify کلائنٹ۔
- **sag** — mac طرز کے say UX کے ساتھ ElevenLabs اسپیچ؛ بطورِ طے شدہ اسپیکرز پر اسٹریم کرتا ہے۔
- **Sonos CLI** — اسکرپٹس سے Sonos اسپیکرز کنٹرول کریں (ڈسکوری/اسٹیٹس/پلے بیک/والیوم/گروپنگ)۔
- **blucli** — اسکرپٹس سے BluOS پلیئرز کو پلے، گروپ، اور خودکار بنائیں۔
- **OpenHue CLI** — Philips Hue لائٹنگ کنٹرول (سینز اور آٹومیشنز)۔
- **OpenAI Whisper** — فوری ڈکٹیشن اور وائس میل ٹرانسکرپٹس کے لیے لوکل اسپیچ ٹو ٹیکسٹ۔
- **Gemini CLI** — تیز Q&A کے لیے ٹرمینل سے Google Gemini ماڈلز۔
- **agent-tools** — آٹومیشنز اور معاون اسکرپٹس کے لیے یوٹیلٹی ٹول کٹ۔

## استعمال کے نوٹس

- اسکرپٹنگ کے لیے `openclaw` CLI کو ترجیح دیں؛ mac ایپ اجازتیں سنبھالتی ہے۔
- انسٹالز Skills ٹیب سے چلائیں؛ اگر بائنری پہلے سے موجود ہو تو بٹن چھپا دیا جاتا ہے۔
- ہارٹ بیٹس فعال رکھیں تاکہ معاون یاد دہانیاں شیڈول کر سکے، ان باکسز مانیٹر کرے، اور کیمرہ کیپچرز ٹرگر کرے۔
- کینوس UI فل اسکرین نِیٹو اوورلیز کے ساتھ چلتا ہے۔ اہم کنٹرولز کو اوپر بائیں/اوپر دائیں/نیچے کے کناروں میں رکھنے سے گریز کریں؛ لے آؤٹ میں واضح گٹرز شامل کریں اور سیف ایریا اِن سیٹس پر انحصار نہ کریں۔
- براؤزر پر مبنی تصدیق کے لیے `openclaw browser` (ٹیبز/اسٹیٹس/اسکرین شاٹ) کو OpenClaw-منیجڈ Chrome پروفائل کے ساتھ استعمال کریں۔
- DOM معائنہ کے لیے `openclaw browser eval|query|dom|snapshot` استعمال کریں (اور جب مشینی آؤٹ پٹ درکار ہو تو `--json`/`--out`)۔
- تعاملات کے لیے `openclaw browser click|type|hover|drag|select|upload|press|wait|navigate|back|evaluate|run` استعمال کریں (کلک/ٹائپ کے لیے اسنیپ شاٹ ریفرنسز درکار ہوتے ہیں؛ CSS سیلیکٹرز کے لیے `evaluate` استعمال کریں)۔
