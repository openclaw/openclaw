# ٹائپنگ اشارے

42. استعمال کریں
    `agents.defaults.typingMode` یہ کنٹرول کرنے کے لیے کہ ٹائپنگ **کب** شروع ہو اور `typingIntervalSeconds`
    یہ کنٹرول کرنے کے لیے کہ یہ **کتنی بار** ریفریش ہو۔ 43. `thinking` صرف اسی صورت میں فائر ہوتا ہے جب رن reasoning اسٹریم کرے (`reasoningLevel: "stream"`)۔

## بطورِ طے شدہ ترتیبات

جب `agents.defaults.typingMode` **سیٹ نہ ہو**، تو OpenClaw پرانا (لیگیسی) رویّہ برقرار رکھتا ہے:

- **براہِ راست چیٹس**: ماڈل لوپ شروع ہوتے ہی ٹائپنگ فوراً شروع ہو جاتی ہے۔
- **ذکر (mention) کے ساتھ گروپ چیٹس**: ٹائپنگ فوراً شروع ہو جاتی ہے۔
- **بغیر ذکر کے گروپ چیٹس**: ٹائپنگ صرف تب شروع ہوتی ہے جب پیغام کا متن اسٹریمنگ شروع کرے۔
- **ہارٹ بیٹ رنز**: ٹائپنگ غیرفعال ہوتی ہے۔

## موڈز

`agents.defaults.typingMode` کو درجِ ذیل میں سے کسی ایک پر سیٹ کریں:

- `never` — کبھی بھی ٹائپنگ اشارہ نہیں۔
- `instant` — **جیسے ہی ماڈل لوپ شروع ہو** ٹائپنگ شروع کریں، چاہے رن بعد میں صرف خاموش جواب ٹوکن ہی کیوں نہ واپس کرے۔
- `thinking` — **پہلے reasoning ڈیلٹا** پر ٹائپنگ شروع کریں (رن کے لیے
  `reasoningLevel: "stream"` درکار ہے)۔
- `message` — **پہلے غیر خاموش متنی ڈیلٹا** پر ٹائپنگ شروع کریں (یہ
  `NO_REPLY` خاموش ٹوکن کو نظرانداز کرتا ہے)۔

“کتنی جلدی فائر ہوتا ہے” کی ترتیب:
`never` → `message` → `thinking` → `instant`

## کنفیگریشن

```json5
{
  agent: {
    typingMode: "thinking",
    typingIntervalSeconds: 6,
  },
}
```

آپ فی سیشن موڈ یا کیڈینس کو اوور رائیڈ کر سکتے ہیں:

```json5
{
  session: {
    typingMode: "message",
    typingIntervalSeconds: 4,
  },
}
```

## نوٹس

- `message` موڈ خاموش-صرف جوابات کے لیے ٹائپنگ نہیں دکھائے گا (مثلاً آؤٹ پٹ دبانے کے لیے استعمال ہونے والا `NO_REPLY` ٹوکن)۔
- 44. اگر ماڈل reasoning deltas خارج نہیں کرتا تو ٹائپنگ شروع نہیں ہوگی۔
  45. اگر ماڈل ریزننگ ڈیلٹاز خارج نہ کرے، تو ٹائپنگ شروع نہیں ہوگی۔
- ہارٹ بیٹس کسی بھی موڈ میں ٹائپنگ نہیں دکھاتے۔
- 46. ڈیفالٹ 6 سیکنڈ ہے۔
  47. چیٹس میں `/status`: ایموجی‑سے بھرپور اسٹیٹس کارڈ جس میں سیشن ٹوکنز + اندازاً لاگت (صرف API کی) شامل ہے۔
