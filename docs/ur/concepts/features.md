## Full list

- WhatsApp Web (Baileys) کے ذریعے WhatsApp انٹیگریشن
- Telegram بوٹ سپورٹ (grammY)
- Discord بوٹ سپورٹ (channels.discord.js)
- Mattermost بوٹ سپورٹ (plugin)
- local imsg CLI (macOS) کے ذریعے iMessage انٹیگریشن
- ٹول اسٹریمنگ کے ساتھ RPC موڈ میں Pi کے لیے ایجنٹ برج
- طویل جوابات کے لیے اسٹریمنگ اور chunking
- ہر ورک اسپیس یا ارسال کنندہ کے لیے الگ تھلگ سیشنز کے ساتھ ملٹی ایجنٹ روٹنگ
- OAuth کے ذریعے Anthropic اور OpenAI کے لیے سبسکرپشن تصدیق
- سیشنز: براہِ راست چیٹس مشترکہ `main` میں ضم ہو جاتی ہیں؛ گروپس الگ تھلگ ہوتے ہیں
- مینشن کی بنیاد پر ایکٹیویشن کے ساتھ گروپ چیٹ سپورٹ
- تصاویر، آڈیو، اور دستاویزات کے لیے میڈیا سپورٹ
- اختیاری وائس نوٹ ٹرانسکرپشن ہُک
- WebChat اور macOS مینو بار ایپ
- pairing اور Canvas سطح کے ساتھ iOS نوڈ
- pairing، Canvas، چیٹ، اور کیمرہ کے ساتھ Android نوڈ
