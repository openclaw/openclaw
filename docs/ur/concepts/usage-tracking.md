# استعمال کی نگرانی

## یہ کیا ہے

- فراہم کنندہ کے استعمال/کوٹہ کو براہِ راست ان کے استعمال اینڈپوائنٹس سے حاصل کرتا ہے۔
- کوئی تخمینی لاگت نہیں؛ صرف فراہم کنندہ کی رپورٹ کردہ ونڈوز۔

## یہ کہاں ظاہر ہوتی ہے

- `/status` چیٹس میں: ایموجی سے بھرپور اسٹیٹس کارڈ جس میں سیشن ٹوکنز + تخمینی لاگت (صرف API کلید) شامل ہوتی ہے۔ جہاں دستیاب ہو وہاں **موجودہ ماڈل فراہم کنندہ** کے لیے فراہم کنندہ کا استعمال دکھایا جاتا ہے۔
- `/usage off|tokens|full` چیٹس میں: ہر جواب کے لیے استعمال فوٹر (OAuth صرف ٹوکنز دکھاتا ہے)۔
- `/usage cost` چیٹس میں: OpenClaw سیشن لاگز سے مجموعی مقامی لاگت کا خلاصہ۔
- CLI: `openclaw status --usage` ہر فراہم کنندہ کی مکمل تفصیل پرنٹ کرتا ہے۔
- CLI: `openclaw channels list` فراہم کنندہ کنفیگ کے ساتھ وہی استعمال اسنیپ شاٹ پرنٹ کرتا ہے (اسے چھوڑنے کے لیے `--no-usage` استعمال کریں)۔
- macOS مینو بار: Context کے تحت “Usage” سیکشن (صرف جب دستیاب ہو)۔

## فراہم کنندگان + اسناد

- **Anthropic (Claude)**: auth پروفائلز میں OAuth ٹوکنز۔
- **GitHub Copilot**: auth پروفائلز میں OAuth ٹوکنز۔
- **Gemini CLI**: auth پروفائلز میں OAuth ٹوکنز۔
- **Antigravity**: auth پروفائلز میں OAuth ٹوکنز۔
- **OpenAI Codex**: auth پروفائلز میں OAuth ٹوکنز (جب موجود ہو تو accountId استعمال ہوتا ہے)۔
- **MiniMax**: API کلید (کوڈنگ پلان کلید؛ `MINIMAX_CODE_PLAN_KEY` یا `MINIMAX_API_KEY`)؛ 5 گھنٹے کے کوڈنگ پلان ونڈو کو استعمال کرتا ہے۔
- **z.ai**: env/config/auth اسٹور کے ذریعے API کلید۔

اگر مطابقت رکھنے والی OAuth/API اسناد موجود نہ ہوں تو استعمال چھپا دیا جاتا ہے۔
