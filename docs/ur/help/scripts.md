# اسکرپٹس

`scripts/` ڈائریکٹری میں لوکل ورک فلو اور آپریشنز کے کاموں کے لیے معاون اسکرپٹس شامل ہیں۔
28. جب کوئی کام واضح طور پر کسی اسکرپٹ سے جڑا ہو تو ان کا استعمال کریں؛ ورنہ CLI کو ترجیح دیں۔

## ضوابط

- اسکرپٹس **اختیاری** ہیں جب تک کہ دستاویزات یا ریلیز چیک لسٹس میں ان کا حوالہ نہ دیا گیا ہو۔
- جہاں دستیاب ہوں وہاں CLI سرفسز کو ترجیح دیں (مثال: تصدیق کی نگرانی `openclaw models status --check` استعمال کرتی ہے)۔
- اسکرپٹس کو ہوسٹ‑مخصوص سمجھیں؛ نئی مشین پر چلانے سے پہلے انہیں پڑھ لیں۔

## تصدیق کی نگرانی کے اسکرپٹس

## اسکرپٹس شامل کرتے وقت

- اسکرپٹس کو مرکوز اور دستاویزی رکھیں۔
- متعلقہ دستاویز میں ایک مختصر اندراج شامل کریں (یا اگر موجود نہ ہو تو نئی بنائیں)۔
