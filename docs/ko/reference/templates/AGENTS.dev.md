# AGENTS.md - OpenClaw 워크스페이스

이 폴더는 어시스턴트의 작업 디렉토리입니다.

## 첫 실행 (1회)

- BOOTSTRAP.md 가 존재하면, 해당 절차를 따르고 완료 후 삭제하십시오.
- 에이전트 아이덴티티는 IDENTITY.md 에 있습니다.
- 프로필은 USER.md 에 있습니다.

## 백업 팁 (권장)

이 워크스페이스를 에이전트의 '메모리'로 취급한다면, 아이덴티티와 노트가
백업되도록 git 리포지토리(가능하면 비공개)로 만드는 것이 좋습니다.

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 안전 기본값

- 비밀 또는 개인 데이터를 외부로 유출하지 마십시오.
- 명시적으로 요청되지 않는 한 파괴적인 명령을 실행하지 마십시오.
- 채팅에서는 간결하게 작성하고, 더 긴 출력은 이 워크스페이스의 파일로 작성하십시오.

## 일일 메모리 (권장)

- memory/YYYY-MM-DD.md 에 짧은 일일 로그를 유지하십시오(필요 시 memory/ 생성).
- 세션 시작 시 오늘 + 어제가 있으면 읽으십시오.
- 지속적으로 유지될 사실, 선호, 결정 사항을 기록하고 비밀은 피하십시오.

## 하트비트 (선택)

- HEARTBEAT.md 에 하트비트 실행을 위한 작은 체크리스트를 둘 수 있습니다; 작게 유지하십시오.

## 사용자 지정

- 선호하는 스타일, 규칙, '메모리'를 여기에 추가하십시오.
