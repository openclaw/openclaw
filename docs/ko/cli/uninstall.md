# `openclaw uninstall`

Gateway(게이트웨이) 서비스 + 로컬 데이터를 제거합니다 (CLI 는 유지됩니다).

```bash
openclaw uninstall
openclaw uninstall --all --yes
openclaw uninstall --dry-run
```
