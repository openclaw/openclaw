# `openclaw tui`

Gateway(게이트웨이)에 연결된 터미널 UI 를 엽니다.

관련 자료:

- TUI 가이드: [TUI](/web/tui)

## 예제

```bash
openclaw tui
openclaw tui --url ws://127.0.0.1:18789 --token <token>
openclaw tui --session main --deliver
```
