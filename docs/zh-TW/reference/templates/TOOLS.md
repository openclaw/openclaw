# TOOLS.md - 本機筆記

Skills 定義了工具的_運作方式_。此檔案用於_您_的特定內容 — 那些專屬於您設定的資訊。

## 應放置於此的內容

諸如：

- 相機名稱與位置
- SSH 主機與別名
- TTS 的偏好語音
- 揚聲器/房間名稱
- 裝置暱稱
- 任何環境特定的內容

## 範例

```markdown
### 相機

- 客廳 → 主要區域，180° 廣角
- 前門 → 入口，動作觸發

### SSH

- home-server → 192.168.1.100, 使用者: admin

### TTS

- 偏好語音: "Nova" (溫暖，略帶英式口音)
- 預設揚聲器: 廚房 HomePod
```

## 為何要區分？

Skills 是共享的。您的設定是您自己的。將它們區分開來意味著您可以在不丟失筆記的情況下更新 Skills，並在不洩露基礎設施的情況下共享 Skills。
