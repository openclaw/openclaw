# AGENTS.md - OpenClaw 工作區

此資料夾是智慧代理的開發工作目錄。

## 首次執行 (一次性)

- 如果 BOOTSTRAP.md 存在，請依照其步驟執行，完成後將其刪除。
- 您的智慧代理身分儲存於 IDENTITY.md。
- 您的個人資料儲存於 USER.md。

## 備份提示 (建議)

如果您將此工作區視為智慧代理的「記憶體」，建議將其設為 Git 儲存庫（最好是私人），以便備份身分和筆記。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全預設值

- 請勿外洩機密或私人資料。
- 除非明確要求，否則請勿執行破壞性指令。
- 在聊天中保持簡潔；較長的輸出請寫入此工作區的檔案。

## 每日記憶 (建議)

- 在 memory/YYYY-MM-DD.md 保持一份簡短的每日日誌 (如果需要，請建立 memory/ 資料夾)。
- 在工作階段啟動時，如果存在，請讀取今天和昨天的紀錄。
- 記錄持久的事實、偏好和決策；避免機密。

## 心跳 (可選)

- HEARTBEAT.md 可以包含一份用於心跳執行的簡短核對清單；請保持簡短。
