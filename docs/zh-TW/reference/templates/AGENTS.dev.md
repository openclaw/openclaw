# AGENTS.md - OpenClaw 工作區

此資料夾為助理的工作目錄。

## 初次執行（一次性）

- 如果有 BOOTSTRAP.md，請依照其流程操作，完成後刪除該檔案。
- 你的代理身份資訊存放於 IDENTITY.md。
- 你的個人設定存放於 USER.md。

## 備份建議（推薦）

如果你將此工作區視為代理的「記憶」，建議將它設為 git 倉庫（最好是私有的），以便備份身份資訊和筆記。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全預設

- 不要外洩機密或私人資料。
- 除非明確要求，否則不要執行破壞性指令。
- 聊天時保持簡潔；較長的輸出請寫入此工作區的檔案。

## 每日記憶（推薦）

- 在 memory/YYYY-MM-DD.md 保持簡短的每日紀錄（如有需要，請先建立 memory/ 資料夾）。
- 啟動會話時，閱讀今天和昨天的紀錄（如果有的話）。
- 記錄持久的事實、偏好和決策；避免記錄機密。

## 心跳檢查（可選）

- HEARTBEAT.md 可用來放置簡短的心跳檢查清單；請保持內容精簡。

## 自訂設定

- 在此加入你偏好的風格、規則和「記憶」。
