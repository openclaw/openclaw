# AGENTS.md - OpenClaw 工作區

此資料夾是智慧代理的工作目錄。

## 初次執行（僅限一次）

- 如果 BOOTSTRAP.md 存在，請按照其程序執行並在完成後刪除。
- 您的智慧代理身分儲存在 IDENTITY.md。
- 您的個人資料儲存在 USER.md。

## 備份建議（推薦）

如果您將此工作區視為智慧代理的「記憶體」，請將其建立為 git 儲存庫（建議設為私有），以便備份身分與筆記。

```bash
git init
git add AGENTS.md
git commit -m "Add agent workspace"
```

## 安全預設值

- 請勿洩漏金鑰或私密資料。
- 除非明確要求，否則請勿執行具破壞性的指令。
- 對話請保持簡潔；較長的輸出請寫入此工作區的檔案中。

## 每日記憶體（推薦）

- 在 memory/YYYY-MM-DD.md 保存簡短的每日日誌（如有需要請建立 memory/ 目錄）。
- 在工作階段開始時，讀取今天與昨天的內容（如果存在）。
- 記錄持久的事實、偏好與決定；避免記錄金鑰。

## 心跳檢查 (Heartbeats)（選填）

- HEARTBEAT.md 可以存放用於心跳執行的極簡檢查清單；請保持內容精簡。

## 自定義

- 在此處加入您偏好的風格、規則與「記憶體」。
