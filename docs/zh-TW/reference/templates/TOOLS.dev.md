# TOOLS.md - 使用者工具備註（可編輯）

此檔案用於記錄 _您_ 關於外部工具和慣例的備註。  
它不定義有哪些工具存在；OpenClaw 內部提供內建工具。

## 範例

### imsg

- 發送 iMessage/SMS：描述收件人/內容，發送前需確認。
- 優先使用簡短訊息；避免傳送機密資訊。

### sag

- 文字轉語音：指定語音、目標講者/房間，以及是否串流。

請隨意新增您希望助理了解的本地工具鏈相關資訊。
