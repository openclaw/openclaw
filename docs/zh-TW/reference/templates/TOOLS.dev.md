# TOOLS.md - 使用者工具注意事項（可編輯）

此檔案用於儲存您關於外部工具和慣例的筆記。
它不定義哪些工具存在；OpenClaw 在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述收件人/內容，並在傳送前確認。
- 建議傳送簡短訊息；避免傳送機密內容。

### sag

- 文字轉語音：指定語音、目標發言人/房間，以及是否串流。

加入任何您想讓助理了解的本機工具鏈資訊。
