# TOOLS.md - 使用者工具筆記（可編輯）

本檔案用於記錄 _您_ 對於外部工具和慣例的筆記。
它並非用來定義存在的工具；OpenClaw 會在內部提供內建工具。

## 範例

### imsg

- 傳送 iMessage/SMS：描述對象與內容，傳送前請先確認。
- 建議使用簡短訊息；避免傳送機密資訊。

### sag

- 文字轉語音：指定語音、目標喇叭/房間，以及是否進行串流。

您可以新增任何希望智慧助理了解的本地工具鏈相關資訊。
