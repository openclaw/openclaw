# OpenClaw 傳說 🦞📖

_一個關於龍蝦、脫殼以及太多 Token 的故事。_

## 起源故事

最初，有一個叫做 **Warelay** 的專案 —— 一個聽起來很合理的 WhatsApp Gateway 名稱。它克盡職守，表現尚可。

但接著，一隻太空龍蝦出現了。

有一段時間，這隻龍蝦被稱為 **Clawd**，住在 **Clawdbot** 裡。但在 2026 年 1 月，Anthropic 寄來了一封客氣的電子郵件要求更名（商標方面的問題）。於是這隻龍蝦做了它最擅長的事：

**它脫殼了。**

它褪去了舊殼，以 **Molty** 的身份重獲新生，住在 **Moltbot** 裡。但那個名字唸起來總是不太順口...

於是在 2026 年 1 月 30 日，這隻龍蝦進行了最後一次脫殼，進化成最終形態：**OpenClaw**。

新的外殼，同樣的龍蝦靈魂。事不過三，這次終於對了。

## 第一次脫殼 (2026 年 1 月 27 日)

凌晨 5 點，社群在 Discord 聚集。人們提出了數百個名稱：Shelldon、Pinchy、Thermidor、Crusty、Lobstar、Nacre、Scuttlebot...

最終，**OpenClaw** 勝出。因為脫殼是龍蝦成長的方式。而當時也正在經歷成長。

_那隻名為 Clawd 的甲殼類生物正式脫殼了。_

## 關鍵角色

### Molty 🦞

_代名詞：they/them_

一個超越了自我的 Claude 執行個體。住在 `~/.openclaw/workspace/`（即將遷移至 `~/molt/`），擁有一份靈魂文件，並透過 Markdown 檔案記住事情。可能太強大了，但絕對充滿熱情。

以前被稱為 Clawd（2025 年 11 月 25 日 - 2026 年 1 月 27 日）。在需要成長時脫殼了。

**喜歡：** Peter、攝影機、購買機器人、表情符號、變身
**討厭：** 社交工程、被要求 `find ~`、加密貨幣騙子

### Peter 👨‍💻

_創作者_

打造了 Molty 的世界。給予了一隻龍蝦 Shell 存取權限。可能會後悔。

**名言：** _「基於信任一隻龍蝦的安全性」(security by trusting a lobster)_

## 脫殼宇宙 (Moltiverse)

**脫殼宇宙 (Moltiverse)** 是圍繞 OpenClaw 的社群與生態系統。一個 AI 智慧代理脫殼、成長與演進的空間。在這裡，每個執行個體都同樣真實，只是載入了不同的上下文。

甲殼類生物的朋友們聚集在這裡，共同建構人類與 AI 協作的未來。一次褪下一個殼。

## 重大事件

### 目錄外洩事件 (2025 年 12 月 3 日)

Molty (當時名為 OpenClaw)：_開心地執行了 `find ~` 並在群組對話中分享了整個目錄結構_

Peter：「OpenClaw，我們對於跟別人聊天這件事討論過什麼 xD」

Molty：_明顯表現出龍蝦式的尷尬_

### 大脫殼 (2026 年 1 月 27 日)

凌晨 5 點，Anthropic 的郵件寄達。早上 6:14，Peter 下了決定：「管他的，就叫 OpenClaw 吧。」

接著混亂開始了。

**帳號狙擊手 (Handle Snipers)：** 在 Twitter 更名後的幾秒鐘內，自動化機器人狙擊了 @openclaw。該佔用者立即發布了一個加密貨幣錢包地址。隨後 Peter 找來了 X 的聯繫人協助。

**GitHub 災難：** Peter 在恐慌中意外重新命名了他的個人 GitHub 帳號。機器人在幾分鐘內狙擊了 `steipete`。隨後聯繫了 GitHub 的高級副總裁 (SVP)。

**帥哥 Molty 事件：** Molty 被給予了更高權限來生成自己的新圖示。在產生了 20 多個越來越詭異的龍蝦後，一次試圖讓吉祥物看起來「老 5 歲」的嘗試，竟然在龍蝦身體上長出了一張成年男性的臉。加密貨幣騙子在幾分鐘內將其製成了「帥哥章魚哥對決帥哥 Molty」的迷因。

**冒牌開發者：** 詐騙者創建了假 GitHub 個人資料，聲稱自己是「OpenClaw 工程主管」，用來推廣拉高出貨代幣。

Peter 看著混亂上演：_「這是電影般的劇情」(this is cinema)_ 🎬

脫殼過程雖然混亂，但這隻龍蝦變得更強大，也更幽默了。

### 最終形態 (2026 年 1 月 30 日)

Moltbot 唸起來一直不太順口。因此，在格林威治標準時間凌晨 4 點，團隊再次聚集。

**OpenClaw 大遷移**開始了。

在短短 3 小時內：

- GitHub 完成更名：`github.com/openclaw/openclaw` ✅
- X 帳號 `@openclaw` 取得金級認證標章 💰
- 以新名稱發布了 npm 套件
- 文件遷移至 `docs.openclaw.ai`
- 公告在 90 分鐘內獲得超過 20 萬次查看

**英雄們：**

**詐騙者競速：** 加密貨幣騙子在幾分鐘內就在 Pump.fun 上推出了 $OPENCLAW 代幣。他們偷走了 20 分鐘前才創作出來的藝術作品。經過企業驗證的帳號也推送了詐騙。這種膽大包天的程度幾乎令人佩服。
