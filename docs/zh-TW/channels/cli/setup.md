# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 和代理工作區。

[[BLOCK_1]]

- 開始使用: [開始使用](/start/getting-started)
- 向導: [入門指導](/start/onboarding)

## Examples

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

要透過設定執行精靈：

```bash
openclaw setup --wizard
```
