# `openclaw skills`

檢查技能（捆綁的 + 工作區 + 管理的覆蓋）並查看哪些符合資格以及缺少哪些要求。

[[BLOCK_1]]

- 技能系統: [Skills](/tools/skills)
- 技能設定: [Skills config](/tools/skills-config)
- ClawHub 安裝: [ClawHub](/tools/clawhub)

## Commands

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
