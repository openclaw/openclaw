# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 及代理工作區。

相關連結：

- 入門指南：[Getting started](/start/getting-started)
- 精靈導引：[Onboarding](/start/onboarding)

## 範例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

透過設定執行精靈導引：

```bash
openclaw setup --wizard
```
