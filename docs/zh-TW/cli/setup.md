# `openclaw setup`

初始化 `~/.openclaw/openclaw.json` 與智慧代理工作區。

相關內容：

- 入門指南：[入門指南](/start/getting-started)
- 精靈：[新手導覽](/start/onboarding)

## 範例

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

若要透過 setup 執行精靈：

```bash
openclaw setup --wizard
```
