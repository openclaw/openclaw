# `openclaw pairing`

核准或檢視 DM 配對請求（適用於支援配對的頻道）。

1. 相關：

- 配對流程：[Pairing](/channels/pairing)

## 指令

```bash
openclaw pairing list whatsapp
openclaw pairing approve whatsapp <code> --notify
```
