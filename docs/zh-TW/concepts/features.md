## 完整列表

- 透過 WhatsApp Web (Baileys) 整合 WhatsApp
- 支援 Telegram 機器人 (grammY)
- 支援 Discord 機器人 (channels.discord.js)
- 支援 Mattermost 機器人（外掛程式）
- 透過本機 imsg CLI (macOS) 整合 iMessage
- Pi 在 RPC 模式下與工具串流傳輸的智慧代理橋接
- 長回應的串流傳輸和區塊串流傳輸
- 每個工作區或傳送者的多智慧代理路由，用於隔離的工作階段
- 透過 OAuth 訂閱 Anthropic 和 OpenAI 的憑證
- 工作階段：直接聊天會合併到共享的 `main`；群組會被隔離
- 支援群組聊天與提及式啟用
- 支援圖片、音訊和檔案等媒體
- 可選的語音備忘錄轉錄掛鉤
- WebChat 和 macOS 選單列應用程式
- 帶有配對和 Canvas 界面的 iOS 節點
- 帶有配對、Canvas、聊天和相機的 Android 節點

<Note>
舊版的 Claude、Codex、Gemini 和 Opencode 路徑已移除。Pi 是唯一的程式設計智慧代理路徑。
</Note>
