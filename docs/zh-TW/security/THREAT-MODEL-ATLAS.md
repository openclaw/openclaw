## 1. 介紹

### 1.1 目的

本威脅模型記錄 OpenClaw AI 代理平台及 ClawHub 技能市集的對抗性威脅，採用專為 AI/ML 系統設計的 MITRE ATLAS 框架。

### 1.2 範圍

| 元件                   | 是否包含 | 備註                                          |
| ---------------------- | -------- | --------------------------------------------- |
| OpenClaw Agent Runtime | 是       | 核心代理執行、工具呼叫、會話管理              |
| Gateway                | 是       | 認證、路由、頻道整合                          |
| Channel Integrations   | 是       | WhatsApp、Telegram、Discord、Signal、Slack 等 |
| ClawHub Marketplace    | 是       | 技能發佈、審核、分發                          |
| MCP Servers            | 是       | 外部工具提供者                                |
| User Devices           | 部分     | 行動應用程式、桌面用戶端                      |

### 1.3 範圍外

此威脅模型未明確定義任何範圍外專案。

| F3   | Agent   | 工具    | 工具呼叫       | 政策執行       |
