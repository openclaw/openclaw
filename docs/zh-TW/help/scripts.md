# 腳本

`scripts/` 目錄包含用於本機工作流程和維運任務的輔助腳本。
當任務明確綁定到腳本時使用它們；否則，請優先使用 CLI。

## 慣例

- 腳本是**可選的**，除非在文件或發布檢查清單中被引用。
- 優先使用現有的 CLI 介面（範例：憑證監控使用 `openclaw models status --check`）。
- 假定腳本是主機專用的；在新機器上執行之前請先閱讀它們。

## 新增腳本時

- 保持腳本專注且有文件說明。
- 在相關文件中新增一個簡短條目（如果缺少則建立一個）。
