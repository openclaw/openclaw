# 腳本

`scripts/` 目錄包含了用於本地工作流程與運維任務的輔助腳本。當任務與特定腳本明確相關時，請使用這些腳本；否則請優先使用 CLI。

## 慣例

- 除非文件或發布檢查清單中另有說明，否則腳本皆為**選填**。
- 若已存在對應的 CLI 介面，請優先使用之（例如：憑證監控使用 `openclaw models status --check`）。
- 請假設腳本是特定於主機的；在新的機器上執行前，請先閱讀其內容。

## 新增腳本時

- 保持腳本功能專注並附上說明文件。
- 在相關文件中加入簡短的說明（若缺少文件則請建立一個）。
