# Venice AI (Venice highlight)

**Venice** 是我們重點推薦的 Venice 設定，適用於隱私優先的推論，並可選擇性地透過匿名方式存取專有模型。

Venice AI 提供隱私導向的 AI 推論，支援無審查模型，並能透過其匿名代理存取主要的專有模型。所有推論預設皆為私有的——不會對您的資料進行訓練，也不會記錄記錄。

## 為什麼在 OpenClaw 中使用 Venice

- **私有推論**：適用於開源模型（無記錄）。
- **無審查模型**：在您需要時提供。
- **匿名存取**：在注重品質時存取專有模型（Opus/GPT/Gemini）。
- 與 OpenAI 相容的 `/v1` 端點。
