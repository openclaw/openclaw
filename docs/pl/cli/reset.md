# `openclaw reset`

Resetowanie lokalnej konfiguracji/stanu (zachowuje zainstalowane CLI).

```bash
openclaw reset
openclaw reset --dry-run
openclaw reset --scope config+creds+sessions --yes --non-interactive
```
