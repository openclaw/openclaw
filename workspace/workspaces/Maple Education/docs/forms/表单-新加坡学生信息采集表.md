# 新加坡学生信息采集表（模板）

> 用途：给客户/家长填写，用于申请与签证材料整理。

## 字段（整理版）
- 基本信息：姓名、性别、民族、电话、邮箱、地址
- 监护人信息：姓名、身份证号
- 最近 5 年住址（含时间段、邮编）
- 新加坡地址（如有，含时间段）
- 签证/入境历史确认：
  - 是否持不同护照/曾用名入境过新加坡
  - 申请者本人及父母是否有拒签史（学生签/旅游签/工作签等）
  - 是否近期办过旅游签
  - 是否犯罪记录/是否被禁止入境
  - 是否换过护照（如换过需扫描显示旧护照号页）
  - 是否同意父母通过邮件/短信/电话接收成绩单
  - 申请者本人及父母是否曾持有新加坡任意工作签证
  - 申请者本人是否持有新加坡学生签证（如有上传学生证正反面）
  - 是否有国外旅居史（提供居住时间与地址）
- 家庭成员表：姓名、国籍、关系、身份证号、护照号
- 选校信息：学校名称、申请专业、是否匹配语言班、开学时间
- 客户签字/日期（确认信息真实）
