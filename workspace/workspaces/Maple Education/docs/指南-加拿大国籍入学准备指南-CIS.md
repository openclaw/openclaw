# 加拿大国籍入学准备指南（CIS）

> 来源：加拿大国籍入学准备指南.pdf（转存为 Markdown，原文为指南文本抽取）

## 关键点（可直接给家长）
- 文具/设备：G4–G6 iPad；G7–G12 MacBook；G4–G12 需要 locker lock。
- 账号/课表：入学前会收 CIS 系统登录凭证邮件；登录 ManageBac 看课表。
- 午餐：Nursery–G3 需开学前在线订餐；G4–G12 用 stored-value 学生证。
- 校车：door-to-door；路线确认至少 2–3 周；邮件 cis@comfortdelgrobus.com.sg。
- 入校：建议第一天家长陪同；需要 Parent ID；每家免费 3 张，补办 $25。
