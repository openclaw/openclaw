## 0. Business Categories & Pricing Logic  
## 业务大类与收费逻辑

> 本节按你说的「大类 / 小类」+ 返佣逻辑来梳理，方便定价和对外话术，再跟后面的 S1–S7 客群视角互相对应。

**统一原则（费用走向）：**  
- Maple 收取的仅为**中介费 / 文书服务费 / 管家服务费**等服务费用；  
- 学费、住宿费、生活费等所有第三方费用，一律由学生/家长**直接打到学校或第三方账号**，不经 Maple 代收。  

### 0.1 私立大学（有返佣）

**大类：私立大学**  
- 小类：  
  - Kaplan  
  - PSB  
  - Amity  
  - 其他私立大学（参考返佣表，返佣结构类似的学校，SIM 除外）  

**收费逻辑：**  
- 主要收入来自院校返佣；  
- 向学生收取少量文书/服务费：  
  - 一般区间：**300–500 SGD**（SIM 可适当更高）；  
  - 目标是每单总收入（返佣 + 学生端收费）约 **2,000 SGD 左右**；  
- 运营要点：  
  - 私立大学可以在营销话术上给出「保录取」和「多校并行申请」（例如 UCD + PSB + Amity 组合保底）。  
  - 合同与法律文本中建议避免绝对承诺，用「如某校不录取，则协助转投其他备选院校」等方式表述。  

### 0.2 公立大学 / 零返佣项目

**大类：公立大学 & 零返佣**  
- 小类：  
  - NUS  
  - NTU  
  - AEIS / S-AEIS  
  - 第一梯队国际学校（顶级国际学校，通常无返佣）  
  - 本地幼儿园（无返佣或极低返佣）  

**收费逻辑：**  
- 这类项目几乎没有院校返佣，因此主要靠向学生收取申请服务费：  
  - 统一内部目标：**2,000 SGD / 单** 的申请服务费；  
  - 以新币计价，对中国客户如需可在报价中标示约等值 RMB（不作为主要对外口径）。  
- NUS / NTU：  
  - 按项目难度与背景提升工作量收取申请费，具体金额在合同中留空，由个案确定。  
- AEIS / 第一梯队国际学校 / 幼儿园：  
  - 视为「零返佣或极低返佣」项目，一般参考 2,000 SGD 的服务费标准；  
  - 其中**幼儿园择校服务**可采用固定 **SGD 2,000** 的标准价（见下文 S6）。  

### 0.3 低返佣项目

**大类：给返佣但是不多**  
- 小类：  
  - SIM  
  - 第二梯队国际学校（美国人国际学校、加拿大国际学校、澳洲国际学校等，通常返佣约 800 SGD 左右）  

**收费逻辑：**  
- 院校返佣约 **800 SGD**；  
- 仍以「每单总收入约 2,000 SGD」为目标：  
  - 对学生额外收取服务费 ≈ 1,200 SGD（可灵活调整）；  
  - 可单独列为「服务费 / 顾问费」，与 300–500 SGD 文书费分开说明。  

### 0.4 管家服务（Relocation / Concierge）

**大类：管家服务**  
- 小类示例（可细拆，用于「单项价目表」）：  
  - 接机 / 送机  
  - 临时住宿安排（酒店 / 服务式公寓推荐与预订协助）  
  - 找房（配合持牌房产中介或二房东公司）  
  - 开户 / 电话卡 / 银行 / 交通卡协助  
  - 学校报到陪同 / 家长陪同参观  
  - 日常杂务代办（如水电网开户、警局登记等）  

**收费逻辑：**  
- 你口头说明：**19,800 RMB** 为一个「一站式管家服务」的标杆价，可视为：  
  - 面向新来新加坡的学生 / 家庭的 **Relocation + 落地管家打包服务**；  
  - 可以附加给任何客户：留学、EP、移民、短期项目等。  
- 运营方式：  
  - 每个「杂活」单独定价，列出清单，客户可以自由组合成自己的管家套餐；  
  - 19,800 RMB 作为推荐的「全程打包价」（内部可折算成 SGD，方便对外统一口径）。  

### 0.5 移民咨询 & 监管类服务外包

**大类：移民 / 税务 / 招聘相关咨询**  

- ACRA / IRAS 相关业务：  
  - 公司注册、做账、报税等合规事项全部由**合作的持牌会计师事务所**执行；  
  - Maple 仅作转介和项目协调，相关收费直接由会计师事务所收取或与其对账。  

- MOM 招聘与 EA 牌照相关业务：  
  - 涉及招聘、雇佣中介等受 EA 牌照监管的工作，全部交给合作的 EA 公司；  
  - Maple 内部可以为雇主做招聘广告发布等简单 admin 工作，但**不以 EA 身份操作**；  
  - 如果需要以 EA 牌照对外展示，统一使用合作 EA 公司的品牌和牌照信息。  

- 租房与物业：  
  - 单间出租：转给合作的「二房东 / 物业管理公司」处理；  
  - 整租：转给合作的「新加坡持牌房产中介」处理；  
  - Maple 不直接以房产中介身份签约，只做对接与协同。  

**移民咨询收费（待定）：**  
- 建议结构（具体数额待你敲定）：  
  - 基础咨询：单次 60–90 分钟，收取固定咨询费（例如 X SGD），可抵扣后续全案服务费的一部分；  
  - 自雇 EP 全案规划：按项目收取一次性规划费（不含会计 / 秘书 / 政府规费，由第三方单独收费）；  
  - 自雇 EP 代办与长期规划：阶段性收费（签约、递交、获批、后续 PR 规划等里程碑节点）。  

### 0.6 课程辅导

**大类：课程 / 考试辅导**  
- 小类：  
  - AEIS / S-AEIS 学术辅导  
  - 私立大学预科 / 本科课程辅导  
  - 语言 / 标化考试（如雅思、托福等）辅导  

**收费逻辑（建议）：**  
- 以小时或课程包计费，根据科目难度和老师背景浮动；  
- Maple 可：  
  - 仅做转介绍（老师或机构直接与学生结算）；或  
  - 做平台 / 管理方（学生付费给 Maple，Maple 与教师结算），此时需在合同中明确 Maple 的角色和责任边界。

## 1. Client Segments Overview  
## 客户细分总览

**S1 – 新加坡私立大学路径（Kaplan / PSB / Amity 等）**  
- 典型画像：  
  - 中国初三 / 高一 / 高二 / 高三毕业生，目标希望尽快拿到新加坡或海外本科学历。  
  - 成绩较好的本科生，考虑通过“合作办学本科 / 荣誉学位 → NUS / NTU 硕士”路径。  
- 目标结果：  
  - 在 Kaplan / PSB / Amity 等私立院校完成大专 / 本科。  
  - 成绩优异者，以该学位为基础申请 NUS / NTU / 其他名校硕士（不承诺录取）。  
- 补充服务：  
  - 可搭配“境外管家服务”（生活/学习落地支持）。  
  - 毕业后可衔接“自雇 EP / 就业 & 移民规划”服务。  

**S2 – 低龄 K12 国际学校（幼儿园 ~ 高中）**  
- 典型画像：  
  - 有能力长期在新加坡或国际环境陪读/生活的家庭。  
  - 希望孩子在国际化体系（IB / IGCSE / AP 等）中成长，未来直申世界前 100 大学或新加坡公立大学。  
- 目标结果：  
  - 从幼儿园 / 小学一路在国际学校体系就读。  
  - 配合家庭身份路径（学生签证 + 家长陪读 / 后续工作或 EP/PR）。  

**S3 – 低龄 K12 公立学校 & AEIS / S-AEIS**  
- 典型画像：  
  - 以“新加坡政府小学 / 中学”为目标，希望走本地公立体系 + 之后 JC/理工/本科路径。  
  - 接受 AEIS / S-AEIS 考试不确定性，愿意结合国际学校 / 新二等 Plan B。  
- 目标结果：  
  - 通过 AEIS / S-AEIS 或其他途径进入政府小学 / 中学。  
  - 后续承接到 JC / 理工 / 本地大学或其他国家。  

**S4 – 新二 / 预科 / 衔接课程 & 合作办学本科**  
- 典型画像：  
  - 国内完成初中 / 高中部分学业，不能直接申请本科或希望降低难度/时间成本。  
  - 在意“整体路径性价比”，愿意通过新二 / 预科 / 合作办学本科绕道名校。  
- 目标结果：  
  - 顺利完成新二 / 预科 / 合作办学本科。  
  - 以此为基础申请新加坡或其他国家的本科/硕士。  

**S5 – 顶尖大学（NUS / NTU 等）本科 / 硕士申请**  
- 典型画像：  
  - 成绩优异（如“高考一本线 +50 分”或同等水平）的学生。  
  - 已有明确目标院校（NUS / NTU / SMU 等），需要从材料到网申的全程顾问服务。  
- 目标结果：  
  - 高质量完成 NUS / NTU / 其他名校的申请流程。  
  - 清晰认知“我们只提供专业协助，不保证录取结果”。  

**S6 – 本地私立幼儿园择校（1 万中介费）**  
- 典型画像：  
  - 有幼儿（如 3–6 岁），短期以内希望孩子在新加坡私立幼儿园就读。  
  - 对后续是否继续留在新加坡、走公立 / 国际学校路线尚未完全确定。  
- 目标结果：  
  - 在预算内匹配合适的本地私立幼儿园并成功入读。  
  - 为后续 K12 路线（公立 / 国际学校）打基础。  

**S7 – 自雇 EP、公司注册 & 家庭移民 / 管家服务**  
- 典型画像：  
  - 在中国或其他国家已有事业/收入，希望通过新加坡自雇 EP 实现工作身份 + 家庭长期规划。  
  - 可能与子女留学（S1–S3）组合购买，要求“全家整体方案”。  
- 目标结果：  
  - 注册合规公司 + 自雇 EP 获批 + 家属 Dependant Pass / 学生证落地。  
  - 后续 PR 规划、税务规划与生活/商务管家服务。

## 2. Contract Coverage Check  
## 合同覆盖性简要评估

**现有主合同：`01_Legal_and_Contracts/Templates/Maple_Study_Abroad_Service_Contract_Complete.md`**  
- 性质：中英双语《出国留学服务合同》，面向自然人学生/监护人。  
- 服务定义：  
  - 明确为“留学咨询与申请服务”，包括规划、选校、材料指导、网申/递交、进度跟进、签证支持等。  
  - 对录取、签证、奖学金、移民结果不作保证，责任与风险提示较完整。  
- 覆盖客群：  
  - 能作为 S1–S6 的**统一留学服务合同母版**，前提是在合同附录或报价中写清“目标项目/院校/阶段”（如 Kaplan / AEIS / 国际学校 / NUS/NTU 硕士等）。  
  - 对低龄 / AEIS / 幼儿园并无年龄限制条款，因此从法律逻辑上也可覆盖这些服务。  
- 不覆盖 / 建议单独处理：  
  - 自雇 EP、公司注册、税务规划、PR 申请等**不属于“留学服务”**范围，建议另起《自雇 EP & 公司服务合同》或独立附件。  
  - 境外管家服务（住宿安排、孩子托管、生活协助）也建议作为单独模块写明范围与免责，而不是混在留学服务合同中。  

> 建议：  
> - 将当前合同作为“Study Abroad Master Agreement”，在每次签约时通过附录列明具体产品模块（例如：“Kaplan 本科路径 + 可选 3 个月境外管家”）。  
> - 自雇 EP & 管家服务另起合同或独立附件，以减少法律风险和责任模糊。
