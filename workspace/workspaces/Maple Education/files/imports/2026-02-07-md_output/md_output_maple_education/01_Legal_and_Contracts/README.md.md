## 使用说明 Usage Instructions

### 创建新合同 Creating New Contracts

1. **复制HTML模板 Copy HTML Template**
   ```
   从Templates文件夹复制相应的HTML模板
   Copy the appropriate HTML template from Templates folder
   ```

2. **编辑内容 Edit Content**
   - 填写具体信息（甲乙方、日期等）
   - Fill in specific information (Party A/B, dates, etc.)
   - 所有填空处已用下划线标出
   - All blank fields are marked with underlines

3. **生成PDF Generate PDF**
   - 在浏览器中打开HTML文件
   - Open HTML file in browser
   - Ctrl+P (Print)
   - 纸张大小：A4，边距：无（0）
   - Paper size: A4, Margins: None (0)
   - 保存为PDF
   - Save as PDF

4. **存档 Archive**
   - 将签署的PDF保存到 Executed 文件夹
   - Save signed PDF to Executed folder
   - 命名格式：YYYY-MM-DD_[Type]_[Description]_Bilingual.pdf
   - Naming format: YYYY-MM-DD_[Type]_[Description]_Bilingual.pdf

## 文件命名规范 File Naming Convention

```
YYYY-MM-DD_[Type]_[Description]_[Language].pdf
```

**Type Codes:**
- `AGT` - Agreement/Contract 协议/合同
- `DAT` - Data Form 数据表格
- `DOC` - Document 文档

**Language:**
- `Bilingual` - 中英文双语
- `EN` - 英文
- `CN` - 中文
