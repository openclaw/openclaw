## 1. 本科项目服务（示例：北京办事中心 19,800 RMB）

- 参考资料：根目录 `新加坡本科北京办事中心服务项目列表-19800.pdf`  
- 适用对象：计划申请新加坡本科、公立/私立院校的高中生及家长。  
- 服务内容（概览）：  
  - 留学规划与选校方案（NUS / NTU / SMU 等公立院校；Kaplan、PSB、Amity 等私立合作院校）；  
  - 学术评估与背景梳理：根据学生成绩与履历给出可行路径；  
  - 文书与背景提升建议：个人陈述、活动规划、科研/竞赛/实习建议；  
  - 申请执行：网申、材料检查与提交、跟进录取结果；  
  - 签证与行前：Student Pass 办理指导、行前说明。  
- 定价说明：  
  - “19,800 RMB” 为示例项目费，用于说明服务层级和预算区间；  
  - Maple 采用透明定价与分阶段付款安排，**实际费用和优惠以当期价目表和正式合同为准**。  

## 2. 枫叶留学境外管家服务（Overseas Butler Service）

- 参考文件：`枫叶留学境外管家服务.pdf`（文字版见 `枫叶留学境外管家服务.pdf.md`）  
- 服务周期：标准为三个月，可按需延长或升级套餐。  
- 核心服务内容（概要）：  
  - 行前指导：签证获批后提供入境须知、行前准备清单；  
  - 抵达接应（可选）：安排专车及老师接机，送至学校或宿舍；  
  - 生活安置：陪同办理电话卡、交通卡，熟悉宿舍及周边环境，采购基础生活用品；  
  - 入学与手续：陪同报到、体检与学生证领取（相关费用由学生自理），协助预约银行开户等；  
  - 持续关怀：定期了解生活与适应情况，提醒主要考试与续签节点，如需可协助安排额外学术辅导（费用另行约定）；  
  - 社交与紧急支持：不定期留学生聚会，7×24 小时联络渠道，在紧急情况时协助协调。  
- 价格说明：  
  - 宣传文案中的价格与优惠示例（如各分项 S$100 / S$200 / S$300 组合及打包价）仅用于说明服务结构和价值组合；  
  - 实际收费标准和优惠方案以当期正式报价单及双方签署的服务合同为准。
