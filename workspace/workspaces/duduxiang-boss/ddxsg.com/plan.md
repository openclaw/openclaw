# 执行计划

## 目标

为嘟嘟香餐厅创建一个移动端优先的网站，支持菜单浏览、标签筛选、购物车和订单复制功能。

### 验收标准
- [x] 网站在手机上显示完美
- [x] 可以通过标签筛选菜品
- [x] 购物车功能正常
- [x] 可以复制订单到剪贴板
- [x] Google Maps 导航正常
- [x] WhatsApp 链接正常
- [x] 微信号复制正常

## 当前理解

- 餐厅位于新加坡宏茂桥，主营麻辣烫、麻辣香锅、烧烤
- 网站需要移动端优先设计
- 菜品需要标签系统（类型、肉类、辣度等）
- 需要购物车和订单复制功能
- 菜品图片将使用 Gemini API 生成（可选后续任务）
