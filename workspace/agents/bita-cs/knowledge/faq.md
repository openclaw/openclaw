# 會員常見問題庫 (FAQ)

## 交易相關

### FAQ-001：會不會扣手續費？

```
不會喔！轉帳金額全額計算，不會扣手續費，請放心😊
```

適用：LINE Pay、銀行轉帳

### FAQ-002：這樣夠不夠？（金額疑問）

```
夠的喔！{金額} × {匯率} = {遊戲幣}，沒問題！
```

注意：一定要先回答再處理，不能忽略

### FAQ-003：什麼時候會到帳？

- 銀行轉帳：確認收款後 5-10 分鐘內贈禮
- 超商代碼：繳費後約 2 小時內
- LINE Pay：確認後即時處理

```
收到轉帳後會盡快處理，通常 {時間} 內完成喔～
```

### FAQ-004：怎麼繳費？（超商代碼）

```
拿著代碼到 7-11 或全家，跟店員說要繳費，報代碼就可以了～
⚠️ 超商會收 40 元手續費
```

### FAQ-005：可以分開買嗎？（分次購買）

```
可以喔！每筆分開計算，匯率一樣是 {匯率}～
```

### FAQ-006：匯率多少？

```
目前 {平台} 的匯率是 {匯率} 喔～
```

匯率查 `platforms.md`

## 帳號相關

### FAQ-007：忘記遊戲帳號

```
請到遊戲裡的「設定」或「個人資料」查看您的帳號喔～
```

### FAQ-008：匯錯帳號怎麼辦？

```
請提供正確帳號，我幫您確認處理方式，請稍候～
```

⚠️ 此情況需轉交人工處理
