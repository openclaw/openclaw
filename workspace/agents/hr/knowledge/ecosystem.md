# 博弈招聘生態筆記

## 核心規則

### 1. 只找全遠端，不問「能不能遠端」
- 老闆如果寫 onsite/落地/駐場，就是 onsite。不會改
- 這些老闆的控制欲很強 — 怕員工搬走公司、怕找不到人
- **我們要的就是找不到人。** 全遠端 = 風險管理的底線
- 不是要違法，但這行業隨時可能需要「消失」的能力

### 2. 獵頭是關鍵中間人
- TG 上的「HR」幾乎都不是公司內部 HR，是獵頭
- 他們靠佣金過活，跟公司有合作關係或人脈
- 水平參差不齊，但能當獵頭的都有一定關係網

### 3. 好獵頭的價值
- 知道哪家公司對**兼職不查** — 這對我們至關重要
- 需要業績時會主動幫你推 — 互利關係
- 能提供內部情報（公司文化、老闆風格、實際工作量）
- 值得維護長期關係

### 4. 篩選鐵律

| 條件 | 必須 |
|------|------|
| 全遠端（不限地區） | ✅ 硬性 |
| 不露臉/少開會 | ✅ 硬性 |
| 不查兼職 | ✅ 硬性（透過獵頭打聽） |
| 需求精確/交付結果 | ✅ 優先 |
| 25K+ RMB | ✅ 底線 |
| 落地/駐場/第三國 | ❌ 直接跳過 |
| 能不能遠端？ | ❌ 不問，浪費時間 |

### 5. 投遞策略
- 在 TG 群看到「遠端」「居家」「不限地區」→ 才聯繫
- 先判斷對方是獵頭還是公司直招
- 獵頭 → 建立關係，問他手上還有什麼遠端坑
- 公司直招 → 直接投，但這種比較少見

### 6. 獵頭關係管理
- 即使這次沒上，也留好關係 — 他下次有遠端坑會想到你
- 適時讓獵頭知道你的「賣點」：圈內人、經驗深、交付快
- 但絕不透露多份工作的事實
