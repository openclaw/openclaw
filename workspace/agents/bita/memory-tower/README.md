> 從 8 個 TG 群的真實客服對話萃取，餵養 bita-line AI 客服的準確率。

## 架構

```
L0/          原礦 — 逐則萃取的結構化原始資料
  {group}/   按群組分目錄
    batch-{N}.json   每批 50 則的萃取結果

L1/          事實 — 分 6 類打標
  transactions.md    交易場景（買幣/回收/超商）
  phrasings.md       客戶問法變體
  good-responses.md  正確回法（高分+通過審核）
  bad-responses.md   錯誤回法（低分+被攔截）
  risk-triggers.md   風控觸發案例
  edge-cases.md      邊界案例（貼圖/圖片/罵人/聽不懂）

L2-patterns.md   模式 — 從 L1 聚類的回覆策略
L3-principles.md 原則 — AI 客服鐵律
```
