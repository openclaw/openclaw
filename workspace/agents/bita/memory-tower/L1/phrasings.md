## 一、買幣問法（25+ 種）

### 標準型（平台+金額）

- 「我要贏1000」「我要贏 1500」「我要贏5000」
- 「天狐宴300」「天狐宴350」「天狐宴1000」
- 「買天狐宴2000」
- 「少年董 買1000」
- 「買幣3000」「買2000」「買入1000」
- 「我要買幣」（不說金額，需追問）
- 「哈囉我要買500」

### 簡稱型（AI 需理解）

- 「中年」= 少年董
- 「少年」= 少年董
- 「天狐」= 天狐宴
- 「匯款天狐1800」= 天狐宴 $1,800

### 超商型

- 「儲值1000」「超商代碼」「我要贏」（分三句）
- 「金爸爸 超商500」
- 「2000 711少年董」
- 「我要贏超商」「3000」（分兩句）
- 「超商3000」「中年」（超商+暱稱）
- 「一樣 超商300」（回頭客重複）
- 「一樣 500超商」

### 回頭客/連續型

- 「再3500」「超商」（加碼）
- 「再買1000」
- 「我再買五百」「一樣帳號嗎」
- 「一樣」= 同平台同金額

### 非整數型

- 「能買580嗎」「轉帳」

### 分單型

- 「1000幫我贈給婷婷那隻」（指定不同帳號）
- 「一千轉另外一個」（分贈第二帳號）

### 帳號碼簡寫型

- 「956 存500」= 帳號後三碼 + 金額
- 「少年董zvvr876 買500」= 全帳號 + 平台 + 金額

### 其他動詞型

- 「買分5000」= 買幣
- 「代碼1700謝謝」= 超商代碼
- 「星城五百轉帳」= 平台 + 金額 + 轉帳

### 急促型

- 「在嗎」（傳焦急貼圖）
- 「有人在家嗎」
- 「不小心刪掉紀錄了」= 需重發帳號

### 改金額型

- 「五百塊的幣 是多少」→ 看完匯率 →「我買一千好了」
- 「改兩千」「說錯了」

## 二、回收問法（15+ 種）

### 標準型

- 「收」（一字）
- 「賣」（一字）
- 「要洗」（= 回收）
- 「回分」
- 「我要回幣」
- 「我要賣幣432000」（精確幣數）
- 「我先賣1500」（金額）
- 「我想要再回1000」

### 幣數報量型

- 「老闆我50萬」（= 500,000 遊戲幣）
- 「我贈禮額度還有356000可以賣多少」
- 「可贈16萬分」

### 帶平台型

- 「妳好我要回3400」（省略平台，需從上下文推）
- 「再回8000」→「10000才對」（自己改數字）

### 分身帳號

- 「可是是分身能洗嗎」（分身帳號回收）

## 四、非交易問法

| 客戶說                        | 意思                 | AI 處理                   |
| ----------------------------- | -------------------- | ------------------------- |
| 「註冊好了，我要賣」          | 新客完成註冊，要回收 | → 轉人工 KYC              |
| 「跟你們買可以升VIP嗎」       | 問 VIP 機制          | AI 可回答（說明會員等級） |
| 「VIP制度」「金牌會員是什麼」 | 問 VIP 詳情          | AI 可回答（發會員表格）   |
| 「申請退退」「申請上週反水」  | 反水申請             | → 轉人工                  |
| 「算一下上周貢獻謝謝」        | 週貢獻計算           | → 轉人工                  |
| 「公會要怎麼進」              | 加入公會             | AI 可引導（提交申請）     |
| 「我戶頭要換 可以嗎」         | 換銀行帳戶           | → 轉人工（收截圖登記）    |
| 「限額解除了嗎」              | 問銀行限額           | → 轉人工確認              |
| 「領500送5000」               | 首充活動             | AI 可確認有此活動         |
| 「現在學乖力有贏要先回」      | 閒聊（贏了先回收）   | AI 可簡單回應             |

## 五、客人確認/回覆用語

| 說法                       | 意思                                            |
| -------------------------- | ----------------------------------------------- |
| 「收」                     | 收到/完成（根據語境可能是「回收」或「收到了」） |
| 「好了」「好」「對」       | 確認                                            |
| 「同意了」「已同意」       | 遊戲內同意贈禮                                  |
| 「完成」「過去了」「轉了」 | 已轉帳/已贈禮                                   |
| 「正確」                   | 銀行帳號確認正確                                |
| 「順便」                   | 附帶要求                                        |
| 「謝謝」「謝謝妳」         | 結束                                            |
| OK 貼圖 / 顏文字           | 確認                                            |
