# Contacts 联系人数据

每个人一个文件夹，用于存储与该联系人相关的：
- 对话记录
- 项目文件
- 个人偏好
- 待办事项

## 结构

```
contacts/
├── chaochao/      # 朝朝 - 具体信息见文件夹内
├── liangxiaolong/ # 梁小龙 - Leonard 的朋友
├── sunwei/        # Sun Wei - 中国留学生
└── ...
```

## 原则

1. **隐私优先** — 敏感信息不上传 git
2. **按需创建** — 只为需要长期交互的人建文件夹
3. **保持更新** — 重要信息及时记录
