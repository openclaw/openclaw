# 每日笔记 (Daily Notes)

## 用途
记录每天做了什么、推进了什么项目、遇到什么问题。

## 命名规范
- 格式：`YYYY-MM-DD.md`
- 示例：`2026-02-26.md`

## 模板
```markdown
# YYYY-MM-DD

## 今日任务
- [ ] 任务1
- [ ] 任务2

## 完成事项
- [x] 事项1
- [x] 事项2

## 遇到的问题
- 问题1：描述 → 解决方案
- 问题2：描述 → 待解决

## 明日计划
- 计划1
- 计划2

## 备注
- 其他需要记录的信息
```

## 自动化
- 每日 02:00 由定时任务自动整理前一天的聊天记录，提取要点更新至此
