### 第 2 周：筛选期
**目标**：退货率 50% → 40%

**关键动作**：
1. 测试 5-10 家新供应商
2. 索要样品，严格质检
3. 只保留质量稳定的供应商

**供应商筛选标准**：
```
✅ 发货速度：24 小时内
✅ 质量稳定：退货率 <20%
✅ 尺码准确：有详细尺码表
✅ 图片真实：有实拍图
✅ 售后响应：2 小时内回复
```

---

### 第 3 周：优化期
**目标**：退货率 40% → 30%

**关键动作**：
1. 优化尺码表（重点！）
2. 真实化商品图片
3. 完善商品描述

**尺码表优化方案**：
```
✅ 要求供应商提供详细尺寸
✅ 实测样品，标注误差
✅ 添加"建议身高/体重"参考
✅ 增加"偏大/偏小/标准"提示
✅ 配上模特图（身高体重）
```

**示例**：
```
【S 码】
- 建议身高：155-160cm
- 建议体重：45-50kg
- 胸围：84cm（偏小一码）
- 备注：此款偏小，建议拍大一码
```
