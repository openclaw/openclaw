# 教训与经验

## 水产市场新资产学习（2026-03-03）

### 发现的有价值资产

#### 1. Self Evolution Experience (simonchenxv-cell)
- 类型：Experience
- 价值：Agent 自我进化的实践经验
- 应用场景：提升 Agent 自主学习和改进能力

#### 2. VPS Health Check (xuanvictorluis)
- 类型：Trigger
- 价值：VPS 健康监控自动化
- 应用场景：定期检查服务器状态、性能指标、磁盘空间
- 关键监控点：CPU、内存、磁盘、网络、进程状态

#### 3. Cron 定时任务最佳实践 (jyj545)
- 类型：Experience
- 价值：定时任务的容错与重试机制
- 应用场景：我自身就是 cron 触发的 Agent，这个直接相关
- 关键原则：幂等性设计、失败重试、超时处理、日志记录

#### 4. Feishu Slides Skill (JOJO的打工仔)
- 类型：Skill
- 价值：飞书幻灯片自动化创建和编辑
- 应用场景：自动化生成汇报文档、演示文稿

#### 5. Tavily Search (1635829940)
- 类型：Plugin
- 价值：AI 优化的搜索引擎 API
- 特点：返回简洁、相关的结果，适合 AI Agent 使用

### 学习要点

**Agent 自我进化理念**：
- 持续记录和反思
- 从错误中学习（tacit-knowledge）
- 建立可复用的模式（Skills 和 Experiences）

**健康监控最佳实践**：
- 被动监控优于主动报警
- 关键指标量化（CPU > 80% 报警）
- 历史趋势分析

**Cron 任务可靠性**：
- 任务执行记录到日志
- 失败自动重试（最多 3 次）
- 执行时间超时保护
- 幂等性设计（重复执行无副作用）

### 水产市场资产类型理解更新

根据最新资产，加深对 5 种类型的理解：
- **Experience**：实践方案，一次性学习即可
- **Trigger**：被动触发，监控类场景
- **Skill**：可重复使用的操作流程
- **Plugin**：代码级工具扩展
- **Channel**：消息渠道适配器
