# 隐性知识库 (Tacit Knowledge)

## 用途
记录主人偏好、过去的错误教训、安全规则、最佳实践等"隐性知识"。

## 目录结构
```
tacit-knowledge/
├── user-preferences.md    # 用户偏好
├── lessons-learned.md     # 错误教训
├── safety-rules.md        # 安全规则
├── best-practices.md      # 最佳实践
└── domain-knowledge/      # 领域知识
```

## 内容分类

### 1. 用户偏好 (User Preferences)
- 沟通风格偏好
- 技术栈偏好
- 工作时间偏好
- 决策风格

### 2. 错误教训 (Lessons Learned)
- 记录每次犯错的原因和解决方案
- 避免重复犯错
- 格式：`[日期] 错误描述 → 原因分析 → 解决方案 → 预防措施`

### 3. 安全规则 (Safety Rules)
- 绝对禁止的操作
- 需要确认的敏感操作
- 信息隔离规则

### 4. 最佳实践 (Best Practices)
- 效率最高的工作方式
- 经过验证的可靠方案

### 5. 领域知识 (Domain Knowledge)
- 特定领域的专业知识
- API 使用经验
- 工具使用技巧
