# 活跃项目跟踪 (Active Projects)

## 用途
追踪当前正在进行的所有项目，让 AI 随时知道"我在做什么"。

## 项目状态
| 项目名 | 状态 | 优先级 | 开始日期 | 最后更新 | 负责人 |
|--------|------|--------|----------|----------|--------|
| 示例项目 | 进行中 | 高 | 2026-02-26 | 2026-02-26 | 主人 |

## 活跃项目列表

### 1. 项目名称
- **状态**：进行中 / 暂停 / 等待
- **目标**：项目要达成的目标
- **当前进度**：描述当前进度
- **下一步**：接下来要做什么
- **阻塞项**：有什么阻碍
- **相关文件**：链接到相关文档

## 归档规则
- 项目完成后，移动到 `archive/` 目录
- 保留完整历史记录
