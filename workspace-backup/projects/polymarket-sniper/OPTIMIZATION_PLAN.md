## 📋 优化目标

### 1. 接入实时新闻 API
**问题**：事件源单一（仅硬编码伊朗事件）

**解决方案**：
- 免费源优先：RSS feeds、NewsAPI（100次/天免费）
- 付费源备用：Brave Search（$0.0018/次）

### 2. 提升关键词匹配
**问题**：关键词匹配粗糙（"伊朗" → 33% 相关性）

**解决方案**：
- 本地 ONNX Embeddings（all-MiniLM-L6-v2）
- 向量相似度计算
- 替代关键词匹配

## 📊 技术栈

### 数据源
- **RSS**：`feedparser`（免费）
- **NewsAPI**：100 次/天免费
- **Brave Search**：$0.0018/次（付费备用）

### 向量相似度
- **模型**：`all-MiniLM-L6-v2` (22MB)
- **安装**：`pip install sentence-transformers`
- **优势**：本地推理，零成本

### 缓存
- **Redis**：事件缓存（15分钟）
- **安装**：`pip install redis`
